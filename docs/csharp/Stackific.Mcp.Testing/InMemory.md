# InMemory class

A test-kit that wires an [`McpClient`](../Stackific.Mcp.Client/McpClient.md) directly to an [`McpServer`](../Stackific.Mcp.Server/McpServer.md) over the in-memory transport (spec-faithful request/response shape, no HTTP). Useful for end-to-end SDK tests and for embedding a server in the same process as its client.
