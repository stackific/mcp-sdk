# JsonValues.IsSafeInteger method

Returns `true` when *n* is a safe integer: it has no fractional part and lies within the safe-integer range. (R-2.5-c, R-2.5-e, AC-02.14)

```csharp
public static bool IsSafeInteger(double n)
```

| parameter | description |
| --- | --- |
| n | The number to test. |

## Return Value

`true` when *n* is a finite, fraction-free value in range.

## Remarks

Mirrors JavaScript's `Number.isSafeInteger`: the comparison against the bounds is done in floating point so that values just past the boundary (for example [`SafeIntegerMax`](./SafeIntegerMax.md) + 1) are correctly rejected even though they are still integral.
