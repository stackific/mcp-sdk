# JsonValues.NumericEqual method

Returns `true` when *a* and *b* are numerically equal regardless of their textual JSON representation (for example `1e2` equals `100`, and `1.0` equals `1`). Two numerically equal JSON numbers MUST be treated as equal. (R-2.5-g, AC-02.15)

```csharp
public static bool NumericEqual(double a, double b)
```

| parameter | description |
| --- | --- |
| a | The left number. |
| b | The right number. |

## Return Value

`true` when the two values are numerically equal.
