# JsonValues.IsJsonValue method

Returns `true` when *value* is a valid JSON value — one of the six wire forms (string, number, boolean, null, object, array) — recursively. Useful as a runtime guard at a system boundary. (R-2.3-a, AC-02.1)

```csharp
public static bool IsJsonValue(JsonNode? value)
```

| parameter | description |
| --- | --- |
| value | The candidate node; `null` represents JSON `null`. |

## Return Value

`true` when the node and all nested nodes are valid JSON values.

## Remarks

In the JsonNode model a JSON `null` is represented by the CLR `null` reference, which is always a valid JSON value. A non-null node is valid when it is a JsonValue (scalar), a JsonObject whose members are all valid, or a JsonArray whose elements are all valid. A JsonValue backed by a non-JSON CLR type (something that has no JSON value kind) is rejected — the analogue of TypeScript rejecting `undefined`, a function, or a `Symbol`.
