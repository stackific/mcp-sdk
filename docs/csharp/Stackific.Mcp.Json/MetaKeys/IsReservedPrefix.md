# MetaKeys.IsReservedPrefix method

Returns `true` if *key* sits under a protocol-reserved prefix — its second label is `modelcontextprotocol` or `mcp` (§4.2). Third parties MUST NOT mint keys under such a prefix.

```csharp
public static bool IsReservedPrefix(string key)
```

| parameter | description |
| --- | --- |
| key | The candidate key (or prefix). |

## Return Value

`true` when the key's prefix is protocol-reserved.

## Remarks

Retained for callers that pass a full key and only want the reservation test (without the stricter validity contract of [`IsValidKey`](./IsValidKey.md)). It is an alias for [`IsReservedMetaKeyPrefix`](./IsReservedMetaKeyPrefix.md) with the prior full-key semantics: a key without a non-empty prefix (a bare key or a leading slash) is never reserved.
