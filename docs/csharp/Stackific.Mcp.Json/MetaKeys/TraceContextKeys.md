# MetaKeys.TraceContextKeys field

The bare keys reserved for W3C trace-context propagation: `traceparent`, `tracestate`, and `baggage` (§2.6.2 / §4.2, R-2.6.2-i). These are always valid despite carrying no prefix — they are permitted by exception to the prefix rule.

```csharp
public static readonly IReadOnlySet<string> TraceContextKeys;
```

## Remarks

This mirrors the TypeScript `TRACE_CONTEXT_KEYS` set exactly. It does NOT include `progressToken`, which is reserved for a different purpose.
