# ParamEncoding.NeedsSentinel method

Returns `true` when *plain* cannot be safely carried as a plain ASCII header value and so MUST be sentinel-encoded. (R-9.5.3-b, R-9.5.3-e)

```csharp
public static bool NeedsSentinel(string plain)
```

| parameter | description |
| --- | --- |
| plain | The plain per-type string form. |

## Return Value

`true` when sentinel encoding is required.

## Remarks

Unsafe when it contains non-ASCII or control characters, has leading or trailing whitespace, or already matches the sentinel shape (to avoid ambiguity — the self-collision guard, R-9.5.3-e). Safe ASCII is visible ASCII `0x21`–`0x7E`, space `0x20`, and horizontal tab `0x09`, with no leading/trailing whitespace.
