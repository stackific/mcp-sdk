# ParamEncoding.IsAnnotatedIntegerInRange method

Returns `true` when *value* is an integer within the safe annotated range. (R-9.5.1-g)

```csharp
public static bool IsAnnotatedIntegerInRange(double value)
```

| parameter | description |
| --- | --- |
| value | The number to test. |

## Return Value

`true` when finite, whole, and within range.

## Remarks

The comparison is performed in floating point so a value just past the boundary (for example [`MaxSafeAnnotatedInteger`](./MaxSafeAnnotatedInteger.md) + 1) is correctly rejected even though it is still integral — matching JavaScript's `Number.isInteger` plus the range test in the TypeScript source.
