## Remarks

A server MAY annotate `inputSchema` parameters with `x-mcp-header` to mirror them into request headers; clients on this transport MUST support it. This module ports the TypeScript SDK's `transport/http/param-headers.ts` and covers:

* `x-mcp-header` annotation validity (§9.5.1) and client rejection of invalid tools, keeping other tools usable.
* client emission of `Mcp-Param-{name}` headers from a tool's schema and the call arguments (§9.5.2), with value encoding ([`ParamEncoding`](./ParamEncoding.md)).
* receiver validation of those headers against the body (§9.5.4), including numeric comparison of integers, with any mismatch yielding `-32001` (HeaderMismatch).
