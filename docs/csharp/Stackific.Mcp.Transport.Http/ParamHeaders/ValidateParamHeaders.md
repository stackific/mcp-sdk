# ParamHeaders.ValidateParamHeaders method

Validates the `Mcp-Param-*` headers of a request against its body. (§9.5.4)

```csharp
public static McpError? ValidateParamHeaders(JsonNode? inputSchema, JsonObject? args, 
    Func<string, string?> getHeader)
```

| parameter | description |
| --- | --- |
| inputSchema | The tool's `inputSchema` (source of annotations). |
| args | The body `params.arguments`, or `null`. |
| getHeader | A case-insensitive header accessor returning `null` when absent. |

## Return Value

`null` on success, or the `-32001` error on a mismatch.

## Remarks

* A recognized header with impermissible characters → `-32001`. (R-9.5.4-b)
* A header whose decoded value does not match the body value → `-32001`; integers are compared numerically. (R-9.5.4-c, R-9.5.4-d)
* A body value present while its header is omitted → `-32001`. (R-9.5.2-k)
* A header present while the body value is absent/null → `-32001`.

Returns `null` when valid; otherwise an [`McpError`](../../Stackific.Mcp.JsonRpc/McpError.md) with code `-32001` for the caller to surface as a `400` error response.
