# AuthGates.Bearer method (2 of 2)

Builds an [`IMcpAuthGate`](../../Stackific.Mcp.Transport/IMcpAuthGate.md) that requires a valid `Bearer` token (spec §23.1). On a missing, malformed, invalid, or expired token it answers `401` with a `WWW-Authenticate: Bearer …` challenge carrying *resourceMetadataUrl*; on a token whose audience does not match *expectedAudience* it answers `401` (§23.6 — a server MUST reject a token not issued for it and never forward it). On success it threads the validated [`AuthInfo`](../../Stackific.Mcp.Transport/AuthInfo.md) into request processing.

```csharp
public static IMcpAuthGate Bearer(string resourceMetadataUrl, string expectedAudience, 
    Func<string, AuthInfo?> validate)
```

| parameter | description |
| --- | --- |
| resourceMetadataUrl | The protected-resource metadata URL advertised via `resource_metadata`. |
| expectedAudience | This resource's canonical identifier; the token's audience MUST equal it (§23.6). |
| validate | Validates a bearer token, returning the caller's identity, or `null` to reject. |

## Return Value

The configured authorization gate.
