# AuthGates.BuildProtectedResourceMetadata method

Builds an RFC 9728 protected-resource metadata document (spec §23.2): the canonical `resource` identifier, the `authorization_servers` that protect it, the `scopes_supported` it recognizes, and the bearer delivery methods it accepts (header only). `scopes_supported` is emitted only when scopes are provided, matching the TypeScript builder.

```csharp
public static JsonObject BuildProtectedResourceMetadata(string resource, 
    IReadOnlyList<string> authorizationServers, IReadOnlyList<string> scopes)
```

| parameter | description |
| --- | --- |
| resource | The canonical resource identifier (the MCP endpoint URL). |
| authorizationServers | The authorization-server issuer URLs that protect this resource. |
| scopes | The scopes the resource recognizes. |

## Return Value

The protected-resource metadata as a JsonObject.
