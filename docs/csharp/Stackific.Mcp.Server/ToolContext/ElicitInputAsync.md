# ToolContext.ElicitInputAsync method

Requests structured or out-of-band user input via elicitation (spec §20), using the multi-round-trip mechanism (§11): on the first round this signals `input_required` and the runtime suspends the call; when the client retries with the answer, this returns it. Requires the client to have declared the `elicitation` capability, else `-32003` (§11.5).

public Task<ElicitResult> ElicitInputAsync(ElicitRequestParams parameters)

| parameters | The elicitation request (form or URL mode). |
