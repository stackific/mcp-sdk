# ToolContext.NotifySubscribersAsync method

Fans a change notification out to every active subscription stream whose filter opted into its kind (spec §10.5). Used by tools that mutate server state (for example to drive the Subscriptions view). Delivery to request-scoped streams is via [`NotifyAsync`](./NotifyAsync.md) instead (§10.6).

```csharp
public Task NotifySubscribersAsync(JsonRpcNotification notification)
```

| parameter | description |
| --- | --- |
| notification | The change notification (a list-changed or resource-updated kind). |

## Return Value

A task that completes when the fan-out is enqueued.
