# BearerAuthGateOptions constructor

Configures the server bearer-auth gate (spec §23): the protected-resource metadata URI advertised on the `401` challenge, this resource's canonical identifier (the expected token audience), the scopes a request must carry, and the token-validation callback.

```csharp
public BearerAuthGateOptions(string ResourceMetadataUrl, string ExpectedAudience, 
    Func<string, AuthInfo?> Validate, IReadOnlyList<string>? RequiredScopes = null)
```

| parameter | description |
| --- | --- |
| ResourceMetadataUrl | The protected-resource metadata URL advertised via `resource_metadata` (R-23.1-v). |
| ExpectedAudience | This resource's canonical identifier; the token's audience MUST equal it (§23.6). |
| Validate | Validates a bearer token, returning the caller's identity, or `null` to reject. |
| RequiredScopes | The scopes a valid token MUST grant; a token lacking one yields a `403` insufficient_scope challenge (§23.8, R-23.8-f). Empty/absent means no scope is enforced. |
