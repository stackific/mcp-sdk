# StdioServerHost.Serve method

Begins serving *server* over *transport*: subscribes to inbound messages and dispatches them until the host is disposed or the transport closes. Returns a running host whose disposal stops handling further inbound messages (the idiomatic C# analogue of the TypeScript `serveStdio` unsubscribe function).

```csharp
public static StdioServerHost Serve(IMcpRequestHandler server, IByteChannelTransport transport, 
    AuthInfo? authInfo = null)
```

| parameter | description |
| --- | --- |
| server | The server runtime to serve. |
| transport | The byte-channel transport (typically a [`StdioServerTransport`](../../Stackific.Mcp.Transport/StdioServerTransport.md)). |
| authInfo | An optional pre-validated identity attached to every dispatched request (§23). |
