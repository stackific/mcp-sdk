# SubscriptionManager.Register method

Registers a subscription stream and returns the filter the server agreed to honor plus a teardown handle (spec §10.3). The honored filter is the intersection of the requested kinds and the kinds the server's capabilities support; the `taskIds` filter is honored only when the Tasks extension is active for this request (§25.10) — see the *tasksActive* parameter.

```csharp
public (SubscriptionFilter Honored, IDisposable Teardown) Register(SubscriptionFilter requested, 
    ServerCapabilities capabilities, string subscriptionId, 
    Func<JsonRpcNotification, Task> deliver, bool tasksActive = false)
```

| parameter | description |
| --- | --- |
| requested | The filter the client requested. |
| capabilities | The server's advertised capabilities. |
| subscriptionId | The subscription id (the `subscriptions/listen` request id, as a string). |
| deliver | The sink that writes a notification onto this subscription's stream. |
| tasksActive | Whether the Tasks extension is active for this `subscriptions/listen` request (§25.10); when `false`, a requested `taskIds` filter is dropped. |

## Return Value

The honored filter and a disposable that unregisters the subscription.
