# SubscriptionManager.FanOutAsync method

Fans a change notification out to every subscription whose honored filter selects it (spec §10.5, §25.10). Supports the four change kinds (list-changed × 3 + resources/updated) plus the `notifications/tasks` push, with sub-resource container coverage for resource URIs (§10.5, R-10.5-j) and per-task `taskIds` gating for task pushes (§25.10, R-25.10-d).

```csharp
public Task FanOutAsync(JsonRpcNotification notification)
```

| parameter | description |
| --- | --- |
| notification | The change or task notification. |

## Return Value

A task that completes when delivery to all matching subscriptions is done.
