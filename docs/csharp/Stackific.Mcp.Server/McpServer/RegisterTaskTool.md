# McpServer.RegisterTaskTool method

Registers a task-augmented tool (spec §25.3): when a caller that declared the Tasks extension invokes it, the server returns a task handle immediately and the handler drives the work to the [`TaskStore`](./TaskStore.md) in the background. The handler creates the task (via [`Tasks`](../ToolContext/Tasks.md)), starts its background work, and returns the handle.
