# McpServer.GetToolInputSchema method

Resolves a registered tool's `inputSchema` by name, or `null` for an unknown tool (§16.4). The Streamable HTTP adapter consumes this so a `tools/call` can have its `Mcp-Param-*` headers validated/decoded against the tool's schema (§9.5.4) — see [`MapMcp`](../../Stackific.Mcp.Transport/StreamableHttpServer/MapMcp.md).

```csharp
public JsonNode? GetToolInputSchema(string toolName)
```

| parameter | description |
| --- | --- |
| toolName | The tool name. |

## Return Value

The tool's input schema node (a clone), or `null` when the tool is not registered.
