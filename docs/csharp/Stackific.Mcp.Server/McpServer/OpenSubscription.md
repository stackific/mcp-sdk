# McpServer.OpenSubscription method

```csharp
public (SubscriptionFilter Honored, IDisposable Teardown) OpenSubscription(
    SubscriptionFilter requested, string subscriptionId, Func<JsonRpcNotification, Task> deliver)
```

## Remarks

§25.10 gating: a `subscriptions/listen` carrying a `taskIds` filter requires the Tasks extension to be active. The transport does not thread the request's client-capability `_meta` into this call, so activeness is determined from the server's advertised Tasks extension — a server that does not advertise Tasks cannot serve task pushes. When `taskIds` is supplied but Tasks is not active, the listen is rejected with `-32003` (R-25.10-e). Threading the client's negotiated tasks capability through the subscription handler is a Phase-5 transport seam.
