# McpServer.CacheScope property

The default top-level `cacheScope` for cacheable-method results (§13.3). Defaults to Private — the privacy-default a server MUST apply when it cannot reliably distinguish authorization contexts (R-13.1-e, R-13.3-h). Emission is routed through [`ResolveCacheScope`](../../Stackific.Mcp.Protocol/Caching/ResolveCacheScope.md) so the resolved scope always lands on the privacy default for any unrecognized value.
