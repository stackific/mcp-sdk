# InMemoryTaskStore.StatusOf method

The current status of a task, or `null` if it is unknown/expired (read defensively).

```csharp
public McpTaskStatus? StatusOf(string taskId)
```

| parameter | description |
| --- | --- |
| taskId | The task id. |

## Return Value

The status, or `null`.

## Remarks

Like [`Get`](./Get.md) this is expiry-aware (§25.6): a task whose non-null `ttlMs` has elapsed is swept and reported as unknown (`null`), so a poller observing status sees an expired task as gone rather than frozen in its last non-terminal state.
