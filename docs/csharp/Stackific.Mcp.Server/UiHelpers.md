# UiHelpers class

Helpers for the Interactive User-Interface extension (spec §26): a server returns an interactive UI by embedding a `ui://` resource (`text/html;profile=mcp-app`) in a tool result, which the host renders sandboxed. The C# counterpart of the SDK's `uiToolResult`.
