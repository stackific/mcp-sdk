# InMemoryTaskStore class

An in-memory, thread-safe store for the Tasks extension (spec §25). It mints opaque task ids, tracks lifecycle state with terminal-state immutability (§25.5), holds the inline result/error of terminal tasks, and answers `tasks/get` with a [`DetailedTask`](../Stackific.Mcp.Protocol/DetailedTask.md). A `now` clock is injectable for deterministic tests. Durability beyond the process lifetime is out of scope for this in-memory implementation; a server needing it supplies its own store.
