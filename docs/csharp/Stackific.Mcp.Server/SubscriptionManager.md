# SubscriptionManager class

Tracks active `subscriptions/listen` streams and fans server-initiated change notifications out to the ones that opted into each kind (spec §10.5). Each delivered notification is tagged with the subscription identifier in `_meta` (§10.4), and the server only honors the filter kinds it actually supports (§10.3).
