# ResponseCache&lt;T&gt;.Set method

Stores *value* under *key* with its raw caching hints. The entry is SKIPPED when the hints are not both-present (R-13.1-g) or are invalid ([`IsCacheHintValid`](../Caching/IsCacheHintValid.md)). A `ttlMs = 0` entry is stored but will never be served as a fresh hit.
