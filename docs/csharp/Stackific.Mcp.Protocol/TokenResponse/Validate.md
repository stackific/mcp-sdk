# TokenResponse.Validate method

Parses and validates a token-endpoint response body, confirming `token_type` is `Bearer` (case-insensitive, per RFC 6749) since MCP presents the token via the `Bearer` scheme (R-23.8-b). On success *token* is populated.

public static AuthorizationResult Validate(JsonNode? value, out TokenResponse? token)
