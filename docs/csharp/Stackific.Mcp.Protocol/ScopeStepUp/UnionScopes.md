# ScopeStepUp.UnionScopes method

Computes the UNION of already-granted/already-requested scopes with the newly-challenged scopes — the scope set a step-up re-authorization requests (R-23.18-o, R-23.18-p, R-23.1-ae). Order-preserving and deduplicating: every already-granted scope is retained (R-23.18-p — never dropped) and the challenged scopes are appended.
