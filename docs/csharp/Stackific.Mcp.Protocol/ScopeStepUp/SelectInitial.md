# ScopeStepUp.SelectInitial method

Selects the least-privilege scopes for the initial authorization handshake, applying the §23.18 priority (R-23.18-a – R-23.18-d): the challenge `scope` (authoritative), else all of `scopes_supported`, else omit (`null`). Delegates to S36's [`Resolve`](../AuthorizationScopes/Resolve.md).

```csharp
public static string? SelectInitial(WwwAuthenticateChallenge? challenge, 
    IReadOnlyList<string>? scopesSupported)
```

| parameter | description |
| --- | --- |
| challenge | The parsed `WWW-Authenticate` challenge, if any. |
| scopesSupported | Protected-resource `scopes_supported`, if any. |

## Return Value

The resolved scope string, or `null` to omit `scope`.
