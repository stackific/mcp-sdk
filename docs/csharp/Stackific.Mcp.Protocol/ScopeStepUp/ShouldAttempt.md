# ScopeStepUp.ShouldAttempt method

Returns `true` when a client SHOULD attempt the step-up flow for a scope-related error: always for a user-acting client (R-23.18-m); for a ClientCredentials client it MAY attempt or abort, so this returns `false` (the conservative default) (R-23.18-l – R-23.18-n).

```csharp
public static bool ShouldAttempt(StepUpActor actor)
```

| parameter | description |
| --- | --- |
| actor | Who the client is acting for. |

## Return Value

`true` when step-up should be attempted.
