# ConformanceRequirements.MayPlaceInputRequest method

Returns `true` when a server MAY place an input request of *method* into an `input_required` result for a client declaring *clientCapabilities* (spec §29.4 item 5, R-29.4-l). An unrecognized method is rejected: a server must not solicit a kind it cannot tie to a declared capability.

```csharp
public static bool MayPlaceInputRequest(string method, JsonObject clientCapabilities)
```

| parameter | description |
| --- | --- |
| method | The input-request method. |
| clientCapabilities | The client's declared capabilities (raw). |

## Return Value

`true` when the server may place the input request.
