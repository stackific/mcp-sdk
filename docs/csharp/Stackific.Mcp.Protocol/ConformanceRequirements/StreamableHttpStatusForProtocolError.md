# ConformanceRequirements.StreamableHttpStatusForProtocolError method

Maps a protocol error *code* to the HTTP status it MUST ride on the Streamable HTTP transport for the §29.8 negotiation/envelope conditions (spec §29.8 item 3, R-29.8-c). `-32602` and `-32003` map to `400`; any other code returns `null` (governed by §9 / S34, not this conformance point).
