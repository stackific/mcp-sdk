# ConformanceRequirements.SatisfiesRole method

Returns `true` when an implementation satisfying ONLY one role's requirements is conformant for *targetRole* (spec §29.1, R-29.1-a, R-29.1-b). A both-roles implementation must satisfy each role; satisfying only the other role's requirements is non-conformant.

```csharp
public static bool SatisfiesRole(IEnumerable<Role> satisfiedRoles, Role targetRole)
```

| parameter | description |
| --- | --- |
| satisfiedRoles | The roles whose requirements the implementation provably satisfies. |
| targetRole | The role whose conformance is being judged. |

## Return Value

`true` when conformant for the target role.
