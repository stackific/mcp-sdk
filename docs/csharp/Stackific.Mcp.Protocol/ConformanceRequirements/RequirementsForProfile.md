# ConformanceRequirements.RequirementsForProfile method

Enumerates every normative requirement that APPLIES to a profile: every baseline requirement for the role(s) it plays, plus every transport requirement (spec §29.1, §29.9 item 1). The §29.4 capability-conditioned atoms apply only when the relevant capability is advertised.
