# ConformanceRequirements.ValidateConformanceProfile method

Validates that a [`ConformanceProfile`](../ConformanceRequirements.ConformanceProfile.md) is well-formed against the structural requirements of §29 (spec §29.5 item 2, §29.8 item 1, §29.9 item 3, R-29.1-b, R-29.5-c, R-29.8-a, R-29.9-c). Accumulates ALL violations: at least one recognized role; non-empty revisions including `2026-07-28`; every extension identifier well-formed per §6 (an empty list is conformant); at least one transport. Capabilities are not constrained beyond being a list.
