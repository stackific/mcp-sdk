# ConformanceRequirements.ClassifyRequirementLevel method

Classifies a normative *keyword* into its [`Level`](../ConformanceRequirements.Level.md) family (spec §2). Returns `null` for an unrecognized token — never throws — so a conformance harness can report rather than crash on a malformed marker.
