# ConformanceRequirements.EvaluateTransportConformance method

Evaluates the authorization-applicability conformance points for a single transport (spec §29.8 items 4 &amp; 5, R-29.8-d, R-29.8-e). An HTTP-based transport SHOULD conform to authorization and conveys credentials as a Bearer token; a stdio transport SHOULD NOT apply it and obtains credentials from its Environment; any other transport applies neither rule (None).
