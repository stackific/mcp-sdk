# ConformanceRequirements.ObligedSectionsForCapabilities method

Returns the spec sections whose MUST-level behavior an implementation is bound to, given the capabilities it advertises (spec §29.4 item 1, R-29.4-a – R-29.4-g). The result is deterministic, de-duplicated, numerically sorted, and includes the additional sections.

```csharp
public static IReadOnlyList<string> ObligedSectionsForCapabilities(IEnumerable<string> advertised)
```
