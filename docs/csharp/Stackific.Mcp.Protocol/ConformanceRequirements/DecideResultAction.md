# ConformanceRequirements.DecideResultAction method

Applies the §29.6 + §3 receiver rules to a result's `resultType` (spec R-29.6-f, R-29.6-g, R-29.6-h). A recognized value (core or an accepted extension value) is acted upon; a present but unaccepted value is treated as an error; an ABSENT or null discriminator resolves to `complete` via the §3 absence rule and is acted upon.
