# ConformanceRequirements.RobustnessDisposition method

Computes the §29.6 robustness disposition for one received element, given whether the receiver recognizes it (spec §29.6, R-29.6-a – R-29.6-h). A recognized element always returns Accept; robustness applies only to the unrecognized (R-29.6-i).
