# GetPromptResponseKind enumeration

The discriminated outcome of inspecting a `prompts/get` reply (spec §18.4): a completed [`GetPromptResult`](./GetPromptResult.md), an `input_required` continuation, or a malformed/unrecognized body.

| Complete | `0` | A completed prompt carrying `messages` (the absent-⇒-complete default applies, R-18.4-p). |
| InputRequired | `1` | The server needs additional client input before the prompt can render (§11, R-18.4-q). |
