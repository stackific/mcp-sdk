# Subscription.MayEmit method

Returns `true` when the server MAY emit change notification *method* on this stream (state `active` and the acknowledged filter permits it) (§10.5, R-10.5-l). For `notifications/resources/updated` pass the URI; for `notifications/tasks` pass the task id.

```csharp
public bool MayEmit(string method, string? subjectKey = null)
```

| parameter | description |
| --- | --- |
| method | The notification method. |
| subjectKey | The per-resource URI or per-task id, when the kind is keyed. |

## Return Value

`true` when emission is permitted.
