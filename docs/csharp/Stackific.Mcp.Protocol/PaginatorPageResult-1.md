# PaginatorPageResult&lt;T&gt; record

The result of [`GetPage`](./OffsetPaginator-1/GetPage.md): either a successful page (with items and an optional next cursor) or a structured invalid-cursor error. The paginator NEVER throws on an unrecognized cursor (RC-3/RC-4). Mirrors TS `PaginatorPageResult`.
