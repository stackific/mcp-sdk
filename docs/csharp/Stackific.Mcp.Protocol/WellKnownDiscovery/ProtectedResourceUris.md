# WellKnownDiscovery.ProtectedResourceUris method

Builds the ordered list of protected-resource-metadata well-known URIs to try for an MCP server endpoint, when no `resource_metadata` header URI is available (R-23.2-e, R-23.2-f): (1) path-aware insertion `/.well-known/oauth-protected-resource/<path>`; then (2) the host root. When the endpoint has no path beyond `/`, only the root URI is returned.
