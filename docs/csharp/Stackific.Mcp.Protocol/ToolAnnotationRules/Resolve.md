# ToolAnnotationRules.Resolve method

Resolves the four boolean [`ToolAnnotations`](../ToolAnnotations.md) hints to concrete values, applying the §16.7 defaults for any absent field: `readOnlyHint ⇒ false`, `destructiveHint ⇒ true`, `idempotentHint ⇒ false`, `openWorldHint ⇒ true`. Note `destructiveHint` and `idempotentHint` are meaningful only when `readOnlyHint` is `false`. (R-16.7-b, R-16.7-c, R-16.7-d, R-16.7-e)
