# ProbeOutcome record

The outcome of interpreting a probe (`server/discover`) response (§5.7). Mirrors TS `ProbeOutcome`. Modeled as a closed record hierarchy — one derived type per [`ProbeOutcomeKind`](./ProbeOutcomeKind.md) carrying ONLY the data valid for that case — so illegal combinations (for example a "not-this-protocol" outcome that nonetheless carries a parsed result) cannot be constructed. Match on the concrete type with a `switch` expression, or branch on [`Kind`](./ProbeOutcome/Kind.md).
