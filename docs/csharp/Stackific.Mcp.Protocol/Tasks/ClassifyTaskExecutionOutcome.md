# Tasks.ClassifyTaskExecutionOutcome method

Classifies how a finished augmented request maps onto a terminal task status, enforcing the strict §25.11 separation (R-25.11-f, R-25.11-h, R-25.11-i): a JSON-RPC PROTOCOL error → `failed`; a request that completed at the protocol level (even one whose `result` conveys an application error such as `isError: true`) → `completed`.

```csharp
public static McpTaskStatus ClassifyTaskExecutionOutcome(TaskExecutionOutcomeKind kind)
```

| parameter | description |
| --- | --- |
| kind | Which kind of outcome the request finished with. |

## Return Value

Failed for a protocol error; otherwise Completed.
