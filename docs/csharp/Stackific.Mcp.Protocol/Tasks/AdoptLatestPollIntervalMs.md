# Tasks.AdoptLatestPollIntervalMs method

Resolves the `pollIntervalMs` a client should honor, ADOPTING THE LATEST observed value (§25.7, R-25.7-m, R-25.7-n): the value from the most recent `tasks/get` result is preferred; failing that, the previously adopted value; failing that, *fallbackMs*.

```csharp
public static long AdoptLatestPollIntervalMs(long? latestObserved, long? previousObserved, 
    long fallbackMs = 1000)
```

| parameter | description |
| --- | --- |
| latestObserved | `pollIntervalMs` from the most recent `tasks/get`, or `null`. |
| previousObserved | The previously adopted `pollIntervalMs`, or `null`. |
| fallbackMs | The interval used when neither has supplied a value. |

## Return Value

The interval in milliseconds.
