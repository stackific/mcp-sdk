# Tasks.IsLegalTaskTransition method

Returns `true` when a task MAY transition from *from* to *to*, per the §25.5 lifecycle (R-25.5-b, R-25.5-c). A transition out of a terminal state is never legal (terminal is immutable). A self-transition between identical non-terminal states (for example `working`→`working`) is not a state change and returns `false`.

```csharp
public static bool IsLegalTaskTransition(McpTaskStatus from, McpTaskStatus to)
```

| parameter | description |
| --- | --- |
| from | The current status. |
| to | The proposed next status. |

## Return Value

`true` when the transition is legal.
