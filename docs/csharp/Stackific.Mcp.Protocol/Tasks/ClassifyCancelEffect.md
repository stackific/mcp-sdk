# Tasks.ClassifyCancelEffect method

Classifies what a server's stored task does on `tasks/cancel` (§25.9, R-25.9-h, R-25.9-i, R-25.9-j): a terminal task is AcknowledgedTerminal (acknowledge only, no state change), a non-terminal task is AcknowledgedPending (acknowledge, and the server MAY move it toward `cancelled` when feasible).
