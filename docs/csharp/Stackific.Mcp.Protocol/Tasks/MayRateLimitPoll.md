# Tasks.MayRateLimitPoll method

Returns `true` when a server MAY rate-limit a `tasks/get` poll that arrived sooner than the most recently advertised *pollIntervalMs* (§25.7, R-25.7-o). A first poll (no prior poll) or a task with no advertised interval is never rate-limitable.

```csharp
public static bool MayRateLimitPoll(long? lastPolledAtMs, long nowMs, long? pollIntervalMs)
```

| parameter | description |
| --- | --- |
| lastPolledAtMs | Epoch ms of the previous poll, or `null` for the first poll. |
| nowMs | The current time in epoch ms. |
| pollIntervalMs | The most recently advertised `pollIntervalMs`, or `null`. |

## Return Value

`true` when the poll is eligible for rate-limiting.
