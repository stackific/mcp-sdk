# Tasks.IsTasksExtensionId method

Returns `true` only when *identifier* is byte-identical to [`ExtensionId`](./ExtensionId.md) (§25.1, R-25.1-a). The comparison is exact and case-sensitive: identifiers differing only in case or by a prefix/suffix are NON-matching.

public static bool IsTasksExtensionId(string identifier)

| parameter | description |

| identifier | The candidate extension identifier. |

## Return Value

`true` when it equals [`ExtensionId`](./ExtensionId.md) octet-for-octet.
