# Tasks.IsPollingTerminalResponse method

Returns `true` when a client should STOP polling after a `tasks/get` response: either a `-32602` error (the task is unknown/expired — terminal and unavailable) or a terminal [`DetailedTask`](../DetailedTask.md) (§25.7, §25.11, R-25.7-s, R-25.11-e). The response may be a raw error object (carrying `code`) or a detailed-task result object (carrying `status`).

```csharp
public static bool IsPollingTerminalResponse(JsonObject? response)
```

| parameter | description |
| --- | --- |
| response | The raw `tasks/get` response object (error or result). |

## Return Value

`true` when the client should stop polling.
