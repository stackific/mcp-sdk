# Tasks.ShouldContinuePolling method

Returns `true` when a client SHOULD continue polling a task: it is non-terminal AND the client has not issued `tasks/cancel` (§25.7, §25.8, R-25.7-p). After `tasks/cancel` the client MAY stop immediately — pass *cancelRequested* as `true` (R-25.9-k).

```csharp
public static bool ShouldContinuePolling(McpTaskStatus status, bool cancelRequested = false)
```

| parameter | description |
| --- | --- |
| status | The task's last observed status. |
| cancelRequested | Whether the client has already issued `tasks/cancel`. |

## Return Value

`true` when the client should keep polling.
