# Tasks.BuildTaskStatusNotification method

Builds a `notifications/tasks` notification carrying a complete [`DetailedTask`](../DetailedTask.md) for the task's current status — identical to what `tasks/get` would return at that moment, so a subscribed client need not issue an extra `tasks/get` (§25.10, R-25.10-a). The notification params are the detailed task; a server MUST NOT push for a task the client did not subscribe to (see [`MayPushTaskNotification`](./MayPushTaskNotification.md)).

```csharp
public static JsonRpcNotification BuildTaskStatusNotification(DetailedTask task)
```

| parameter | description |
| --- | --- |
| task | The task's current detailed state. |

## Return Value

The `notifications/tasks` notification.
