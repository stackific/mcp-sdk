# Tasks.MayPollNow method

Returns `true` when polling at *nowMs*, given the last poll at *lastPolledAtMs*, respects the recommended minimum interval (§25.4, R-25.4-d). A first poll (no prior poll) is always allowed.

```csharp
public static bool MayPollNow(long? lastPolledAtMs, long nowMs, long? pollIntervalMs, 
    long fallbackMs = 1000)
```

| parameter | description |
| --- | --- |
| lastPolledAtMs | Epoch ms of the previous poll, or `null` for the first poll. |
| nowMs | The current time in epoch ms. |
| pollIntervalMs | The task's `pollIntervalMs`, or `null` when absent. |
| fallbackMs | The interval used when *pollIntervalMs* is absent. |

## Return Value

`true` when enough time has elapsed to poll again.
