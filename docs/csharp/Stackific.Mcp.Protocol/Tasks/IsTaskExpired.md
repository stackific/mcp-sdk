# Tasks.IsTaskExpired method

Returns `true` when a task with a non-null *ttlMs* has expired by *nowMs* — its lifetime has elapsed since *createdAtMs*, so a server MAY discard it (§25.4, §25.6, R-25.4-c, R-25.6-f). A `null`*ttlMs* is an unbounded lifetime and never expires.

```csharp
public static bool IsTaskExpired(long createdAtMs, long? ttlMs, long nowMs)
```

| parameter | description |
| --- | --- |
| createdAtMs | The task's creation time in epoch milliseconds. |
| ttlMs | The task's `ttlMs` (a non-negative number, or `null`). |
| nowMs | The current time in epoch milliseconds. |

## Return Value

`true` when the lifetime has elapsed.
