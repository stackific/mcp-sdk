# Tasks.ResolvePollIntervalMs method

Returns the interval, in milliseconds, a client SHOULD wait before its next `tasks/get` poll (§25.4, R-25.4-d). When the task's *pollIntervalMs* is present, that value (the recommended minimum) is returned; otherwise *fallbackMs*.

```csharp
public static long ResolvePollIntervalMs(long? pollIntervalMs, long fallbackMs = 1000)
```

| parameter | description |
| --- | --- |
| pollIntervalMs | The task's `pollIntervalMs`, or `null` when absent. |
| fallbackMs | The interval used when none is recommended (default 1000 ms). |

## Return Value

The interval in milliseconds.
