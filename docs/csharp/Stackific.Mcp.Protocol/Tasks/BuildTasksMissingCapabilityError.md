# Tasks.BuildTasksMissingCapabilityError method

Builds the `-32003` error a server returns when a Tasks method is invoked but the extension is unavailable (§25.2, R-25.2-f). The `data.requiredCapabilities` names the Tasks extension so the client can re-declare it, and `data.method` records which Tasks method triggered the rejection so the client can correlate it with the originating request.
