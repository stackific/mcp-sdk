# UiHostExtensionCapability record

The value a host advertises under the `io.modelcontextprotocol/ui` key of its `extensions` capability map (spec §26.2). A server MUST NOT declare UI associations unless a host has advertised this with a [`MimeTypes`](./UiHostExtensionCapability/MimeTypes.md) array that includes [`MimeType`](./UiResource/MimeType.md).
