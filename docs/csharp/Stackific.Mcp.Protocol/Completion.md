# Completion class

The §19.4–§19.5 normative completion helpers ported from the TypeScript SDK's `completion.ts`: the 100-item cap with truncation signalling ([`ComputeCompletion`](./Completion/ComputeCompletion.md), only emitting `total`/`hasMore` when matches are dropped), the prefix matcher, the context key-exclusion guard (R-19.2-k), and the unknown-ref / unknown-argument resolution that maps to `-32602` rather than a not-found result (R-19.5-r).
