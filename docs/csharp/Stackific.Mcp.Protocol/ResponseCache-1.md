## Remarks

Freshness is computed via [`IsFresh`](./Caching/IsFresh.md): `ttlMs = 0` entries are stored but never served fresh (RC-3). [`InvalidateByNotification`](./ResponseCache-1/InvalidateByNotification.md) evicts all entries ‚ÄĒ including paginated cursor pages keyed `method::‚Ä¶` ‚ÄĒ for every method mapped to the notification (RC-9). Scope is resolved conservatively to Private for unknown values (RC-5).

Values are supplied alongside their raw `ttlMs`/`cacheScope` hints so the cache can apply [`HasBothOrNeitherCacheHints`](./Caching/HasBothOrNeitherCacheHints.md) and [`IsCacheHintValid`](./Caching/IsCacheHintValid.md) ‚ÄĒ an entry whose hints are missing or invalid is silently skipped. This type is not thread-safe.
