# Completion.GuardContextExcludesArgument method

Enforces the R-19.2-k guard: a `context.arguments` key MUST NOT name the argument being completed. Throws `-32602` (Invalid params) when it does. A server MAY otherwise ignore `context` entirely. (§19.2, R-19.2-j, R-19.2-k)
