# Completion.PrefixMatch method

A reference prefix matcher: returns the *candidates* whose value starts with the *seed*, in input order. When the seed is empty, every candidate matches (the empty-input suggestions). Matching is case-sensitive by default. (§19.5, R-19.5-d, R-19.2-i)

```csharp
public static IReadOnlyList<string> PrefixMatch(string seed, IReadOnlyList<string> candidates, 
    bool caseInsensitive = false)
```

| parameter | description |
| --- | --- |
| seed | The current partial value (`argument.value`). |
| candidates | The full candidate pool. |
| caseInsensitive | When `true`, folds case before matching. |

## Return Value

The matching candidates, in input order.
