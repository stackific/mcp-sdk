# Completion.ResolveCompletionTarget method

Resolves a validated `ref` + `argument.name` against the server's catalog, enforcing R-19.5-r: an unknown prompt / resource template, OR an `argument.name` that is not a declared argument/variable of the referenced target, MUST be rejected with `-32602` (Invalid params) — NOT a not-found result. A KNOWN argument with no registered completer is NOT an error; it simply yields empty values. (§19.5, R-19.5-r)

A [`CompletionTargetResolution`](../CompletionTargetResolution.md); on failure it carries the `-32602` error.
