# AuthorizationServerMetadata.Parse method

Parses and structurally validates a fetched authorization-server metadata document: the REQUIRED fields, the `response_types_supported` ⊇ `code` constraint (R-23.3-i), and the `code_challenge_methods_supported` ⊇ `S256` constraint (R-23.3-j). Returns `null` on a structurally invalid document. The issuer-match check is applied separately by [`Validate`](./Validate.md).
