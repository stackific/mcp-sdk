# CancelTaskResult record

The result of a `tasks/cancel` request (spec §25.9): an empty acknowledgement. The runtime supplies the base `resultType` as `"complete"` (see [`Complete`](./ResultTypes/Complete.md)). Cancellation is cooperative and eventually consistent: acknowledgement does not guarantee the task will reach the Cancelled terminal status.
