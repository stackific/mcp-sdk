# UiVisibility enumeration

The audiences a tool with a declared UI is exposed to (spec §26.3 `visibility`). When the `visibility` array is omitted it is treated as both Model and App.

```csharp
public enum UiVisibility
```

## Values

| name | value | description |
| --- | --- | --- |
| Model | `0` | The tool is visible to, and callable by, the model/agent through the ordinary tool-calling flow (§26.3, §16). |
| App | `1` | The tool is callable by the rendered UI through the message-channel dialect, subject to host mediation and consent (§26.3, §26.5, §26.7). |
