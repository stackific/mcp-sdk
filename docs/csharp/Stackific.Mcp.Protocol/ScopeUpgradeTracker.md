# ScopeUpgradeTracker class

Tracks bounded step-up retry attempts per resource-and-operation combination, so a client retries no more than a few times and treats persistent failure as a permanent authorization failure (spec §23.18, R-23.18-q, R-23.18-r, R-23.1-af, R-23.1-ag).
