# AuthorizationScopes.WithOfflineAccess method

Adds `offline_access` to a `scope` string when, and only when, the authorization-server metadata advertises it in `scopes_supported` (R-23.9-b). Returns the scope unchanged when `offline_access` is not advertised or already present.

```csharp
public static string? WithOfflineAccess(string? scope, IReadOnlyList<string>? asScopesSupported)
```

| parameter | description |
| --- | --- |
| scope | The current `scope` string, or `null`. |
| asScopesSupported | The selected authorization server's `scopes_supported`. |

## Return Value

The scope, possibly with `offline_access` appended.
