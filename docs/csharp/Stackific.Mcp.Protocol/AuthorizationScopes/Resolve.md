# AuthorizationScopes.Resolve method

Resolves the `scope` parameter to send in the authorization request, applying the scope priority (R-23.5-f): (1) the `WWW-Authenticate` challenge's `scope`; (2) all of protected-resource `scopes_supported`; (3) otherwise `null` (omit `scope`).

```csharp
public static string? Resolve(WwwAuthenticateChallenge? challenge, 
    IReadOnlyList<string>? scopesSupported)
```

| parameter | description |
| --- | --- |
| challenge | The parsed `WWW-Authenticate` challenge, if any. |
| scopesSupported | Protected-resource `scopes_supported`, if any. |

## Return Value

The resolved scope string, or `null` to omit `scope`.
