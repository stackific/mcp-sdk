# ConformanceRequirements.ConformanceProfile record

The abstract descriptor that fully describes an implementation's conformance: the tuple of roles, advertised revisions, advertised capabilities, advertised extensions, and implemented transports (spec §29.9 item 3, story §6). NOT a wire message — it is used to reason about and report conformance.
