## Remarks

Resolved from the server's catalog so [`ResolveCompletionTarget`](./Completion/ResolveCompletionTarget.md) can detect an unknown ref or an argument that is not part of the referenced target. A [`PromptReference`](./PromptReference.md) resolves against the server's offered prompts (by name); a [`ResourceTemplateReference`](./ResourceTemplateReference.md) against the offered resource templates (by uri/uriTemplate). A target found but carrying an empty argument-name set is still "known" — only an absent target is unknown.
