# ReadResourceResponseKind enumeration

The discriminated outcome of inspecting a `resources/read` reply (spec §17.5): a completed [`ReadResourceResult`](./ReadResourceResult.md), an `input_required` continuation, or a malformed body.

## Values

| name | value | description |
| --- | --- | --- |
| Complete | `0` | A completed read carrying `contents` (the absent-⇒-complete default applies, R-17.5-q). |
| InputRequired | `1` | The server needs additional client input before the read can complete (R-17.5-w). |
| Error | `2` | The reply did not match either expected shape. |
