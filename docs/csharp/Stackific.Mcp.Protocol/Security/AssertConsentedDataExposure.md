# Security.AssertConsentedDataExposure method

Asserts that user/resource data is exposed to a server (or onward) ONLY with the user's consent (spec §28.4, R-28.4-b, R-28.1-e, R-28.1-f; AC-44.3, AC-44.11). Wraps [`EvaluateConsent`](./EvaluateConsent.md) with the `resource-exposure` operation.

```csharp
public static IsolationValidation AssertConsentedDataExposure(string scope,

```
