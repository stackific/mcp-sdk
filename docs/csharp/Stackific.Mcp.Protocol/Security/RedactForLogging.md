# Security.RedactForLogging method

Returns a copy of a JSON value intended for a log/trace/telemetry sink with credential/token values redacted (spec §28.9, R-28.9-c, R-28.9-d, R-28.9-e; AC-44.23, AC-44.17). Walks the value recursively; any property whose key names a credential/token has its value replaced with [`RedactedPlaceholder`](./RedactedPlaceholder.md). The input is never mutated.

```csharp
public static JsonNode? RedactForLogging(JsonNode? value)
```

| parameter | description |
| --- | --- |
| value | The value about to be logged. |

## Return Value

A NEW value with sensitive entries redacted.
