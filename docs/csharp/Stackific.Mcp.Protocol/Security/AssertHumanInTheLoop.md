# Security.AssertHumanInTheLoop method

Asserts the human-in-the-loop invariant for a proposed tool invocation: the user could review and understand it and the decision did not rest solely with the model (spec §28.3, R-28.3-d, R-28.3-e; AC-44.8).

```csharp
public static HumanInTheLoopValidation AssertHumanInTheLoop(bool userCouldReviewAndDeny, 
    bool modelDecidedAlone)
```

| parameter | description |
| --- | --- |
| userCouldReviewAndDeny | The user was able to review, understand, and deny before it ran (R-28.3-d). |
| modelDecidedAlone | The decision rested solely with the model, with no human gate (R-28.3-e). |

## Return Value

The validation outcome.
