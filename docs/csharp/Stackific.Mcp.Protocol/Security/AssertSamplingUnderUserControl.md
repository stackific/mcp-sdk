# Security.AssertSamplingUnderUserControl method

Asserts a server-driven sampling flow remained under user control: the MUST-level §28.7 obligations are met (human review of prompt and completion before they are acted upon or transmitted) and the host disclosed no more conversation context than the user authorized (spec §28.7, R-28.7-a, R-28.7-f, R-28.7-g; AC-44.20).

```csharp
public static UserControlValidation AssertSamplingUnderUserControl(
    SamplingConsentObligations obligations, bool promptReviewed, bool completionReviewed, 
    bool disclosedContextWithinAuthorization)
```

| parameter | description |
| --- | --- |
| obligations | The host's §21.2.10 consent-obligation claims (R-28.7-a). |
| promptReviewed | The prompt sent to the model was human-reviewed/approved (R-28.7-f). |
| completionReviewed | The completion was human-reviewed before being acted upon (R-28.7-f). |
| disclosedContextWithinAuthorization | The disclosed conversation context was within what the user authorized (R-28.7-g). |

## Return Value

The validation outcome.
