# Security.SanitizeToolOutputText method

Sanitizes a tool-output text string so a result cannot carry control sequences that would compromise the client, model, or downstream consumers (spec §28.3, R-28.3-i; AC-44.9). Strips C0/C1 control characters (excluding the ordinary whitespace `\t`, `\n`, `\r`).
