# Security.AssertUiSandboxConforming method

Asserts a server-provided UI is rendered conformingly: it runs in an isolated sandbox that denies DOM/cookies/storage/navigation, under a restrictive CSP, and exposes no credentials/tokens/unrelated context (spec §28.8, R-28.8-a, R-28.8-e, R-28.8-f, R-28.8-g; AC-44.21, AC-44.22).

```csharp
public static UserControlValidation AssertUiSandboxConforming(
    IEnumerable<string> sandboxDeniedAccess, bool restrictiveCspApplied, JsonObject exposedToUi)
```

| parameter | description |
| --- | --- |
| sandboxDeniedAccess | The categories the sandbox denies (R-28.8-a). |
| restrictiveCspApplied | Whether a restrictive content-security policy is applied (R-28.8-a). |
| exposedToUi | The data the host hands to the UI, exposure-checked (R-28.8-e). |
