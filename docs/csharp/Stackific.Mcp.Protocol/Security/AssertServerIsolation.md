# Security.AssertServerIsolation method

Asserts the §28.4 server-isolation invariant: a server receives only host-elected context, never another server's requests/results/context/credentials (spec §28.4, R-28.4-a, R-28.4-d, R-28.4-e, R-28.4-f; AC-44.11).

```csharp
public static IsolationValidation AssertServerIsolation(string destinationServerId, 
    bool hostElected, string? sourceServerId = null)
```

| parameter | description |
| --- | --- |
| destinationServerId | The server the host is about to send context to. |
| hostElected | `true` when the host deliberately elected to share this context (R-28.4-a). |
| sourceServerId | The server the context/credential came from, or `null`. |

## Return Value

The validation outcome.
