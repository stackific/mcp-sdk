# Security.ValidateResourceUriAccess method

Validates a resource URI before dereferencing or matching it: it parses as an absolute URI, its location is one the user has authorized, and (when it could trigger a network request) it is not an SSRF target (spec §28.10, R-28.10-f, R-28.10-g, R-28.10-h; AC-44.25).

```csharp
public static ResourceUriValidation ValidateResourceUriAccess(string uri, 
    Func<Uri, bool> isAuthorizedLocation, bool guardSsrf = false)
```

| parameter | description |
| --- | --- |
| uri | The resource URI to validate (R-28.10-f). |
| isAuthorizedLocation | Predicate: is this URL a location the user authorized? (R-28.10-g) |
| guardSsrf | When `true`, reject private/loopback/link-local hosts (R-28.10-h). |

## Return Value

The validation outcome.
