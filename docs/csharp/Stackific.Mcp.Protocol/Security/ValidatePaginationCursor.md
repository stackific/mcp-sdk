# Security.ValidatePaginationCursor method

Validates a pagination cursor as opaque, untrusted input: it is rejected with a `-32602` error when malformed, unknown, or expired, rather than having its attacker-controlled contents interpreted (spec §28.10, R-28.10-j; AC-44.27). The *isKnown* predicate is the server's own recognition check. An absent cursor is valid — it requests the first page.

```csharp
public static CursorValidation ValidatePaginationCursor(string? cursor, Func<string, bool> isKnown)
```

| parameter | description |
| --- | --- |
| cursor | The cursor the client supplied, or `null` for the first page. |
| isKnown | Predicate: did this server issue this cursor and is it still valid? |

The validation outcome.
