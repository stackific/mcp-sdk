# Security.ValidateAuthorizationIssuer method

Validates the exact-issuer mix-up defense for an authorization response: the client MUST have recorded the expected issuer before redirect and MUST compare any returned issuer by exact string comparison, rejecting mismatches (spec §28.5, R-28.5-h, R-28.5-i; AC-44.14).
