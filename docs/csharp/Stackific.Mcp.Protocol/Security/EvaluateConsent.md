# Security.EvaluateConsent method

The host consent gate every operation acting on the user's behalf passes before it reaches a server (spec §28.2, R-28.2-a … R-28.2-f; AC-44.2, AC-44.7). Allows the operation ONLY when it matches a prior grant for the SAME operation and scope, or the user freshly, informedly approved THIS proposal. Denies (with a reason) on no-consent, not-informed, or silent-escalation. The gate never treats a missing approval as approval.
