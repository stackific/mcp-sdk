# Security.ValidatePeerToolCall method

Validates `tools/call` arguments against a tool's declared input schema and, optionally, structured results against an output schema, reporting a failure as a `-32602` error rather than acting on the input (spec §28.10, R-28.10-a … R-28.10-e; AC-44.24). The schema check enforces the common JSON Schema constraints (`required`, top-level property `type`, `enum`), mirroring the SDK's pragmatic validator.
