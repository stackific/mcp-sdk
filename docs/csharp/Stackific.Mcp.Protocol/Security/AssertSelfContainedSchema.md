# Security.AssertSelfContainedSchema method

Asserts a tool schema is self-contained — it carries no external `$ref` that the server would have to dereference — unless external resolution is explicitly permitted against a trusted source (spec §28.10, R-28.10-m, R-28.10-n; AC-44.29). A pure structural inspection that performs no I/O. An in-document `$ref` (starting with `#`) is allowed.

```csharp
public static SchemaSelfContainmentValidation AssertSelfContainedSchema(JsonNode? schema, 
    bool allowTrustedExternalRefs = false, int? maxDepth = null)
```

| parameter | description |
| --- | --- |
| schema | The tool schema to inspect (R-28.10-m). |
| allowTrustedExternalRefs | Opt-in: external refs are resolved only against trusted sources (R-28.10-n). Default `false`. |
| maxDepth | Recursion bound; defaults to the schema-depth limit. |

The validation outcome.
