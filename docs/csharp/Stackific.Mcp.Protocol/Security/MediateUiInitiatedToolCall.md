# Security.MediateUiInitiatedToolCall method

Mediates a UI-requested `tools/call`, routing it through the host's normal consent / human-in-the-loop path; the UI can never cause a tool to run without host mediation and user consent (spec §28.8, R-28.8-b, R-28.8-c, R-28.8-d; AC-44.21). Routes only when the tool has `app` visibility AND the user consented AND host policy allows.
