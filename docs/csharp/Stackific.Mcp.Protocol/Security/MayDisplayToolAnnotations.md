# Security.MayDisplayToolAnnotations method

Returns whether a host MAY surface a tool's annotation hints to the user for THIS server (spec §28.3, R-28.3-b; AC-44.6). Displaying a hint from a trusted server is permitted; relying on it as a guarantee is not ([`ToolAnnotationIsSecurityGuarantee`](./ToolAnnotationIsSecurityGuarantee.md), R-28.3-c).

```csharp
public static bool MayDisplayToolAnnotations(bool serverIsTrusted)
```

| parameter | description |
| --- | --- |
| serverIsTrusted | Whether the host explicitly trusts the server. |

## Return Value

`true` when the annotations may be displayed.
