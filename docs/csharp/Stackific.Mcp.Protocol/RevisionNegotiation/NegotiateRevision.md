# RevisionNegotiation.NegotiateRevision method

Selects the highest mutually supported protocol revision, returning a structured outcome (§5.4, R-5.4-b). On an empty intersection the result carries both sides' revision sets so the caller can surface an [`IncompatibleProtocolError`](../IncompatibleProtocolError.md) (R-5.4-c, R-5.4-d). Mirrors TS `negotiateRevision`.

```csharp
public static RevisionNegotiationResult NegotiateRevision(IReadOnlyList<string> clientPreference, 
    IReadOnlyList<string> serverSupported)
```

| parameter | description |
| --- | --- |
| clientPreference | The client's acceptable revisions, most-preferred first. |
| serverSupported | The server's advertised revisions (order ignored). |

## Return Value

An [`RevisionNegotiationResult`](../RevisionNegotiationResult.md) describing the selection or the failure.
