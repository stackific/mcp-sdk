# RevisionNegotiation.AugmentClientCapabilities method

Produces the `clientCapabilities` object for a retry after a `MissingRequiredClientCapability` (`-32003`) error: the originally declared capabilities merged with the required ones (§5.6, R-5.6-i). The merge is shallow — a required capability replaces any previously declared value for that key — and never mutates its inputs. Mirrors TS `augmentClientCapabilities`.

```csharp
public static JsonObject AugmentClientCapabilities(JsonObject declared, 
    JsonObject requiredCapabilities)
```

| parameter | description |
| --- | --- |
| declared | The originally declared client capabilities. |
| requiredCapabilities | The capabilities the server named as required. |

## Return Value

A fresh, merged JsonObject.
