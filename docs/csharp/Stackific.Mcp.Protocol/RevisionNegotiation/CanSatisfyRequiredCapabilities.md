# RevisionNegotiation.CanSatisfyRequiredCapabilities method

Returns `true` when the client can declare every capability the server named as required — each required top-level key is one the client already declares (§5.6, R-5.6-i). The comparison is by top-level key presence; capabilities are never inferred from a prior request. Mirrors TS `canSatisfyRequiredCapabilities`.

## Return Value

`true` when the client can satisfy every required capability.
