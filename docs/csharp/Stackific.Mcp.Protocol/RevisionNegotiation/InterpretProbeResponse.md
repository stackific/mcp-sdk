Interprets a response to a probe `server/discover` request (§5.7), classifying it as Supported, UnsupportedVersion, or NotThisProtocol. Mirrors TS `interpretProbeResponse`.

## Return Value

The classified [`ProbeOutcome`](../ProbeOutcome.md).

## Remarks

A success carrying a valid `DiscoverResult` means the server speaks this protocol family. A recognized `-32004` whose `data` carries `supported` + `requested` means same family, different revision — the client re-selects from `data.supported`. Anything else (a different error code, a malformed response, or `null` for a timeout) means the server does not speak this protocol revision (R-5.7-c).
