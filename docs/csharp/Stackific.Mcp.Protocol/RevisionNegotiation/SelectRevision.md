# RevisionNegotiation.SelectRevision method

Selects a protocol revision from a server's *supportedVersions* using the client's own preference order (§5.4, R-5.3.2-d) — never the order of the server's array. The first client-preferred revision the server also supports is chosen; reordering *supportedVersions* cannot change the result. Returns `null` when the two share no revision. Mirrors TS `selectRevision`.

```csharp
public static string? SelectRevision(IReadOnlyList<string> supportedVersions, 
    IReadOnlyList<string>? clientAcceptable = null)
```

| parameter | description |
| --- | --- |
| supportedVersions | The server's advertised revisions (order ignored). |
| clientAcceptable | The client's acceptable revisions, most-preferred first. When `null`, defaults to `[ProtocolRevision.Current]`. |

## Return Value

The selected revision, or `null` when the intersection is empty.
