# RevisionNegotiation.ReselectAfterUnsupportedVersion method

Reacts to an `UnsupportedProtocolVersion` (`-32004`) error by re-selecting a revision from the error's authoritative `data.supported` set (§5.5, R-5.5-h). Because this is a pure re-selection over the server's set, an empty result is terminal — the client MUST NOT retry indefinitely (R-5.5-i) and SHOULD surface an incompatibility (R-5.5-j). Mirrors TS `reselectAfterUnsupportedVersion`.
