# ToolNames.Disambiguate method

Applies a disambiguation strategy for an aggregated tool name by prefixing the server identifier (for example `server.tool`). A client/proxy that hits a name collision SHOULD apply such a strategy. (R-16.3-h)

```csharp
public static string Disambiguate(string serverId, string name, string separator = ".")
```

| parameter | description |
| --- | --- |
| serverId | The server identifier to prefix with. |
| name | The tool's original name. |
| separator | The prefix separator (default `.`, a permitted name char). |

## Return Value

The disambiguated name.
