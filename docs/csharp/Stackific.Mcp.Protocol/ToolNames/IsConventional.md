# ToolNames.IsConventional method

Returns `true` when a tool `name` follows the recommended conventions: 1–128 characters, only `A–Z a–z 0–9 _ - .` (so no spaces/commas/other special characters). Names SHOULD be treated case-sensitively. (§16.3, R-16.3-b, R-16.3-c, R-16.3-d, R-16.3-e)

```csharp
public static bool IsConventional(string name)
```

| name | The candidate tool name. |

## Return Value

`true` when the name follows the conventions.
