# ElicitationForm.IsLegacyTitledEnumSchema method

Returns `true` when *value* is the Deprecated legacy enum form (a string `enum` carrying the non-standard `enumNames` parallel array). Useful for a conformance check that new functionality does not adopt it, while a legacy schema received from a peer is still accepted (spec §20.4, R-20.4-f).
