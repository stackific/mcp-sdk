# ElicitationForm.ClassifyEnumSchema method

Classifies an enum schema into one of its five structural forms by the distinguishing keyword, or returns `null` when *value* is not a well-formed enum schema (spec §20.4).

```csharp
public static EnumSchemaForm? ClassifyEnumSchema(JsonNode? value)
```

| parameter | description |
| --- | --- |
| value | The candidate enum schema. |

## Return Value

The classified form, or `null`.

## Remarks

Classification order resolves overlaps:

* `type: "array"` ⇒ multi-select; `items.anyOf` ⇒ titled, `items.enum` ⇒ untitled.
* `type: "string"` with `oneOf` ⇒ titled single-select.
* `type: "string"` with `enum` + `enumNames` ⇒ legacy titled.
* `type: "string"` with `enum` (no `enumNames`) ⇒ untitled single-select.

`enumNames` is the deciding marker for the Deprecated legacy form (R-20.4-f).
