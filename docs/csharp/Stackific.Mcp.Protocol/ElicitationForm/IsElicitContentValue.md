# ElicitationForm.IsElicitContentValue method

Returns `true` when *value* is a permitted `content` VALUE: a string, number, boolean, or array of strings — the only value types a form-mode `content` map may carry (spec §20.5, R-20.5-c). Objects, `null`, and mixed arrays are rejected.
