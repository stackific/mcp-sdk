# ElicitationForm.AssertFormModeMayCollect method

Asserts that a form-mode `requestedSchema` does not request sensitive credential data — the §20.7 prohibition (spec §20.7, R-20.7-h, R-20.7-i). Returns `Ok` when no sensitive fields are detected, otherwise names the offending fields; the server MUST then use URL mode for those interactions instead (R-20.7-i).
