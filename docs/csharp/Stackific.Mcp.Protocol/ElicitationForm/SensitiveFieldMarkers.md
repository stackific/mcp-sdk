# ElicitationForm.SensitiveFieldMarkers property

Heuristic markers for sensitive credential fields a server MUST NOT request via form mode (passwords, API keys, access tokens, payment credentials). Matched against a lower-cased field name / `title` / `description` (spec §20.7, R-20.7-h). This is a best-effort guard, not an exhaustive list; servers remain responsible for routing sensitive interactions to URL mode (R-20.7-i).
