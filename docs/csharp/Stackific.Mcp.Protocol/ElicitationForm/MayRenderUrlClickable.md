# ElicitationForm.MayRenderUrlClickable method

Returns `true` when a URL MAY be rendered as a clickable link for the given field, enforcing the §20.7 rule that ONLY the `url` field of a URL-mode request is clickable; no other field of any elicitation request may be (spec §20.7, R-20.7-r, R-20.7-y).

```csharp
public static bool MayRenderUrlClickable(string fieldName, string mode)
```

| parameter | description |
| --- | --- |
| fieldName | The field the URL would be rendered in. |
| mode | The mode of the elicitation request. |

## Return Value

`true` when the URL may be clickable.
