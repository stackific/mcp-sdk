# ElicitationForm.ExtractDefaults method

Extracts the per-field `default` values declared in a restricted form schema, so a defaults-supporting client can pre-populate the corresponding fields (spec §20.4, R-20.4-c). Returns a map from field name to its declared `default` node (cloned), including only the fields that declare one. A client that supports defaults SHOULD use these; one that does not MAY ignore them.
