# ElicitationForm.ClassifyPrimitiveSchema method

Classifies a property schema by the `PrimitiveSchemaDefinition` member it selects, or returns `null` when it is not a valid primitive schema (spec §20.4). Selection is structural: boolean by `type`; number for `"number"`/`"integer"`; enum for a string/array schema carrying `enum`/`oneOf`/`items`; otherwise string for a plain `"string"`.
