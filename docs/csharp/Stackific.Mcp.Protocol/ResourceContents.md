# ResourceContents record

The concrete contents of a resource (spec §14.5). A value is the text variant if and only if it carries [`Text`](./ResourceContents/Text.md), and the blob variant if and only if it carries [`Blob`](./ResourceContents/Blob.md); a value MUST NOT carry both (R-14.5-h). Use [`OfText`](./ResourceContents/OfText.md)/[`OfBlob`](./ResourceContents/OfBlob.md) to construct a well-formed variant.
