# UriTemplate.Variables method

Extracts the variable names referenced by a URI template's `{…}` expressions, in first-seen order with duplicates removed. Useful for driving completion (§19) or prompting the user for values before expansion. (§17.4, R-17.4-n)

```csharp
public static IReadOnlyList<string> Variables(string template)
```

| parameter | description |
| --- | --- |
| template | The URI template to inspect. |

## Return Value

The distinct variable names, in first-seen order.

## Remarks

Returns an empty list for a literal-only template. The leading operator and any `*`/`:N` modifiers are stripped from the reported names. Does NOT itself validate the template; pair it with [`IsUriTemplate`](./IsUriTemplate.md) when validation is required.
