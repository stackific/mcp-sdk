# UriTemplate.Expand method

Expands a URI template against a set of variable values, producing a concrete URI (RFC 6570 Levels 1–4). A client substitutes the named `{…}` variables to form a `resources/read` URI. (§17.4, R-17.4-m, R-17.4-n)

```csharp
public static string Expand(string template, UriTemplateValues values)
```

| parameter | description |
| --- | --- |
| template | The RFC 6570 template. |
| values | The variable bindings. |

## Return Value

The expanded concrete URI.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | When the template is malformed (unbalanced braces). |

## Remarks

Supports the simple expansion (no operator) and the reserved (`+`), fragment (`#`), label (`.`), path-segment (`/`), path-style (`;`), query (`?`), and query-continuation (`&`) operators, including the `*` explode (over list values) and `:N` prefix modifiers. An undefined variable contributes nothing (per the RFC). Scalar values are looked up in *values*; a value MAY be a single string or a list of strings (via [`UriTemplateValues`](../UriTemplateValues.md)).
