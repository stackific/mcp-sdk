# UriTemplate.IsResourceUri method

Returns `true` when *value* is a string usable as a concrete `Resource.uri` [RFC 3986]: it carries a conformant scheme and parses as an absolute URI. A relative reference (no scheme) or an empty string is rejected, because a concrete resource URI MUST identify the resource uniquely. (§17.4, R-17.4-a, R-17.4-b)

```csharp
public static bool IsResourceUri(string? value)
```

| parameter | description |
| --- | --- |
| value | The candidate URI string. |

## Return Value

`true` when the value is a non-empty absolute URI with a scheme.

## Remarks

Mirrors the TS implementation, which combines a scheme-grammar pre-check with the WHATWG `URL` parser (absolute URIs only). Here the .NET Uri parser plays the role of the WHATWG parser, so values such as `urn:isbn:0451450523` (empty authority) and `custom-scheme:thing` are accepted consistently.
