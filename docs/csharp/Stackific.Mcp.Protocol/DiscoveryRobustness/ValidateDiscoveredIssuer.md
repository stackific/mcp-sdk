# DiscoveryRobustness.ValidateDiscoveredIssuer method

Validates that a fetched authorization-server metadata document's `issuer` is IDENTICAL to the issuer used to construct the well-known URL; if it differs the document MUST NOT be used (R-23.17-h, R-23.17-i). Exact string comparison — the same mix-up defence as [`Validate`](../AuthorizationServerMetadata/Validate.md), surfaced for callers that have already structurally validated the document.
