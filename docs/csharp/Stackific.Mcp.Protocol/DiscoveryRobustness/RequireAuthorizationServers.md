# DiscoveryRobustness.RequireAuthorizationServers method

Validates that protected-resource metadata carries the REQUIRED `authorization_servers` array of one or more issuer identifiers (R-23.17-c). On success *authorizationServers* is populated. When more than one is listed, each is independent and the client maintains separate registration state per AS (R-23.17-d, via [`IssuerBoundCredentialStore`](../IssuerBoundCredentialStore.md)).
