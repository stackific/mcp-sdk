# DiscoveryRobustness.ProtectedResourceMetadataUris method

Resolves the ordered protected-resource-metadata URIs to try, honouring the `WWW-Authenticate``resource_metadata` precedence (R-23.17-a, R-23.17-b): the `401`'s `resource_metadata` URL MUST be used when present; otherwise the well-known URIs (path-prefixed first, then host root).

```csharp
    string? mcpEndpointUrl)
```

| parameter | description |
| --- | --- |
| resourceMetadataUrl | The `resource_metadata` URL from the `401`'s header, if any. |
| mcpEndpointUrl | The MCP endpoint URL, used to build the well-known fallbacks. |

The ordered candidate URIs (possibly empty).
