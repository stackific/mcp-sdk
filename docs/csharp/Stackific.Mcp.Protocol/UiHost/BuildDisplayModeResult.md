# UiHost.BuildDisplayModeResult method

Applies a `ui/request-display-mode` request: the host MAY grant a mode different from the one requested, and the result reports the mode actually applied (spec §26.5.3, R-26.5.3-e). Returns the result params object carrying the applied mode.

```csharp
public static JsonObject BuildDisplayModeResult(string applied)
```
