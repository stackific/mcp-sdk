# UiHost.DedicatedRenderOrigin method

Returns the dedicated render origin a UI resource declared via its `domain` UI-metadata field (spec §26.4, R-26.4-l), or `null` when none was declared. A host SHOULD render such a resource under its own isolated origin so it cannot reach the cookies/storage of other UI surfaces.

```csharp
public static string? DedicatedRenderOrigin(ResourceUiMeta? meta)
```

| meta | The resource's UI metadata, or `null`. |

## Return Value
