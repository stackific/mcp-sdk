# UiHost.ForbiddenUiExposureKeys property

The keys a host MUST NOT expose to the UI: credentials, authorization tokens (§23), and unrelated conversation/context data (spec §26.7, R-26.7-m). This list is illustrative of the categories a host must withhold; the authoritative rule is the inclusion test [`UiExposureIsClean`](./UiExposureIsClean.md), which keys off the allow-list rather than this deny-list.
