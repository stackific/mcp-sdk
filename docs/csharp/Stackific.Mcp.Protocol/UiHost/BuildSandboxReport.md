# UiHost.BuildSandboxReport method

Builds the `hostCapabilities.sandbox` report for the initialize result as a JSON object: the EFFECTIVE CSP the host applied and the GRANTED permission set (spec §26.7, R-26.7-g, R-26.7-h). The CSP is resolved via [`ResolveCsp`](../Ui/ResolveCsp.md) (declared `csp`, else deny-by-default), and the permissions via [`GrantedPermissions`](./GrantedPermissions.md).

```csharp
public static JsonObject BuildSandboxReport(UiContentSecurityPolicy effectiveCsp, 
    UiPermissions granted)
```

| parameter | description |
| --- | --- |
| effectiveCsp | The effective CSP the host applied; reported verbatim under `sandbox.csp`. |
| granted | The granted permission set; reported verbatim under `sandbox.permissions`. |

## Return Value

The sandbox report object.
