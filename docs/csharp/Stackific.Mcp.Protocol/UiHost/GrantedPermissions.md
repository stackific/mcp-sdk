# UiHost.GrantedPermissions method

Computes the GRANTED permission set for a UI resource, enforcing R-26.7-h: the host MUST NOT grant any permission the resource did not request, and MAY decline a requested one (spec §26.7, R-26.7-h). The result is exactly what `hostCapabilities.sandbox.permissions` reports. Starts from the resource's requested set, keeps only members the resource requested, and drops any the host chose to decline.

```csharp
public static UiPermissions GrantedPermissions(UiPermissions? requested, 
    IEnumerable<UiPermissionName>? declined = null)
```

| parameter | description |
| --- | --- |
| requested | The resource's declared `permissions` (S41), or `null`. |
| declined | The permission names the host declines (the host's own R-26.7-h choice); members not requested are ignored. |

## Return Value

The granted permissions.
