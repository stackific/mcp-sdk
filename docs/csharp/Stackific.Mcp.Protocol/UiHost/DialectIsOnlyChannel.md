# UiHost.DialectIsOnlyChannel method

Returns `true` when the §26.5 dialect channel is the ONLY path granted between the rendered UI and the host — i.e. no other ambient path to host or user data exists (spec §26.7, R-26.7-c).

```csharp
public static bool DialectIsOnlyChannel(IEnumerable<string> grantedPaths)
```

| parameter | description |
| --- | --- |
| grantedPaths | The set of paths the host grants the UI to reach host/user data. |

## Return Value

`true` when only the dialect channel is granted.
