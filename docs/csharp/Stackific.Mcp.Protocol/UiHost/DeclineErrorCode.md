# UiHost.DeclineErrorCode method

Maps a [`DeclineReason`](../UiHost.DeclineReason.md) to the §22 error code a host returns when it declines a UI-initiated request (spec §26.8, R-26.8-b): `unknown-method` → `-32601`, `invalid-params` → `-32602`, `no-consent`/`policy` → `-32603`. Whichever reason applies, the host MUST return an error — never a silent drop.

```csharp
public static int DeclineErrorCode(DeclineReason reason)
```

| reason | The decline reason. |
