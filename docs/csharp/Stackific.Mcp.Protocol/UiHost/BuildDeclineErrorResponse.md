# UiHost.BuildDeclineErrorResponse method

Builds the §22 error response a host returns when it DECLINES a UI-initiated request, instead of silently dropping it (spec §26.8, R-26.8-b). The code is selected from *reason* by [`DeclineErrorCode`](./DeclineErrorCode.md).

```csharp
public static JsonRpcErrorResponse BuildDeclineErrorResponse(RequestId id, DeclineReason reason, 
    string? message = null)
```

| parameter | description |
| --- | --- |
| id | The request id being declined. |
| reason | Why the host declined. |
| message | An optional human-readable message; defaults to a reason-derived message. |

## Return Value

The decline error response.
