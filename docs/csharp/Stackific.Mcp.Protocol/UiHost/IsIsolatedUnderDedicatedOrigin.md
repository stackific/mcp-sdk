# UiHost.IsIsolatedUnderDedicatedOrigin method

Returns `true` when a UI is isolated under its declared dedicated origin (spec §26.4, R-26.4-l): either no *declaredDomain* was declared (no isolation constraint applies), or the *renderOrigin* equals the declared domain AND is not shared with any other UI surface in *otherUiOrigins*. Origins are compared byte-for-byte (never case-folded).

```csharp
public static bool IsIsolatedUnderDedicatedOrigin(string? declaredDomain, string renderOrigin, 
    IEnumerable<string> otherUiOrigins)
```

| parameter | description |
| --- | --- |
| declaredDomain | The resource's declared dedicated origin, or `null` when none. |
| renderOrigin | The origin the host actually rendered the UI under. |
| otherUiOrigins | The render origins of the host's other UI surfaces. |

## Return Value

`true` when the isolation requirement is satisfied (or not applicable).
