# UiHost.BuildDialectErrorResponse method

Builds a JSON-RPC error response for a failed dialect request, per §3 and §22 (spec §26.8, R-26.8-a). Reuses the foundation's [`JsonRpcError`](../../Stackific.Mcp.JsonRpc/JsonRpcError.md) / [`JsonRpcErrorResponse`](../../Stackific.Mcp.JsonRpc/JsonRpcErrorResponse.md) so the `error` shape is the single authoritative one.
