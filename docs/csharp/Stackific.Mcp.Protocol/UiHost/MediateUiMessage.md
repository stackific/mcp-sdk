# UiHost.MediateUiMessage method

Decides whether a host may honor a `ui/message` insertion. The host SHOULD confirm with the user before inserting the message into the conversation (spec §26.7, R-26.7-l). Same gate shape as [`MediateOpenLink`](./MediateOpenLink.md).

```csharp
public static MediationDecision MediateUiMessage(bool hostHonors, bool userConfirmed)
```

| parameter | description |

| userConfirmed | Whether the user confirmed inserting the message. |
