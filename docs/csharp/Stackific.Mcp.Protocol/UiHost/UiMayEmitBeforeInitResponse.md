# UiHost.UiMayEmitBeforeInitResponse method

Returns `true` when a conforming UI MAY emit a dialect message with *method* BEFORE it has received the `ui/initialize` response. Only `ui/initialize` itself qualifies; every other dialect message — including `ui/notifications/initialized` — MUST wait for the response (spec §26.5.1, R-26.5.1-a).

```csharp
public static bool UiMayEmitBeforeInitResponse(string method)
```

| parameter | description |
| --- | --- |
| method | The method/notification name the UI intends to send. |

## Return Value

`true` when permitted before the init response.
