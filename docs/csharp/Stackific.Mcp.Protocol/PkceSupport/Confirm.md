# PkceSupport.Confirm method

Confirms, from *codeChallengeMethodsSupported*, that the AS supports PKCE with the `S256` method (R-28.5-k). Support is confirmable ONLY when the field is present AND includes `S256`; an absent field means support is unconfirmable (the client MUST refuse).
