# PkceSupport.AssertConfirmed method

Asserts PKCE `S256` support is confirmable from AS metadata, throwing [`PkceSupportException`](../PkceSupportException.md) when it is not — so the client refuses to proceed rather than starting a flow against an AS that may not support PKCE (R-28.5-k).
