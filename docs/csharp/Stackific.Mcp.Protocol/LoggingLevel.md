# LoggingLevel enumeration

A structured log severity (spec §15.3.1), mapping to the standard syslog message severities. The members are declared from least to most severe; that ordering is significant because a request opts in at a minimum level and the emitter MUST emit only messages at or above it.

```csharp
public enum LoggingLevel
```

## Values

| name | value | description |
| --- | --- | --- |
| Debug | `0` | Detailed debugging information (lowest severity). |
| Info | `1` | General informational messages. |
| Notice | `2` | Normal but significant events. |
| Warning | `3` | Warning conditions. |
| Error | `4` | Error conditions. |
| Critical | `5` | Critical conditions. |
| Alert | `6` | Action must be taken immediately. |
| Emergency | `7` | System is unusable (highest severity). |

## Remarks

Deprecated [SEP-2577]: the logging-message mechanism of §15.3 is Deprecated and retained only for interoperability with peers that emit it.
