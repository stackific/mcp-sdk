# WwwAuthenticate.ChallengedScopes method

Resolves the scopes a client MUST treat as required for the request from a challenge (R-23.1-x, R-23.1-y). The challenged scope set is authoritative; this derives the required scopes SOLELY from the challenge's `scope`, never from `scopes_supported`. Returns an empty list when the challenge carried no `scope`.
