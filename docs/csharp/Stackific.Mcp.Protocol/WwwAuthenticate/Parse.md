# WwwAuthenticate.Parse method

Parses a `WWW-Authenticate` header value carrying a `Bearer` challenge into its structured fields (R-23.1-z). A client MUST be able to parse `WWW-Authenticate` headers and react to a `401`; this is that parser.

```csharp
public static WwwAuthenticateChallenge? Parse(string headerValue)
```

| parameter | description |
| --- | --- |
| headerValue | The raw `WWW-Authenticate` header value. |

## Return Value

The parsed challenge, or `null` when not a Bearer challenge.

## Remarks

Accepts the auth-param forms RFC 7235 permits — quoted (`key="value"`) and bare (`key=value`) — comma-separated, with arbitrary surrounding whitespace, unescaping `\"`/`\\` inside quoted values. The scheme match is case-insensitive. Returns `null` when the value does not use the `Bearer` scheme.
