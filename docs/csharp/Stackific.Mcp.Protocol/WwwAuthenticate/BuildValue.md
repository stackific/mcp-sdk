# WwwAuthenticate.BuildValue method

Builds the `WWW-Authenticate` header value for a `Bearer` challenge from its structured fields (R-23.1-u – R-23.1-w, R-23.1-ab – R-23.1-ad). Parameters are emitted in a stable order — `error`, `scope`, `resource_metadata`, `error_description` — each only when present; the `Bearer` scheme always leads.
