# WwwAuthenticate.BuildInsufficientScopeValue method

Builds an MCP server's `403 Forbidden` insufficient-scope `WWW-Authenticate` header value carrying `error="insufficient_scope"`, the `scope`, and `resource_metadata` (R-23.1-ab), with an OPTIONAL `error_description` (R-23.1-ad) (spec §23.1, R-23.1-aa – R-23.1-ad). The caller SHOULD pass the union of all scopes the operation needs so this is a single, complete challenge (R-23.1-ac).

public static string BuildInsufficientScopeValue(string scope, string resourceMetadata, 
    string? errorDescription = null)
