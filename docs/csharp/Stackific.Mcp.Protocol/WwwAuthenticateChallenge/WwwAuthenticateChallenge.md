# WwwAuthenticateChallenge constructor

The structured fields of a `Bearer``WWW-Authenticate` challenge (spec §23.1) — the parameter set carried in the HTTP response header, not a JSON object.

```csharp
public WwwAuthenticateChallenge(string? ResourceMetadata = null, string? Scope = null, 
    string? Error = null, string? ErrorDescription = null)
```

| parameter | description |
| --- | --- |
| ResourceMetadata | Absolute URI of the protected-resource metadata document (R-23.1-v). |
| Scope | Space-delimited scopes required for the operation (R-23.1-w, R-23.1-ab). |
| Error | The failure code; `insufficient_scope` on a `403` (R-23.1-ab). |
| ErrorDescription | OPTIONAL human-readable description of the failure (R-23.1-ad). |

## Remarks

On a `401` (§7.4) [`ResourceMetadata`](./ResourceMetadata.md) is REQUIRED and [`Scope`](./Scope.md) SHOULD be present; on a `403` insufficient-scope challenge (§7.5) [`Error`](./Error.md) is `insufficient_scope` and [`Scope`](./Scope.md), [`ResourceMetadata`](./ResourceMetadata.md), and an OPTIONAL [`ErrorDescription`](./ErrorDescription.md) accompany it.
