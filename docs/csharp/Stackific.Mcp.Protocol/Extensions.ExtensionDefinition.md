# Extensions.ExtensionDefinition record

A declarative description of the surface a single extension contributes — the machine-checkable form of "an active extension MAY extend the surface ONLY in the four enumerated ways" (spec §24.5). A conformance suite validates a claimed surface with [`ValidateDefinition`](./Extensions/ValidateDefinition.md).
