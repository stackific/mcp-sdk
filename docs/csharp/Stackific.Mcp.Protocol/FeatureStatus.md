# FeatureStatus enumeration

Feature lifecycle status (spec §27, R-1.3-b, R-2.2-f – R-2.2-h). A feature whose status is Deprecated remains defined and MUST still be accepted by receivers, but SHOULD NOT be relied upon by new implementations.

public enum FeatureStatus

## Values

| name | value | description |
| --- | --- | --- |
| Active | `0` | The feature is current and may be relied upon. |
| Deprecated | `1` | The feature is deprecated: still accepted by receivers, but discouraged for new use. |
