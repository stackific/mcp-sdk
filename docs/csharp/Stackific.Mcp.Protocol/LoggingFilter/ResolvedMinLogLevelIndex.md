# LoggingFilter.ResolvedMinLogLevelIndex method

Returns the minimum numeric severity index that should be emitted for a request bearing *logLevelOptIn*, or `-1` when no valid opt-in is present — in which case NO log notifications MUST be emitted. Mirrors TS `resolvedMinLogLevelIndex`. (R-15.3.3-a)
