# LoggingFilter.ValidateLogLevelOptIn method

Validates the `io.modelcontextprotocol/logLevel` opt-in value from a request's `_meta`: returns success when it is a recognized level string, or a `-32602` error when it is not (a number, an unknown string, or `null` are all rejected). Mirrors TS `validateLogLevelOptIn`. (R-15.3.3-g)
