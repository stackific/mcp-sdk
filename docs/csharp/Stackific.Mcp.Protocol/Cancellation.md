# Cancellation class

The §15.2 request-cancellation machinery — the C# counterpart of the cancellation utilities in the TypeScript `protocol/progress.ts` module. Cancellation is a best-effort, same-direction-only mechanism: a party MAY cancel only requests IT issued, never requests it received, and races are tolerated. The pieces here are:

* [`CancellationHandler`](./CancellationHandler.md) — receiver-side registry of abort callbacks (R-15.2.2-d).
* [`CancelledRequestSet`](./CancelledRequestSet.md) — sender-side set of cancelled ids whose late responses are ignored (R-15.2.3-e).
* [`ValidateCancellationTarget`](./Cancellation/ValidateCancellationTarget.md) — own-in-flight + `server/discover` exclusion (R-15.2.1-a/b, R-15.2.2-b).
* [`IsDiscoverMethod`](./Cancellation/IsDiscoverMethod.md) — the discover-not-cancellable guard.
