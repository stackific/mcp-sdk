# Security.SecurityBaselineClaims constructor

A host's self-assertion that it addresses each of the four §28.1 core principles (spec R-28.1-a; AC-44.1).

```csharp
public SecurityBaselineClaims(bool UserConsentAndControl, bool DataPrivacy, bool ToolSafety, 
    bool HostMediatedTrust)
```

| parameter | description |
| --- | --- |
| UserConsentAndControl | Users explicitly consent to and control all data access/operations (R-28.1-b, R-28.1-c). |
| DataPrivacy | A server receives only host-elected context; no transmission without consent (R-28.1-e, R-28.1-f). |
| ToolSafety | Tools are treated as arbitrary code; definitions/annotations are untrusted (R-28.1-h, R-28.1-i). |
| HostMediatedTrust | Trust is mediated and enforced at the host, never delegated to a server (§28.1(4)). |
