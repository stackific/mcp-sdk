# ConformanceRequirements.StatelessConformanceInvariants constructor

The stateless-model invariants that bind every role (spec §29.7, R-29.7-a – R-29.7-e). Each is unconditionally `true`; exposed as a flat checklist a conformance harness can assert against.

```csharp
public StatelessConformanceInvariants(bool IndependentRequests, bool ExplicitCrossRequestState, 
    bool ConnectionIsNotLifetimeBoundary, bool RequestStateIsUntrusted, 
    bool RequestStateIntegrityProtected)
```

| parameter | description |
| --- | --- |
| IndependentRequests | Each request is processed independently; no context inferred from an earlier one (R-29.7-a). |
| ExplicitCrossRequestState | Cross-request state rides an explicit client-supplied identifier/opaque value (R-29.7-b). |
| ConnectionIsNotLifetimeBoundary | The connection/process is NOT the lifetime boundary of a conversation/task/subscription (R-29.7-c). |
| RequestStateIsUntrusted | A requestState passing through a client is attacker-controlled input (R-29.7-d). |
| RequestStateIntegrityProtected | A security-significant requestState is integrity-protected; failed verification is rejected (R-29.7-e). |
