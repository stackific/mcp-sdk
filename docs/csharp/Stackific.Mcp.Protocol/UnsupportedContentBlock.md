## Remarks

The original wire object is retained on [`Raw`](./UnsupportedContentBlock/Raw.md) (including the unknown `type` and any extension fields) and is re-emitted byte-faithfully on serialisation. This type is produced only by [`ContentBlockConverter`](./ContentBlockConverter.md); it is never used for the five known types and is never produced for the forbidden sampling types, which are rejected instead.
