# PaginationUtilities class

The §12 cursor-based pagination behavioral layer — the C# counterpart of the TypeScript `protocol/pagination.ts` module. A `cursor` is an OPAQUE string: the empty string `""` is a PRESENT cursor (not absence), and only the absence of a cursor (`null`) means "first page" / "last page". This class adds the cursor predicates, the per-cursor cache-key helper (R-12.5-a), the invalid-cursor error builder (`-32602`), and the reference [`OffsetPaginator`](./OffsetPaginator-1.md) that NEVER throws on a bad cursor.
