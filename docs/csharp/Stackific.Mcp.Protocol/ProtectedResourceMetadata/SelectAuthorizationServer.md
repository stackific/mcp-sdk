# ProtectedResourceMetadata.SelectAuthorizationServer method

Selects one authorization-server `issuer` from this document (R-23.2-j). By default the first listed issuer is chosen; an optional *prefer* predicate lets a caller impose its own selection policy (the first issuer for which *prefer* returns `true` wins, falling back to the first listed). Returns `null` only for an empty list (which a valid document never has).

```csharp
public string? SelectAuthorizationServer(Func<string, bool>? prefer = null)
```

| parameter | description |
| --- | --- |
| prefer | OPTIONAL predicate selecting a preferred issuer. |

## Return Value

The selected issuer, or `null` when none is listed.
