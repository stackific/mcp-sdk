# ProtectedResourceMetadata.Validate method

Validates a fetched protected-resource metadata document against the MCP server it is contacting (spec §23.2, R-23.2-h, R-23.2-i, R-23.2-j): structurally valid, and `resource` equals the server's canonical resource identifier (accepting uppercase scheme/host per R-23.1-p). On success *metadata* is populated.
