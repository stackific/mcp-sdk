# ToolChoiceMode enumeration

Controls how the model uses tools during sampling (spec §21.2.5). The default behavior when the request omits `toolChoice` is `{ "mode": "auto" }`.

```csharp
public enum ToolChoiceMode
```

## Values

| name | value | description |
| --- | --- | --- |
| Auto | `0` | The model decides whether to use tools. This is the default (spec §21.2.5). |
| Required | `1` | The model MUST use at least one tool before completing (spec §21.2.5). |
| None | `2` | The model MUST NOT use any tools (spec §21.2.5). |
