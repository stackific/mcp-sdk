# SubscriptionRegistry class

Routes incoming subscription notifications to the correct active [`Subscription`](./Subscription.md) by `io.modelcontextprotocol/subscriptionId` — essential on stdio where all subscriptions share one channel, and supported on HTTP where the key is still present (§10.4, R-10.4-c, R-10.4-d). Holds NO state across connections; closing a subscription removes it. The C# counterpart of the TypeScript `SubscriptionRegistry`.
