# Prompts class

The §18.4 normative prompt-response helpers ported from the TypeScript SDK's `prompts.ts`: the `resultType` discrimination that tolerates an ABSENT discriminator (absent ⇒ complete) and recognises the `input_required` continuation, plus the required-argument extractor.
