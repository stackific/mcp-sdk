# CimdHosting.IsPreferredPath method

Returns `true` when both a client and an authorization server should prefer CIMD as the registration path — both SHOULD support the mechanism (R-23.12-a).


| parameter | description |
| --- | --- |
| clientSupportsCimd | Whether the client implements CIMD. |
| serverSupportsCimd | Whether the AS advertises `client_id_metadata_document_supported: true`. |
