## Remarks

This union is separate from [`ContentBlock`](./ContentBlock.md) (§14.4): it adds the sampling-only `tool_use` ([`ToolUseContent`](./ToolUseContent.md)) and `tool_result` ([`ToolResultContent`](./ToolResultContent.md)) blocks and EXCLUDES `resource_link`/`resource`. It belongs to the Deprecated Sampling capability (spec §21.2); implementations SHOULD NOT adopt it for new functionality and SHOULD instead integrate directly with a model provider. It remains defined for interoperability.
