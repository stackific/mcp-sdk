## Remarks

This record models the wire shape only; the §14.2 security validation (scheme allowlist, magic-byte MIME detection, credential-free fetch) lives on [`IconSecurity`](./IconSecurity.md) and is applied deliberately before an advertised icon is fetched or rendered. Unknown fields are preserved in [`Extensions`](./Icon/Extensions.md) for forward-compatibility (the TS schema's `.passthrough()`).
