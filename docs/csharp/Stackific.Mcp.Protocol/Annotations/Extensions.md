# Annotations.Extensions property

Unknown, forward-compatible fields preserved verbatim from the wire (the TS `.passthrough()`). Absent when no extension fields were present.

## Remarks

This is NOT a JsonExtensionDataAttribute property: [`Annotations`](../Annotations.md) carries a type-level [`AnnotationsConverter`](../AnnotationsConverter.md) that reads/writes unknown members by hand (so it can enforce the [`Priority`](./Priority.md) range), and STJ rejects a JsonNode-valued extension dictionary anyway.
