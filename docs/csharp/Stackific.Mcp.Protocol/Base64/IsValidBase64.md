# Base64.IsValidBase64 method

Returns `true` when *s* contains only valid Base64 characters (including optional `=` padding), accepting both the standard (`+/`) and URL-safe (`-_`) alphabets so the SDK stays interoperable (R-14.5-f, R-14.4.2-b, R-14.4.3-b). An empty string is valid.

```csharp
public static bool IsValidBase64(string s)
```

| parameter | description |
| --- | --- |
| s | The candidate Base64 string. |

## Return Value

`true` when every character is in the (extended) Base64 alphabet.
