# ToolSchemas.IsInDocumentRef method

Returns `true` when a `$ref` / `$dynamicRef` value resolves WITHIN the same schema document — a document-local JSON Pointer (`#`, `#/…`) or a plain-name fragment anchor (`#anchor`). An absolute or relative URI naming another document is NOT in-document. (§16.4(5), R-16.4-f)
