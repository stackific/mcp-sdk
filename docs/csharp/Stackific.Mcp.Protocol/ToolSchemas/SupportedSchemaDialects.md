# ToolSchemas.SupportedSchemaDialects property

The complete set of JSON Schema dialects this implementation can validate against (§16.4(9), R-16.4-s, R-16.4-u): ONLY JSON Schema 2020-12 — both the canonical `https://json-schema.org/draft/2020-12/schema` and its `#`-suffixed spelling. No dialect beyond 2020-12 is supported; a tool schema declaring any other `$schema` (for example Draft-07) is rejected by [`IsSupportedSchemaDialect`](./IsSupportedSchemaDialect.md) / [`ValidateToolSchema`](./ValidateToolSchema.md) rather than silently treated as permissive (R-16.4-t).
