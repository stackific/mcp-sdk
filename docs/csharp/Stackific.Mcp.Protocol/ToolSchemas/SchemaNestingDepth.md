# ToolSchemas.SchemaNestingDepth method

Returns the maximum nesting depth of a schema document (objects + arrays). Counting stops at *cap* so a pathologically deep value cannot exhaust the stack. A leaf object such as `{"type":"object"}` has depth 1. (§16.4(6), R-16.4-l)

```csharp
public static int SchemaNestingDepth(JsonNode? node, int cap = 65)
```

| parameter | description |
| --- | --- |
| node | The schema value. |
| cap | A hard recursion cap; the returned depth never exceeds it. |

## Return Value

The schema's nesting depth, bounded by *cap*.
