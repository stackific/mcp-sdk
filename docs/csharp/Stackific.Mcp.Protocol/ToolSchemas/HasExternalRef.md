# ToolSchemas.HasExternalRef method

Walks a schema document and returns `true` when any `$ref` / `$dynamicRef` targets a location OUTSIDE the document. Such a reference MUST NOT be dereferenced or fetched over network or file system; this pure inspection never performs I/O (so it cannot itself trigger an SSRF fetch) — it only reports the presence of an external reference so callers can reject it. (§16.4(5), R-16.4-f, R-16.4-g, R-16.4-k, R-16.4-r)

```csharp
public static bool HasExternalRef(JsonNode? node, int maxDepth = 64)
```

| parameter | description |
| --- | --- |
| node | The schema (or sub-schema) to inspect. |
| maxDepth | A recursion bound so a pathological schema cannot exhaust the stack. |

## Return Value

`true` when an external reference is present.
