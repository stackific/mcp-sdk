# ContinuationTokenStore&lt;TState&gt;.Validate method

Validates a presented continuation token, returning the protected state on success or a structured rejection (spec R-28.6-b, R-28.6-c). A receiver MUST reject (never act on) a token that fails integrity; replay (expiry or re-use) is refused too. A successful validation consumes the single-use token.
