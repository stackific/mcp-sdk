# ConformanceRequirements.Level enumeration

A normative requirement level in the RFC 2119 / RFC 8174 sense (spec §2). The `MUST` family also covers MUST NOT / REQUIRED / SHALL / SHALL NOT; `SHOULD` also covers SHOULD NOT / RECOMMENDED; `MAY` also covers OPTIONAL.
