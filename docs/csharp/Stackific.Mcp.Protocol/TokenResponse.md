# TokenResponse record

The token-endpoint JSON response (spec §23.5 Step 4, §23.9). [`AccessToken`](./TokenResponse/AccessToken.md) and [`TokenType`](./TokenResponse/TokenType.md) (`Bearer`) are REQUIRED; the rest are OPTIONAL — a client MUST NOT assume a refresh token will be issued (R-23.9-d).
