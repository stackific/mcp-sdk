# UriTemplate class

RFC 3986 resource-URI and RFC 6570 URI-Template validation, variable extraction, expansion, and matching for the Resources feature (spec §17.4). This is the C# counterpart of the TypeScript SDK's `isResourceUri` / `isUriTemplate` / `uriTemplateVariables` helpers, plus the expand/match machinery the server runtime uses to turn a concrete `resources/read` URI back into the variables of a registered template.

## Remarks

The validation surface intentionally mirrors the TS grammar checks exactly so the two SDKs accept and reject the same strings:

* [`IsResourceUri`](./UriTemplate/IsResourceUri.md) — an RFC 3986 absolute URI with a conformant scheme (a relative reference with no scheme is rejected, R-17.4-a/b).
* [`IsUriTemplate`](./UriTemplate/IsUriTemplate.md) — the full RFC 6570 grammar: literals interspersed with balanced `{…}` expressions, each an OPTIONAL leading operator from `+#./;?&` followed by one or more comma-separated `varspec`s, where a `varspec` is a `varname` with an OPTIONAL `*` (explode) or `:N` (prefix, 1–9999) modifier (R-17.4-m).
* [`Variables`](./UriTemplate/Variables.md) — the referenced variable names in first-seen order with duplicates removed, operators and modifiers stripped (R-17.4-n).

[`Expand`](./UriTemplate/Expand.md) performs Level 1–4 expansion and [`TryMatch`](./UriTemplate/TryMatch.md) compiles a template into a capturing matcher that recovers each variable's value from a concrete URI — replacing the previous trivial `{name}`-only regex in the server runtime.
