# AccessTokenUsage.ValidateTokenAudience method

Validates, on the MCP server side, that a presented token was issued for THIS server as the intended audience, rejecting any token whose audience is some other resource (R-23.6-f, R-23.6-g, R-23.6-h). Accepts a single audience OR an array, comparing each by canonical resource identity (uppercase scheme/host tolerant, trailing-slash tolerant; R-23.1-p).

public static AuthorizationResult ValidateTokenAudience(IReadOnlyList<string> tokenAudiences,
