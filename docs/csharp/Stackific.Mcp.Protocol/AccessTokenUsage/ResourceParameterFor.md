# AccessTokenUsage.ResourceParameterFor method

Returns the `resource` parameter value for the MCP server — its canonical resource identifier — that MUST be sent in BOTH the authorization and token requests, regardless of whether the AS advertises `resource` support (R-23.6-b – R-23.6-e). Surfaced as a named helper so the "always send it" rule is explicit.
