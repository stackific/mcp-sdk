# AccessTokenUsage.SelectTokenForServer method

Selects the access token a client may send to a given MCP server — strictly the one issued by that server's authorization server for that server, and no other (R-23.6-i). Returns a failed outcome (and *accessToken*`null`) when no matching token exists, so the client sends nothing rather than a wrong-audience token.

```csharp
public static AuthorizationResult SelectTokenForServer(string serverIssuer, 
    string serverCanonicalResource, string tokenIssuer, IReadOnlyList<string> tokenAudiences, 
    string candidateAccessToken, out string? accessToken)
```

| parameter | description |
| --- | --- |
| serverIssuer | The issuer of the server's authorization server. |
| serverCanonicalResource | The server's canonical resource id. |
| tokenIssuer | The issuer that minted the candidate token. |
| tokenAudiences | The candidate token's audience(s). |
| candidateAccessToken | The candidate access token. |
| accessToken | On success, the access token to send; otherwise `null`. |

## Return Value

The selection outcome.
