# AccessTokenUsage.ExtractBearerToken method

Extracts the bearer token from an `Authorization` header value, or `null` when the header is absent or does not use the `Bearer` scheme (R-23.8-b). The scheme match is case-insensitive per RFC 7235.

```csharp
public static string? ExtractBearerToken(string? headerValue)
```

| parameter | description |
| --- | --- |
| headerValue | The raw `Authorization` header value, if any. |

## Return Value

The extracted token, or `null`.
