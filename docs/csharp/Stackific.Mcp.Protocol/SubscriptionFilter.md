# SubscriptionFilter record

The notification kinds a client opts in to on a `subscriptions/listen` stream (spec §10.2). All fields are OPTIONAL; an omitted/false field means "not subscribed". The server echoes the subset it will honor in the acknowledgement (§10.3).
