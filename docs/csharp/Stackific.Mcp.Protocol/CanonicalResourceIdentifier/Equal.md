# CanonicalResourceIdentifier.Equal method

Compares *a* and *b* as canonical resource identifiers, accepting an uppercase scheme/host on either side (R-23.1-p). Path, query, and port are compared case-sensitively; only scheme and host are case-insensitive. Returns `false` when either side is not a valid identifier.
