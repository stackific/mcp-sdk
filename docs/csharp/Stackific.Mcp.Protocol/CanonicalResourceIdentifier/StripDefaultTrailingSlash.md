# CanonicalResourceIdentifier.StripDefaultTrailingSlash method

Returns *uri* with a single trailing slash removed when the slash is not semantically significant (R-23.1-s). A path of just `"/"` (the bare-host root) is left intact.

```csharp
public static string StripDefaultTrailingSlash(string uri, bool slashIsSignificant = false)
```

| parameter | description |
| --- | --- |
| uri | The candidate URI. |
| slashIsSignificant | When `true`, the trailing slash is preserved. |
