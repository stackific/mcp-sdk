# ElicitRequestURLParams record

URL-mode elicitation parameters (spec §20.3, `mode: "url"`): out-of-band interaction via navigation to a URL. Data other than the URL itself is NOT exposed to the client, making this the mode a server MUST use for sensitive flows such as authorization or payment (§20.7).
