## Remarks

All fields are OPTIONAL; an absent or empty [`Annotations`](./Annotations.md) object is valid. Unknown fields are preserved in [`Extensions`](./Annotations/Extensions.md) for forward-compatibility (the TS schema's `.passthrough()`). The (de)serialiser enforces the [`Priority`](./Annotations/Priority.md) 0..1 range (R-14.6-d).
