# SamplingMessage.Content property

REQUIRED. The message content (spec §21.2.6). On the wire this is either a single content block or an array of content blocks; this SDK models it as an array (a single-block message is the one-element case). Per §21.2.7, a `user` message containing tool results MUST contain ONLY [`ToolResultContent`](../ToolResultContent.md) blocks.
