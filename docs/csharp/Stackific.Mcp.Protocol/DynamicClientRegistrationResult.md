# DynamicClientRegistrationResult record

The outcome of a DCR registration attempt, modelling the failure cases a client MUST be prepared to handle (spec §23.4, R-23.4-p, R-23.4-q, R-23.4-r). On failure the client surfaces [`Reason`](./DynamicClientRegistrationResult/Reason.md) rather than crashing; [`Retryable`](./DynamicClientRegistrationResult/Retryable.md) flags redirect-URI/application-type rejections the client MAY retry.
