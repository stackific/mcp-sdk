# MultiRoundTrip.BuildMissingCapabilityError method

Builds the `-32003` error a server returns when it cannot complete without an input-request kind the client did not declare (§11.5, R-11.5-i, R-11.5-j). On HTTP the response status MUST be 400. The `data.requiredCapabilities` names the unsupported capabilities.
