# MultiRoundTrip.ClientSupportsInputRequestKind method

Returns `true` when the client declared support for the capability an input-request *method* requires (§11.2, §11.5, R-11.2-j, R-11.5-a). Used both server-side (may a server emit a kind) and client-side (may a client fulfill it). An unrecognized method is never supported.

```csharp
public static bool ClientSupportsInputRequestKind(string method, JsonObject clientCapabilities)
```

| parameter | description |
| --- | --- |
| method | The input-request method name. |
| clientCapabilities | The client's declared capabilities as a raw object. |

## Return Value

`true` when the required capability is declared.
