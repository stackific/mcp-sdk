# MultiRoundTrip.IsLoadSheddingResult method

Returns `true` when *result* is a load-shedding signal: `resultType` is `input_required`, `inputRequests` is absent or empty, and `requestState` is present (§11.5, R-11.5-l). A client MUST NOT treat this as an error; it MAY retry echoing `requestState`, applying backoff on repeated non-progress.

| result | The raw result object. |

## Return Value

`true` when it is a load-shedding signal.
