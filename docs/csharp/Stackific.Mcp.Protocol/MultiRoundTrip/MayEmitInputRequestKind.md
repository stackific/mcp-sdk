# MultiRoundTrip.MayEmitInputRequestKind method

Server-side gate: returns `true` when the server MAY emit an input-request of *method* given the client's declared capabilities (§11.2, §11.5, R-11.2-j, R-11.5-g). A server MUST NOT emit a kind the client has not declared.

```csharp
public static bool MayEmitInputRequestKind(string method, JsonObject clientCapabilities)
```

| parameter | description |
| --- | --- |
| method | The input-request method name. |
| clientCapabilities | The client's declared capabilities as a raw object. |

## Return Value

`true` when the server may emit the kind.
