# MultiRoundTrip.ValidateInputResponseKinds method

Validates that each value in *inputResponses* conforms to the expected response shape for the `InputRequest` kind sent under the same key (§11.4, R-11.4-e, R-11.4-f): `elicitation/create` → an `action`; `roots/list` → a `roots` array; `sampling/createMessage` → `role`, `content`, and `model`. Keys with no matching request are skipped (caught by [`ValidateInputResponseKeys`](./ValidateInputResponseKeys.md)); unrecognized request kinds are skipped. A mismatch lets a server reject the retry with a JSON-RPC error (R-11.5-s).

## Return Value
