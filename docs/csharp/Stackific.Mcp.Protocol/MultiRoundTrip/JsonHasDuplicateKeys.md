# MultiRoundTrip.JsonHasDuplicateKeys method

Scans raw JSON text for a duplicate object member name. JsonDocumentOptions) silently collapses duplicate keys (last-wins), so duplicate detection MUST work on the raw token stream — this tracks the member names seen within each object scope and reports the first repeat (§11.2, R-11.2-f).
