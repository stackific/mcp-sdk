# UiContentSecurityPolicy record

A content-security-policy descriptor for a UI resource (spec §26.4). Each member lists origin strings; an origin not present in the applicable member MUST be blocked by the host. When the whole [`Csp`](./ResourceUiMeta/Csp.md) is omitted the host applies a deny-by-default policy (§26.7).
