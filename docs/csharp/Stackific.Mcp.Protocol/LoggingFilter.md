# LoggingFilter class

The §15.3 logging-filter behavioral layer — the C# counterpart of the TypeScript `protocol/logging.ts` helpers — for the Deprecated logging-message mechanism. The [`LoggingLevel`](./LoggingLevel.md) enum (and its [`LoggingLevelExtensions`](./LoggingLevelExtensions.md) ordering) carries the severities; this class adds the per-request opt-in validation (`-32602` on a bad value, R-15.3.3-g), the min-emit-level resolution (absent/invalid ⇒ emit nothing, R-15.3.3-a), and the global [`LogRateLimiter`](./LogRateLimiter.md).
