# Extensions.IsReservedBareVendorPrefix method

Returns `true` when a vendor prefix is one of the bare reserved tokens `modelcontextprotocol` or `mcp` (a single-label prefix with no dot) (spec R-24.2-f). This is distinct from [`IsReservedPrefix`](./IsReservedPrefix.md), which reserves a prefix whose second label is reserved; a bare single-label prefix has no second label, so that check alone would miss `modelcontextprotocol/x` and `mcp/x`.

```csharp
public static bool IsReservedBareVendorPrefix(string prefix)
```

| parameter | description |
| --- | --- |
| prefix | The candidate vendor prefix. |

## Return Value

`true` when the prefix is a bare reserved token.
