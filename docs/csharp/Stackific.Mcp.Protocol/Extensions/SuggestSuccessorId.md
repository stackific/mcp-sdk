# Extensions.SuggestSuccessorId method

Suggests a successor extension identifier for an incompatible change, keeping the two distinct in the negotiation map (spec R-24.6-d), e.g. `com.example/my-extension → com.example/my-extension-2`. The suffix is appended to the name segment so the result is itself a well-formed identifier under the same vendor prefix.

```csharp
public static string SuggestSuccessorId(string identifier, string suffix = "2")
```

| parameter | description |
| --- | --- |
| identifier | The current extension identifier. |
| suffix | The successor suffix (default `2`). |

## Return Value

The successor identifier.
