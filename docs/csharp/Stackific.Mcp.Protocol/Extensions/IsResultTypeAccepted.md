# Extensions.IsResultTypeAccepted method

Returns `true` when *resultType* is accepted: a core value, or a value contributed by an extension in the active set (spec R-24.5-e, R-24.5-f). A value that is neither is INVALID — the receiver MUST treat the response as an error.

```csharp
public static bool IsResultTypeAccepted(string resultType, IEnumerable<string> activeSet, 
    IReadOnlyDictionary<string, IEnumerable<string>>? activeContributions = null)
```

| parameter | description |
| --- | --- |
| resultType | The discriminator value. |
| activeSet | Identifiers active for this interaction. |
| activeContributions | Map of extension identifier → the `resultType` values it contributes. |

## Return Value

`true` when accepted.
