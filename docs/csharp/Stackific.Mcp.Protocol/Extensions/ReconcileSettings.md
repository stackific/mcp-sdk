# Extensions.ReconcileSettings method

Reconciles the settings a peer advertised for *identifier* on each side (spec R-24.3-g). Returns `null` when the extension is not advertised by BOTH peers (it is not active, so there is nothing to reconcile). Each side's settings are returned as-is; the extension itself decides how to combine them.

```csharp
public static ReconciledSettings? ReconcileSettings(JsonObject? clientExtensions, 
    JsonObject? serverExtensions, string identifier)
```

| parameter | description |
| --- | --- |
| clientExtensions | The client's advertised `extensions` map (raw). |
| serverExtensions | The server's advertised `extensions` map (raw). |
| identifier | The extension whose settings to reconcile. |

## Return Value

The reconciled settings, or `null` when not active on both sides.
