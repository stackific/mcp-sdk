# Extensions.BuildRequiredExtensionError method

Builds an actionable error for the case where an implementation genuinely requires an extension the other side does not advertise (spec R-24.7-d, R-24.7-e). The error identifies the required extension in both the message and the payload so the failure is not opaque.

public static RequiredExtensionError BuildRequiredExtensionError(string identifier)
