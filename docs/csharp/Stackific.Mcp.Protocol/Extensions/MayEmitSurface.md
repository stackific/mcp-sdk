# Extensions.MayEmitSurface method

Returns `true` when an extension MAY emit its surface in the current interaction: it is present in *activeSet* (spec R-24.1-c, R-24.3-e, R-24.5-c). Extensions are disabled by default — a peer MUST NOT emit a method, notification, reserved `_meta` key, `resultType` value, or field defined by an extension this predicate reports as not active.

```csharp
public static bool MayEmitSurface(string identifier, IEnumerable<string> activeSet)
```

| parameter | description |
| --- | --- |
| identifier | The extension identifier. |
| activeSet | The identifiers active for this interaction. |

## Return Value

`true` when the extension may emit surface.
