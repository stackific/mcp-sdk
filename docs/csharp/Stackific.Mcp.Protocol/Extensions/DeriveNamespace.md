# Extensions.DeriveNamespace method

Derives the method namespace prefix an extension owns from its identifier's NAME segment (spec R-24.5-b). The Tasks extension `io.modelcontextprotocol/tasks` defines methods such as `tasks/get` — the namespace is the identifier's extension-name followed by `/`. Returns `null` when the identifier is malformed or its name is empty.

```csharp
public static string? DeriveNamespace(string identifier)
```

| identifier | The extension identifier. |

## Return Value

The namespace (e.g. `tasks/`), or `null`.
