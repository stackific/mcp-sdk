# Extensions.GetSettings method

Returns the settings object a peer advertised for *identifier*, or `null` when the extension is not validly advertised (absent, `null`, or malformed) (spec R-6.5-h, R-6.5-j). The returned object MAY contain keys the receiving extension does not define; those MUST be ignored — use [`PickKnownSettings`](./PickKnownSettings.md) to project to known keys.
