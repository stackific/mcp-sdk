# Extensions.IsValidMap method

Returns `true` when *map* is a valid producer-built `extensions` map: every value is a settings object and no value is the JSON `null` literal (spec R-6.5-i). A receiver processing an untrusted map should instead call [`NormalizeMap`](./NormalizeMap.md).

```csharp
public static bool IsValidMap(JsonObject? map)
```

| parameter | description |
| --- | --- |
| map | The candidate `extensions` map. |

## Return Value

`true` when every value is a settings object.
