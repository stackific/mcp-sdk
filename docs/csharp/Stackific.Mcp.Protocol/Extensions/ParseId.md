# Extensions.ParseId method

Splits an extension identifier at its FIRST slash into prefix and name (spec R-6.5-a). Returns `null` when the string contains no slash at all — an identifier without a separating slash has no prefix and is malformed. Later slashes are retained in the name so [`IsValidName`](./IsValidName.md) rejects them.

public static ParsedExtensionId? ParseId(string identifier)

## Return Value

The parsed parts, or `null` when there is no slash.
