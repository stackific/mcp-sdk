# Extensions.IsActive method

Returns `true` when extension *identifier* is ACTIVE between two peers — i.e. both peers validly advertise it (spec R-6.5-l). A peer MUST NOT exercise an extension's behavior unless this returns `true`.

```csharp
public static bool IsActive(string identifier, JsonObject? clientExtensions, 
    JsonObject? serverExtensions)
```

| parameter | description |
| --- | --- |
| identifier | The extension identifier. |
| clientExtensions | The client's advertised `extensions` map (raw). |
| serverExtensions | The server's advertised `extensions` map (raw). |

## Return Value

`true` when active on both sides.
