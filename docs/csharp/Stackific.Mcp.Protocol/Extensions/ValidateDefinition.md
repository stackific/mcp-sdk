# Extensions.ValidateDefinition method

Validates that an [`ExtensionDefinition`](../Extensions.ExtensionDefinition.md) conforms to the §24 framework: a valid identifier, namespaced methods, controlled `_meta` keys, and no redefinition of core surface (spec R-24-a, R-24.5-b, R-24.5-d, R-24.5-e, R-24.5-i). Accumulates ALL violations (except when the identifier itself is invalid — then namespaces cannot be derived, so it reports and stops).
