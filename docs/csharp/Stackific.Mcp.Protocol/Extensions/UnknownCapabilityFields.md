# Extensions.UnknownCapabilityFields method

Returns the capability fields in *caps* that *known* does not recognize, in document order (spec R-6.6-b, R-6.6-c, R-6.6-f). A receiver MUST ignore exactly these fields and MUST NOT reject the capability object because they are present.

