# Extensions.IsExtensionControlledMetaKey method

Returns `true` when *metaKey* is a reserved `_meta` key that the extension identified by *identifier* is entitled to define — i.e. the key's prefix labels are the same dot-separated labels as the extension identifier's vendor prefix (spec R-24.5-d). For `io.modelcontextprotocol/ui` the controlled keys are those under `io.modelcontextprotocol/…`; for `com.example/x`, under `com.example/…`. A bare `_meta` key controls no namespace.

```csharp
public static bool IsExtensionControlledMetaKey(string metaKey, string identifier)
```

| parameter | description |
| --- | --- |
| metaKey | The candidate reserved `_meta` key. |
| identifier | The extension identifier. |

## Return Value

`true` when the extension controls the key.
