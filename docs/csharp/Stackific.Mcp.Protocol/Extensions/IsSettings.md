# Extensions.IsSettings method

Returns `true` when *value* is a non-`null`, non-array JSON object — the only legal shape for an extension settings value (spec R-6.5-h). An empty object `{}` qualifies (a valid enabling declaration, not absence). A JSON `null` literal does NOT qualify (R-6.5-i).

```csharp
public static bool IsSettings(JsonNode? value)
```

| parameter | description |
| --- | --- |
| value | The candidate settings node. |

## Return Value

`true` when the node is a JSON object.
