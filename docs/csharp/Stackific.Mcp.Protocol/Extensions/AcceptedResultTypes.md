# Extensions.AcceptedResultTypes method

Returns the set of `resultType` values a receiver will accept for an interaction: the core values together with every value contributed by an extension in *activeContributions* that is also in *activeSet* (spec R-24.5-e). Contributions from a NON-active extension are excluded (R-24.5-f).

```csharp
public static ISet<string> AcceptedResultTypes(IEnumerable<string> activeSet, 
    IReadOnlyDictionary<string, IEnumerable<string>>? activeContributions = null)
```

| parameter | description |
| --- | --- |
| activeSet | Identifiers active for this interaction. |
| activeContributions | Map of extension identifier → the `resultType` values it contributes. |

## Return Value

The accepted `resultType` set.
