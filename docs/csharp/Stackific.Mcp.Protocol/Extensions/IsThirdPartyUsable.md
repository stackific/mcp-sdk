# Extensions.IsThirdPartyUsable method

Returns `true` when a THIRD PARTY may define an extension under *identifier* per the S11 second-label rule — the identifier must be well-formed and its prefix must not be reserved (spec R-6.5-g). A malformed identifier is not third-party usable. (This is the lexical S11 check; the full S38 policy in [`ValidateThirdPartyId`](./ValidateThirdPartyId.md) additionally rejects the bare reserved tokens.)

```csharp
public static bool IsThirdPartyUsable(string identifier)
```

| parameter | description |
| --- | --- |
| identifier | The extension identifier. |

## Return Value

`true` when a third party may use it.
