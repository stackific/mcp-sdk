# Extensions.DecideFallback method

Decides how to handle an operation given whether the extension is active and whether it is mandatory (spec R-6.5-l, R-6.5-n). A peer MUST NOT reject merely because the extension is one-sided; rejection happens only when the extension is mandatory.

```csharp
public static FallbackDecision DecideFallback(bool active, bool mandatory)
```

| parameter | description |
| --- | --- |
| active | Whether the extension is advertised by both peers. |
| mandatory | Whether the extension is mandatory for the operation. |

## Return Value

The fallback decision.
