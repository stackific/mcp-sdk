# Extensions.ActiveSetForRequest method

Computes the active set for ONE request under the stateless model: it reads the client's capabilities from the request being processed and intersects them with the server's advertised capabilities (spec R-24.4-a, R-24.4-b, R-24.4-c). The result depends solely on *requestClientExtensions* and *serverExtensions*; nothing from a prior request is consulted — a request that does not advertise an extension yields an active set without it.

```csharp
public static IReadOnlyList<string> ActiveSetForRequest(JsonObject? requestClientExtensions, 
    JsonObject? serverExtensions)
```

| parameter | description |
| --- | --- |
| requestClientExtensions | This request's advertised client `extensions` map (raw; `null` ⇒ none). |
| serverExtensions | The server's advertised `extensions` map (raw). |

## Return Value

The per-request active set.
