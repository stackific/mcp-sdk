# Extensions.IsValidId method

Returns `true` when *identifier* is a well-formed extension identifier: a REQUIRED prefix, a single separating slash, and a (possibly empty) name, each conforming to the §6.5 grammar (spec R-6.5-a, R-6.5-b, R-6.5-e, R-6.5-f). Well-formedness is independent of whether the prefix is reserved.
