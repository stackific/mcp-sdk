# Extensions.IsAdvertised method

Returns `true` when a receiver should treat *identifier* as ADVERTISED by a peer whose raw `extensions` map is *raw* — the key is present and maps to a valid (non-`null`, object) settings value (spec R-6.5-h, R-6.5-j). A `null`-valued or otherwise-malformed entry is treated as not advertised.
