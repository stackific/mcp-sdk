# Tasks class

Pure, side-effect-free helpers for the Tasks extension (spec §25): the case-sensitive extension identifier and its matching rule (§25.1), the five-state lifecycle and its legal transitions (§25.5), inline-outcome consistency (§25.5), `ttlMs` expiry / backstop eligibility (§25.4, §25.6, §25.11), polling-cadence decisions (§25.7), the protocol-vs-application error classification (§25.11), the `notifications/tasks` push gating and builder (§25.10), and the task-error builders/codes (§22). The C# counterpart of the exported functions in the TypeScript `protocol/tasks.ts` (S39) and the lifecycle helpers of `protocol/tasks-lifecycle.ts` (S40) that operate on the task model. The model records ([`McpTask`](./McpTask.md), [`DetailedTask`](./DetailedTask.md), …) live alongside this class.
