# TraceContext.RelayTraceContext method

Copies the three trace-context keys present in *inbound* onto a fresh copy of *outbound*, UNCHANGED, for intermediary relay. Only keys present in *inbound* are copied; existing same-named outbound values are overwritten so the inbound values propagate unchanged; *outbound* is NOT mutated. Mirrors TS `relayTraceContext`. (R-15.4.2-h)

```csharp
public static JsonObject RelayTraceContext(JsonObject inbound, JsonObject outbound)
```

| parameter | description |
| --- | --- |
| inbound | The inbound `_meta` carrying trace context. |
| outbound | The outbound `_meta` to relay onto. |

## Return Value

A new object merging *outbound* with the relayed keys.
