# TraceContext.ExtractTraceContext method

Extracts only the string trace-context keys from *meta*, returning an object that contains at most `traceparent`, `tracestate`, and `baggage`. A non-tracing receiver can safely ignore the result. Values are copied verbatim (not parsed). Mirrors TS `extractTraceContext`. (R-15.4.2-c/g)
