# ProgressNotificationParams record

Parameters of the `notifications/progress` notification (spec §15.1.3), sent by the party that is processing a request to report incremental progress on it. The notification is request-scoped: it MUST reference a token the peer supplied in an active request's `_meta.progressToken`, and MUST be sent before that request's final response (§15.1.4).
