# ConformanceRequirements.ConformanceRequirement constructor

One normative requirement ("atom") of §29/§30, identified by its stable id, its subsection, the role(s)/axis it binds, and its RFC 2119 level. A conformance harness enumerates these to know exactly what to check.

```csharp
public ConformanceRequirement(string Id, string Section, string Keyword, Level Level, Axis Axis, 
    IReadOnlyList<Role> Roles, string Statement)
```

| parameter | description |
| --- | --- |
| Id | The stable requirement id, e.g. `R-29.2-h`. |
| Section | The §29/§30 subsection, e.g. `29.2`. |
| Keyword | The RFC 2119 keyword exactly as the story marks it. |
| Level | The canonical level family derived from *Keyword*. |
| Axis | Which conformance axis the requirement constrains. |
| Roles | The role(s) the requirement binds; empty ⇒ binds every role. |
| Statement | A one-line restatement of the obligation. |
