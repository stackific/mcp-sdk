# ResourceContentsConverter class

(De)serialiser for [`ResourceContents`](./ResourceContents.md) that enforces the §14.5 variant rules on read: a value MUST carry exactly one of `text` or `blob` (R-14.5-d, R-14.5-g), MUST NOT carry both (R-14.5-h), and any `blob` MUST be valid Base64 (R-14.5-f). Writing is the default record projection.
