# ContentBlockConverter class

The custom polymorphic (de)serialiser for [`ContentBlock`](./ContentBlock.md). It dispatches on the case-sensitive `type` field, tolerates unknown types as [`UnsupportedContentBlock`](./UnsupportedContentBlock.md) (R-14.4-b), and rejects the forbidden sampling types (R-14.8-a, R-14.8-b).
