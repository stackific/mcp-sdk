# SamplingValidation.ValidateUserToolResultExclusivity method

Validates the §21.2.7 content constraint on a single `user` message: when a `user` message contains any `tool_result` block, it MUST contain ONLY `tool_result` blocks — mixing with text/image/audio (or any other type) is NOT allowed (R-21.2.7-a). Non-`user` messages, and `user` messages without tool results, are unconstrained. Mirrors TS `validateUserToolResultExclusivity`.
