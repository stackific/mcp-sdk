# SamplingValidation.MayServerSendSamplingRequest method

Server-side gate: returns `true` only when the server MAY send the given sampling params to a client with *clientCaps* (R-21.2.3-a). A server MUST NOT invoke sampling at all unless the client declared `sampling`, MUST NOT send a tool-enabled request to a client lacking `sampling.tools`, and MUST NOT use a Deprecated `includeContext` value without `sampling.context` (R-21.2.3-c, R-21.2.4-e — via [`MayUseIncludeContext`](../CapabilityNegotiation/MayUseIncludeContext.md)). Mirrors TS `mayServerSendSamplingRequest`.
