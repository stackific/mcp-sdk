# SamplingValidation.ClampToMaxTokens method

Clamps a produced token count to the request's `maxTokens` upper bound. The client MAY sample fewer (R-21.2.4-i) but MUST NOT exceed `maxTokens` (R-21.2.4-j). Mirrors TS `clampToMaxTokens`.
