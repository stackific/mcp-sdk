# SamplingValidation.SelectFirstHintMatch method

Selects the first [`ModelHint`](../ModelHint.md) whose `name` is a substring of a candidate model name, honoring the order-sensitive first-match rule (R-21.2.9-b/f). Hints are advisory; the caller makes the final selection. Mirrors TS `selectFirstHintMatch`.

```csharp
public static ModelHintMatch? SelectFirstHintMatch(IReadOnlyList<ModelHint>? hints, 
    IReadOnlyList<string> availableModels)
```

| parameter | description |
| --- | --- |
| hints | The ordered hints from [`ModelPreferences`](../ModelPreferences.md), or `null`. |
| availableModels | The candidate model names the client can run. |

## Return Value

The first matching hint/model pair, or `null` when none matches.
