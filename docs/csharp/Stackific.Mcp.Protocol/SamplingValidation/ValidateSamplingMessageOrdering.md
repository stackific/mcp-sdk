# SamplingValidation.ValidateSamplingMessageOrdering method

Validates the §21.2.7 ordering/matching constraint across a `messages` sequence: every `assistant` message containing one or more [`ToolUseContent`](../ToolUseContent.md) blocks MUST be followed IMMEDIATELY by a `user` message consisting ENTIRELY of [`ToolResultContent`](../ToolResultContent.md) blocks, with each tool use matched by a corresponding result (by id), before any other message; multiple parallel tool uses are permitted (R-21.2.7-b). Also enforces the per-message exclusivity rule (R-21.2.7-a) on each `user` message. Mirrors TS `validateSamplingMessageOrdering`.
