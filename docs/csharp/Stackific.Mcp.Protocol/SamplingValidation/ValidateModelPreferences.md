# SamplingValidation.ValidateModelPreferences method

Validates the `ModelPreferences` numeric priorities: each, when present, MUST be a number in the inclusive range 0..1 (R-21.2.9-e). Returns the first out-of-range field, or `null` when all priorities are in range (or absent). Mirrors the TS `PrioritySchema` bound.
