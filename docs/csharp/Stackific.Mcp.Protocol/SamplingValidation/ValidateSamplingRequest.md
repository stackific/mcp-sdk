# SamplingValidation.ValidateSamplingRequest method

Full client-side validation of an inbound sampling request: structural minimum (non-empty `messages`, present `maxTokens`) plus the `ModelPreferences` priority bounds and the tool-use capability gate (R-21.2.4-a/h, R-21.2.9-e, R-21.2.3-b, R-21.2.4-n/o). Returns [`Allowed`](../SamplingGateResult/Allowed.md) on success, or a rejection carrying the `-32602` error. Mirrors TS `validateSamplingRequest` (deserialization already enforces the required-field shape; this re-checks the same constraints plus the gate).
