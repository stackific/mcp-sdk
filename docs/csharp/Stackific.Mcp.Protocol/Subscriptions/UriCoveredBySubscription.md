# Subscriptions.UriCoveredBySubscription method

Returns `true` when *updatedUri* is covered by *subscribedUri* — either an exact match or a sub-resource of a subscribed CONTAINER URI (the updated URI MAY be a descendant) (§10.5, R-10.5-j). Container matching is path-prefix based after a normalized scheme+host compare: `file:///dir` covers `file:///dir/file.txt`. A bare prefix that is not a path boundary (`file:///dir` vs `file:///directory`) does NOT match.

public static bool UriCoveredBySubscription(string updatedUri, string subscribedUri)

| subscribedUri | A subscribed container or exact URI. |

## Return Value

`true` when covered.
