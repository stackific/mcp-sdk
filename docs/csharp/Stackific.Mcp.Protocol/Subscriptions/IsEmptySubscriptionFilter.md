# Subscriptions.IsEmptySubscriptionFilter method

Returns `true` when the filter requests no kinds at all — every boolean is absent/`false` and both `resourceSubscriptions` and `taskIds` are absent/empty (§10.2, R-10.2-k). Such a filter yields an acknowledgement-only stream.
