# Subscriptions.MayDeliverResourceUpdate method

Returns `true` when a `notifications/resources/updated` for *updatedUri* is permitted on a subscription whose acknowledged `resourceSubscriptions` are *subscribedUris* — i.e. the URI (or a parent container) was listed (§10.2, §10.5, R-10.2-l, R-10.5-h). A server MUST NOT send an update for an unlisted resource.

```csharp
public static bool MayDeliverResourceUpdate(string updatedUri, 
    IReadOnlyCollection<string>? subscribedUris)
```

| parameter | description |
| --- | --- |
| updatedUri | The URI of the updated resource. |
| subscribedUris | The acknowledged subscribed URIs. |

## Return Value

`true` when delivery is permitted.
