# Subscriptions.IsViolationOnSubscriptionStream method

Returns `true` when receiving notification *method* on a SUBSCRIPTION stream is a protocol violation — i.e. it is a request-scoped (progress/logging) kind, which MUST NOT appear there (§10.6, R-10.6-b, R-10.6-e, R-10.6-g).

```csharp
public static bool IsViolationOnSubscriptionStream(string method)
```

## Return Value

`true` when it is a violation on a subscription stream.
