# Subscriptions.IsViolationOnRequestStream method

Returns `true` when receiving notification *method* on an unrelated request's response stream is a protocol violation — i.e. it is one of the four change kinds, which MUST NOT appear on a non-`subscriptions/listen` response stream (§10.6, R-10.6-d, R-10.6-f, R-10.6-g).

```csharp
public static bool IsViolationOnRequestStream(string method)
```

| parameter | description |
| --- | --- |
| method | The notification method. |

## Return Value

`true` when it is a violation on a request stream.
