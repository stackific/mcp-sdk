# Subscriptions.MayEmitChangeNotification method

Returns `true` when the server MAY emit the change notification *method* on a subscription stream whose acknowledged filter is *acknowledged* (§10.5, R-10.5-l). For `notifications/resources/updated`, pass the updated URI as *subjectKey*; for `notifications/tasks`, pass the task id (§25.10). A kind is emittable only when its filter field is reflected in the acknowledged filter.

```csharp
public static bool MayEmitChangeNotification(string method, SubscriptionFilter acknowledged, 
    string? subjectKey = null)
```

| parameter | description |
| --- | --- |
| method | The notification method. |
| acknowledged | The acknowledged honored filter. |
| subjectKey | The per-resource URI or per-task id, when the kind is keyed. |

## Return Value

`true` when emission is permitted.
