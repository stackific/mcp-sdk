# OffsetPaginator&lt;T&gt;.GetPage method

Returns a page of items for the given cursor. An absent cursor (`null`) yields the first page; a present cursor yields the page starting at the encoded offset; an unrecognized cursor yields an error WITHOUT throwing (RC-3/RC-4). Mirrors TS `getPage`.
