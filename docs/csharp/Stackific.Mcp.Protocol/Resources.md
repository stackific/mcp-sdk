# Resources class

The §17.4–§17.6 normative resource helpers ported from the TypeScript SDK's `resources.ts` / `resources-read.ts`: RFC 3986 / RFC 6570 validation (delegating the grammar to [`UriTemplate`](./UriTemplate.md)), the not-found error model (modern `-32602` plus legacy `-32002` client acceptance), the empty-`contents` guard, and the read-reply discrimination that tolerates an absent `resultType` and recognises the `input_required` continuation.
