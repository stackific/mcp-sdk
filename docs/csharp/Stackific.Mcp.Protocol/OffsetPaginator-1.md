# OffsetPaginator&lt;T&gt; class

A reference cursor-based paginator over an in-memory list — the C# counterpart of the TypeScript `OffsetPaginator`. Cursors are deterministic DECIMAL offset strings (matching TS, not base64): the same position always yields the same cursor token (RC-2: stability). An unrecognized or malformed cursor is returned as a structured error rather than thrown (RC-3/RC-4), so the server remains operational after a bad cursor.
