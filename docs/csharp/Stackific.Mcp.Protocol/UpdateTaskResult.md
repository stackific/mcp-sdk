# UpdateTaskResult record

The result of a `tasks/update` request (spec §25.8): an empty acknowledgement. The runtime supplies the base `resultType` as `"complete"` (see [`Complete`](./ResultTypes/Complete.md)). The acknowledgement is eventually consistent: it MAY be returned before the task's observable status reflects the responses.
