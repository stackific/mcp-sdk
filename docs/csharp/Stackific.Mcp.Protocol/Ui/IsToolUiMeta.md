# Ui.IsToolUiMeta method

Returns `true` when *value* is a well-formed `ToolUiMeta` — a JSON object carrying a REQUIRED `resourceUri` that uses the `ui://` scheme and an OPTIONAL `visibility` array drawn from `"model"`/`"app"` (spec §26.3, R-26.3-a, R-26.3-b, R-26.3-d). A non-`ui://``resourceUri` is rejected.
