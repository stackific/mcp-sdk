# Ui.MayServerExpectRendering method

Returns `true` when a server MAY expect a UI resource to be rendered — only when the host advertised the extension with the required [`MimeType`](../UiResource/MimeType.md). A server MUST NOT expect rendering otherwise (spec §26.2, R-26.2-g). Same gate as [`MayServerDeclareUi`](./MayServerDeclareUi.md), named separately so each prohibition reads clearly at the call site.

```csharp
public static bool MayServerExpectRendering(JsonObject? hostExtensionsMap)
```

| parameter | description |
| --- | --- |
| hostExtensionsMap | The host's advertised `extensions` map (raw). |

## Return Value

`true` when the server may expect rendering.
