# Ui.ResolveCsp method

Resolves the CSP a host applies for a UI resource: the declared `csp` when present, otherwise the restrictive [`DenyByDefaultCsp`](./DenyByDefaultCsp.md) (deny-by-default) (spec §26.4, R-26.4-h). A present `csp` is returned as-is for the host to constrain its policy by (R-26.4-o); an absent `csp` yields the all-empty deny-by-default policy.

```csharp
public static UiContentSecurityPolicy ResolveCsp(UiContentSecurityPolicy? csp)
```

| parameter | description |
| --- | --- |
| csp | The UI resource's declared `csp`, or `null`. |

The effective CSP.
