# Ui.IsUiHostExtensionCapability method

Returns `true` when *value* is a well-formed `UiHostExtensionCapability` — a JSON object carrying a `mimeTypes` string array (spec §26.2, R-26.2-d). This does NOT require the UI MIME type to be present — use [`CapabilityRendersUi`](./CapabilityRendersUi.md) for that.
