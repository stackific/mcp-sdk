# Ui.ToolsVisibleToModel method

Filters tools to those visible to the model, applying the §26.3 hide rule: a tool whose effective UI visibility is `["app"]`-only is omitted from the model's tool list (spec §26.3, R-26.3-f). The extension must be active for the rule to apply (R-26.3-g): when inactive, `_meta.ui` is ignored and every tool is treated as an ordinary, model-visible tool. A tool with no UI declaration is always model-visible.
