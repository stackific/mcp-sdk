# Ui.GetResourceUiMeta method

Extracts the [`ResourceUiMeta`](../ResourceUiMeta.md) hints from a UI resource `contents` entry — i.e. parses `contents._meta.ui` — returning `null` when there are no hints or they are malformed (spec §26.4, R-26.4-e). When present, these hints take effect for rendering.

```csharp
public static ResourceUiMeta? GetResourceUiMeta(JsonNode? contents)
```

| parameter | description |

| contents | A UI resource `contents` entry (raw). |

## Return Value

The parsed hints, or `null`.
