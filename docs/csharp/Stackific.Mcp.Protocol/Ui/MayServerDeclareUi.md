# Ui.MayServerDeclareUi method

Returns `true` when a server MAY declare UI associations on its tools — only when the host has advertised the extension with a `mimeTypes` array that includes the verbatim [`MimeType`](../UiResource/MimeType.md). A server MUST NOT declare UI associations otherwise (spec §26.2, R-26.2-f).

```csharp
public static bool MayServerDeclareUi(JsonObject? hostExtensionsMap)
```

| parameter | description |
| --- | --- |
| hostExtensionsMap | The host's advertised `extensions` map (raw). |

## Return Value

`true` when the server may declare UI.
