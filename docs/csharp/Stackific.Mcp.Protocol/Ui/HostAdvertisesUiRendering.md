# Ui.HostAdvertisesUiRendering method

Returns `true` when a host's `extensions` map advertises the apps extension in a way that enables UI rendering: the [`ExtensionId`](../UiResource/ExtensionId.md) key is present with a capability whose `mimeTypes` includes the verbatim [`MimeType`](../UiResource/MimeType.md) (spec §26.2, R-26.2-c, R-26.2-d, R-26.2-e). This is the predicate behind the server's two prohibitions — [`MayServerDeclareUi`](./MayServerDeclareUi.md) / [`MayServerExpectRendering`](./MayServerExpectRendering.md).

## Return Value

`true` when UI rendering is conformingly advertised.
