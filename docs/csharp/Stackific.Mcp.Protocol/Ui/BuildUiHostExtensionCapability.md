# Ui.BuildUiHostExtensionCapability method

Builds a conformant [`UiHostExtensionCapability`](../UiHostExtensionCapability.md) for a host that supports UI rendering. [`MimeType`](../UiResource/MimeType.md) is always included (deduplicated) so the result satisfies R-26.2-e; additional renderable MIME types are appended in order (spec §26.2, R-26.2-d, R-26.2-e).
