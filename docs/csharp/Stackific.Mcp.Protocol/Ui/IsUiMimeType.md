# Ui.IsUiMimeType method

Returns `true` when *mimeType* is exactly the UI MIME type [`MimeType`](../UiResource/MimeType.md) — matched verbatim and case-sensitively, with NO whitespace tolerance (spec §26.2/§26.4, R-26.2-e, R-26.4-d). `"text/html; profile=mcp-app"` (extra space) and `"TEXT/HTML;PROFILE=MCP-APP"` (wrong case) do NOT satisfy the requirement.

public static bool IsUiMimeType(string? mimeType)

## Return Value

`true` when byte-exactly the UI MIME type.
