# Ui.IsUiResourceContents method

Returns `true` when *value* is a well-formed UI resource `contents` entry: a `ui://``uri`, the verbatim [`MimeType`](../UiResource/MimeType.md), exactly one of `text`/`blob`, and — when present — well-formed `_meta.ui` hints (spec §26.4, R-26.4-d, R-26.4-e).
