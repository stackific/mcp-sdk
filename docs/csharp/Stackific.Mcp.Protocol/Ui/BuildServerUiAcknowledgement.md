# Ui.BuildServerUiAcknowledgement method

Builds the `capabilities.extensions` fragment a server includes in its `server/discover` result to acknowledge the apps extension: a single [`ExtensionId`](../UiResource/ExtensionId.md) key mapped to an empty object (spec §26.2, R-26.2-j). Acknowledgement is OPTIONAL; a server merges this fragment into the `extensions` map of its result capabilities when it chooses to acknowledge.
