# Ui.IsUiExtensionActive method

Returns `true` when, for an interaction, the apps extension is ACTIVE between client and server — both validly advertise [`ExtensionId`](../UiResource/ExtensionId.md) in their `extensions` maps (spec §26.2, R-26.2-a). When inactive, the host treats a tool carrying `_meta.ui` as a normal tool and ignores the UI key (R-26.2-i).

```csharp
public static bool IsUiExtensionActive(JsonObject? clientExtensions, JsonObject? serverExtensions)
```

| parameter | description |
| --- | --- |
| clientExtensions | The client/host's advertised `extensions` map (raw). |
| serverExtensions | The server's advertised `extensions` map (raw). |

## Return Value

`true` when active on both sides.
