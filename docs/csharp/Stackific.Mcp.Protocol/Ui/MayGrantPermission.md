# Ui.MayGrantPermission method

Returns `true` when a host MAY grant the named sandbox capability for a UI resource: ONLY when it was requested (the host MUST NOT grant an unrequested capability) AND the host did not decline it (the host MAY decline a requested one) (spec §26.4, R-26.4-j, R-26.4-k).

```csharp
public static bool MayGrantPermission(UiPermissions? permissions, UiPermissionName name, 
    bool hostDeclines = false)
```

| parameter | description |
| --- | --- |
| permissions | The UI resource's declared `permissions`. |
| name | The capability under consideration. |
| hostDeclines | Whether the host chooses to decline this requested capability; defaults to `false`. |

## Return Value

`true` when the host may grant it.
