# Ui.HostShouldRejectUiOriginatedCall method

Returns `true` when a host SHOULD REJECT a `tools/call` that originates from a rendered UI, given the tool's UI declaration: it is rejected exactly when the tool's effective visibility excludes `"app"`, OR when the tool has no UI declaration at all (it was not exposed to the UI) (spec §26.3, R-26.3-e).

```csharp
public static bool HostShouldRejectUiOriginatedCall(ToolUiMeta? meta)
```

| parameter | description |
| --- | --- |
| meta | The tool's `ToolUiMeta`, or `null` when it has none. |

## Return Value

`true` when the host should reject the UI-originated call.
