# Ui.CapabilityRendersUi method

Returns `true` when an advertised host capability value enables UI rendering: it is a well-formed [`IsUiHostExtensionCapability`](./IsUiHostExtensionCapability.md) AND its `mimeTypes` array contains the verbatim [`MimeType`](../UiResource/MimeType.md) (spec §26.2, R-26.2-d, R-26.2-e).
