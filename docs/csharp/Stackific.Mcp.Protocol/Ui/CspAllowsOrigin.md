# Ui.CspAllowsOrigin method

Returns `true` when *origin* is ALLOWED for the given CSP *directive* of a *csp* descriptor — it is explicitly listed in that member. An origin not listed (including when the member is absent, or when `csp` is `null`) MUST be blocked — deny-by-default applies (spec §26.4, R-26.4-g, R-26.4-h).

```csharp
public static bool CspAllowsOrigin(UiContentSecurityPolicy? csp, UiCspDirective directive, 
    string origin)
```

| parameter | description |
| --- | --- |
| csp | The resolved CSP descriptor, or `null` when `csp` was omitted. |
| directive | Which CSP member to consult. |
| origin | The origin string to test. |

## Return Value

`true` when explicitly allowed.
