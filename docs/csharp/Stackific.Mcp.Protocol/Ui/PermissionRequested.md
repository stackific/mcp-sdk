# Ui.PermissionRequested method

Returns `true` when a UI resource's `permissions` REQUESTS the named sandbox capability — i.e. the member is present. Absence means the capability is not requested, and the host MUST NOT grant it (spec §26.4, R-26.4-i, R-26.4-j).

```csharp
public static bool PermissionRequested(UiPermissions? permissions, UiPermissionName name)
```

| parameter | description |
| --- | --- |
| permissions | The UI resource's declared `permissions`, or `null`. |
| name | The capability to test. |

## Return Value

`true` when requested.
