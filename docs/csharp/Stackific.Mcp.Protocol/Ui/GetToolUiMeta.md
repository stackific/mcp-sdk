# Ui.GetToolUiMeta method

Extracts the [`ToolUiMeta`](../ToolUiMeta.md) from a tool — i.e. parses `tool._meta.ui` — returning `null` when there is no `_meta`, no `ui` key, or the value is not a well-formed declaration (spec §26.3). This does NOT gate on negotiation: a receiver that has not negotiated the extension MUST ignore the key (R-26.3-g) — use [`ReadToolUiMeta`](./ReadToolUiMeta.md) for that.

```csharp
public static ToolUiMeta? GetToolUiMeta(JsonNode? tool)
```

| parameter | description |
| --- | --- |
| tool | A tool object (or anything with an optional `_meta.ui`) (raw). |

## Return Value

The parsed declaration, or `null`.
