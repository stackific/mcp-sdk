# Ui.RequestedPermissions method

Returns the set of sandbox capabilities a UI resource requests, as the subset of the four permission names present in *permissions* (spec §26.4, R-26.4-i). The host MUST NOT grant any capability outside this set (R-26.4-j) and MAY decline any within it (R-26.4-k).

```csharp
public static IReadOnlyList<UiPermissionName> RequestedPermissions(UiPermissions? permissions)
```

| parameter | description |
| --- | --- |
| permissions | The UI resource's declared `permissions`, or `null`. |

## Return Value

The requested permission names, in spec order.
