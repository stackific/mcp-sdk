# Ui.IsUiResourceUri method

Returns `true` when *uri* is a `ui://`-scheme URI string. The authority and path after `ui://` are server-defined and opaque; this only checks the scheme — the host MUST treat the whole URI as an opaque identifier and derive no network origin from it (spec §26.4, R-26.3-b, R-26.4-b, R-26.4-c).

```csharp
public static bool IsUiResourceUri(string? uri)
```

| parameter | description |
| --- | --- |
| uri | The candidate URI. |

## Return Value

`true` when it begins with the `ui://` scheme.
