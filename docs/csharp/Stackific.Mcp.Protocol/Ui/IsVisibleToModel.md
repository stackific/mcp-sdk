# Ui.IsVisibleToModel method

Returns `true` when a tool's effective visibility includes `"model"` — i.e. it appears in the model's tool list and is callable via ordinary tool-calling. A tool with `visibility``["app"]` is callable ONLY by the UI and is HIDDEN from the model's list, so this returns `false` (spec §26.3, R-26.3-f).

```csharp
public static bool IsVisibleToModel(ToolUiMeta meta)
```

| parameter | description |
| --- | --- |
| meta | The tool's `ToolUiMeta`. |

## Return Value

`true` when visible to the model.
