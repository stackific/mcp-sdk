# Ui.BuildUiResourceContents method

Builds a UI resource `contents` entry: the `ui://`*uri*, the verbatim [`MimeType`](../UiResource/MimeType.md), the `text` OR `blob` payload, and — when supplied — the [`ResourceUiMeta`](../ResourceUiMeta.md) hints nested under `_meta.ui` (spec §26.4, R-26.4-d, R-26.4-e). Exactly one of *text*/*blob* MUST be supplied (S21 exclusivity).

```csharp
public static JsonObject BuildUiResourceContents(string uri, string? text = null, 
    string? blob = null, ResourceUiMeta? ui = null)
```

| parameter | description |
| --- | --- |
| uri | The `ui://` URI of the resource. |
| text | The HTML document as text, or `null`. |
| blob | The document as Base64, or `null`. |
| ui | OPTIONAL presentation/security hints nested under `_meta.ui`. |

## Return Value

The contents entry as a JSON object.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | When *uri* is not a `ui://` URI, or when neither/both of *text*/*blob* are supplied. |
