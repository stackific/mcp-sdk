## Remarks

The wire [`Icon`](./Icon.md) record (carried on [`Implementation`](./Implementation.md), [`ResourceLink`](./ResourceLink.md), and other descriptors) models the happy-path shape; the security model lives here so it can be applied deliberately by a consumer that is about to render an advertised icon.

Security model (§14.2):

* Only `https:` URLs and `data:` URIs are accepted (R-14.2-o).
* `javascript:`, `file:`, `ftp:`, `ws:`, `http:` and other schemes are rejected (R-14.2-n).
* The MIME type is detected from magic bytes, never trusted from the declared type (R-14.2-s).
* Only image types on the allowlist are rendered (R-14.2-u).
* A fetch is credential-free and refuses cross-origin / scheme-change redirects (R-14.2-p, R-14.2-q).
