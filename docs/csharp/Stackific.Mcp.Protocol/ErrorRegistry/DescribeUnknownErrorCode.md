# ErrorRegistry.DescribeUnknownErrorCode method

Surfaces an error response carrying a code the receiver does not recognise. Per R-22.7-e a receiver MUST treat an unknown code as a failed request and surface it using the message and data, NOT reject it as malformed (AC-34.24).
