# ErrorRegistry.ClassifyErrorCode method

Classifies any integer *code* into one of the [`ErrorCodeClass`](../ErrorCodeClass.md) ranges, even codes not present in the registry. A registry entry's own class always wins; otherwise the code is placed by range (R-22.7-a).
