# ErrorRegistry.LookupErrorCode method

Looks up the registry entry for *code*, or `null` if it is not a §22 registry code. An absent entry is not an error — receivers MUST tolerate unknown codes (see [`DescribeUnknownErrorCode`](./DescribeUnknownErrorCode.md)). (R-22.7-e)

```csharp
public static ErrorCodeRegistryEntry? LookupErrorCode(int code)
```

| parameter | description |
| --- | --- |
| code | The numeric error code to look up. |

## Return Value

The matching entry, or `null` when the code is unregistered.
