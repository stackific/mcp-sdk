# ErrorRegistry.HttpStatusForRegistryCode method

Maps an error *code* to the Streamable HTTP status it MUST ride on (§22.6). `-32003`/`-32004` (negotiation) and `-32001` (HeaderMismatch) all map to `400`; codes the registry does not pin to a status return `null`.
