# ErrorRegistry.BuildNullIdParseErrorResponse method

Builds the `null`-id parse-error response for unparseable input — the one circumstance in which an error response's id need not match a request id (R-22.1-f, R-22.6-h, AC-34.4).

```csharp
public static JsonRpcErrorResponse BuildNullIdParseErrorResponse(string message = "Parse error")
```

| parameter | description |
| --- | --- |
| message | An optional override; defaults to `"Parse error"`. |

## Return Value

An error response with a `null` id and a `-32700` error.
