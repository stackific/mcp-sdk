# ErrorRegistry.ResourceNotFoundLegacyCode field

The legacy MCP "Resource not found" error code literal, `-32002` (§22.4).

```csharp
public const int ResourceNotFoundLegacyCode;
```

## Remarks

In §22's registry a `resources/read` for a non-existent URI is canonically a `-32602` Invalid params condition (R-22.4-g) carrying `data.uri` (R-22.4-h); the registry also recognises this dedicated `-32002` literal. The name is suffixed `Legacy` to stay collision-free with the Resources feature's own bindings.
