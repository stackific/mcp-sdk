# ErrorRegistry.BuildResourceNotFoundParamsError method

Builds a `-32602` Invalid params resource-not-found error whose `data` includes the requested *uri*, per the §22.4 canonical mapping (R-22.4-g/h). A non-existent resource MUST be signalled this way and MUST NOT be signalled by an empty `contents` array (R-22.4-i).

```csharp
public static JsonRpcError BuildResourceNotFoundParamsError(string uri, 
    string message = "Resource not found")
```

| parameter | description |
| --- | --- |
| uri | The requested resource URI that was not found. |
| message | An optional override; defaults to `"Resource not found"`. |

## Return Value

The constructed wire error object carrying `data.uri`.
