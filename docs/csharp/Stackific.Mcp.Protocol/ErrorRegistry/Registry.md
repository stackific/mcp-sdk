# ErrorRegistry.Registry property

The complete §22 error-code registry, keyed by numeric code. The same `code` applies on every transport; the optional [`HttpStatus`](../ErrorCodeRegistryEntry/HttpStatus.md) is the Streamable HTTP overlay (§22.6). Note that `-32602` has a single entry even though several distinct conditions collapse onto it — the code is the key; the condition is conveyed by message/data.
