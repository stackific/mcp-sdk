# DiscoverResult record

The result of the `server/discover` request (spec §5.3.2): the protocol revisions the server supports, its advertised capabilities, its identity, optional natural-language instructions, and optional result-level metadata. It is a `Result`, so on the wire it also carries the base `resultType` discriminator (normally `complete`); that is added by the runtime.
