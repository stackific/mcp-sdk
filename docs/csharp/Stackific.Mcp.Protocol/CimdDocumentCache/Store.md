# CimdDocumentCache.Store method

Caches a fetched CIMD document keyed by its `client_id` URL, honouring HTTP cache directives and the trust policy (R-23.12-k, R-23.12-l). Returns `true` when the document was stored, `false` when caching was declined (untrusted host, `no-store`, or a non-positive `max-age`).

public bool Store(string clientIdUrl, ClientIdMetadataDocument document,

`true` when stored.
