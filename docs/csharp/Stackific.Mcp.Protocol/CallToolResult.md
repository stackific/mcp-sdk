# CallToolResult record

The result of a completed `tools/call` (spec §16.5): unstructured [`Content`](./CallToolResult/Content.md) blocks, optional [`StructuredContent`](./CallToolResult/StructuredContent.md) (any JSON value), and an [`IsError`](./CallToolResult/IsError.md) flag for tool-execution failures (§16.6 — distinct from a JSON-RPC protocol error). The base `resultType` is supplied by the runtime.
