# SamplingValidation class

The §21.2 sampling behavioral layer — the C# counterpart of the TypeScript `protocol/sampling.ts` helpers — for the Deprecated Sampling capability. The wire records above carry the data shapes; this static class adds the rules the spec layers on top: the capability gate ([`GateSamplingToolUse`](./SamplingValidation/GateSamplingToolUse.md) / [`MayServerSendSamplingRequest`](./SamplingValidation/MayServerSendSamplingRequest.md), rejecting tool use without `sampling.tools` with `-32602`; gating the Deprecated `includeContext` values via `sampling.context`), the §21.2.7/§21.2.6 message-content constraints (ordering, user tool_result exclusivity, tool_result back-references), the `ModelPreferences` priority 0..1 validation (R-21.2.9-e), and the various resolvers/defaults.
