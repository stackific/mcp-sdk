# AuthorizationResponseParams constructor

The redirect query parameters the authorization server returns (spec §23.5, §23.7). On success [`Code`](./Code.md) is present; [`State`](./State.md) echoes the request `state`; [`Iss`](./Iss.md) identifies the authorization server (SHOULD). On error, the [`Error`](./Error.md) fields are present and MUST NOT be acted on when `iss` validation fails (R-23.7-h).

```csharp
public AuthorizationResponseParams(string? Code = null, string? State = null, string? Iss = null, 
    string? Error = null, string? ErrorDescription = null, string? ErrorUri = null)
```

| parameter | description |
| --- | --- |
| Code | The authorization code to redeem (success). |
| State | Echo of the request `state` (present if sent) (R-23.5-h). |
| Iss | The authorization server's issuer identifier (SHOULD) (R-23.5-k, R-23.7-b). |
| Error | Error code (error responses). |
| ErrorDescription | OPTIONAL human-readable error description. |
| ErrorUri | OPTIONAL URI with error information. |
