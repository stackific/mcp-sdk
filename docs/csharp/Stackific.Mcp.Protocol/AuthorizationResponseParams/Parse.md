# AuthorizationResponseParams.Parse method

Parses an authorization-redirect URL (or raw query string) into its decoded parameters (spec §23.5, Step 3). Percent-decoding is applied; the decoded `iss` is later compared by EXACT string match with no further normalization (R-23.7-g) — this parser performs no normalization beyond the form-decoding the wire requires.
