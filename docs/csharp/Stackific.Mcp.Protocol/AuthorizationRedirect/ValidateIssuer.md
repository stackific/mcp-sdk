# AuthorizationRedirect.ValidateIssuer method

Validates the authorization response's `iss` against the recorded issuer per §23.7 — the check a client MUST perform BEFORE transmitting the authorization code to any token endpoint (R-23.7-a, R-23.7-d – R-23.7-g). When the decision is Compare, the present `iss` is compared by EXACT string match (no normalization; R-23.7-g).

public static AuthorizationResult ValidateIssuer(string? iss, string recordedIssuer,

| parameter | description |

| recordedIssuer | The `issuer` recorded in Step 1 (R-23.5-c). |
