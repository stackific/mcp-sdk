# AuthorizationRedirect.Decision method

Applies the §23.7 four-row decision table to determine how to treat the `iss` parameter (R-23.7-d, R-23.7-e, R-23.7-f). A present `iss` is ALWAYS compared, regardless of advertisement (R-23.7-f).

```csharp
public static IssuerValidationDecision Decision(bool? issParameterSupported, bool issPresent)
```

| parameter | description |
| --- | --- |
| issParameterSupported | The AS metadata flag (`null` ⇒ not advertised). |
| issPresent | Whether the response carried an `iss`. |

## Return Value

The decision to apply.
