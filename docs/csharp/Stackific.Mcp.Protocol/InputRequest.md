# InputRequest record

A single input request a server embeds in an [`InputRequiredResult`](./InputRequiredResult.md) for the client to fulfill (spec §11.2). The kind is discriminated by [`Method`](./InputRequest/Method.md): `elicitation/create` (§20), `sampling/createMessage` (§21), or `roots/list` (§21). The runtime dispatches on the method name, so [`Params`](./InputRequest/Params.md) is kept as raw JSON.
