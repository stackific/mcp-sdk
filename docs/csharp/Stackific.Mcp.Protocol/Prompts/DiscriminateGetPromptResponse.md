# Prompts.DiscriminateGetPromptResponse method

Branches a `prompts/get` response on its `resultType` discriminator (§18.4): `"input_required"` ⇒ a continuation (R-18.4-q); `"complete"` or an ABSENT `resultType` ⇒ a completed [`GetPromptResult`](../GetPromptResult.md) (the absent-⇒-complete default, R-18.4-p); any other value or a body that fails its shape ⇒ an error branch (R-18.4-r). A client MUST inspect `resultType` before parsing the body.
