# Elicitation.ResolveElicitationMode method

Resolves the effective elicitation mode of a `params` object, applying the backwards-compatibility rule that an absent `mode` means form mode (spec §20.3, R-20.3-b, R-20.3-c). Returns `"form"` when `mode` is absent or the literal `"form"`, `"url"` when it is the literal `"url"`, and `null` for any other (malformed) value.

## Return Value

The resolved mode, or `null` when malformed.
