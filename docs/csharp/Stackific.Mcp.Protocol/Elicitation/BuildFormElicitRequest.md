# Elicitation.BuildFormElicitRequest method

Builds a well-formed form-mode [`ElicitRequestFormParams`](../ElicitRequestFormParams.md) (spec §20.2, §20.3). The *requestedSchema* is validated against the §20.4 restricted form schema before the request is built (R-20.4); a malformed schema is rejected rather than sent. Form mode is the implicit baseline, so the `mode` field is not modeled on the record (it is emitted on the wire by the converter).
