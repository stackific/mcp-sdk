# Elicitation.GateElicitationRequest method

Decides whether a server MAY send an `elicitation/create` request of *mode* to a client with the given declared capabilities (spec §20.1, R-20.1-d, R-20.1-e). A server MUST NOT return such a request to a client that has not declared `elicitation` (CapabilityNotDeclared), nor one whose `mode` the client's declared sub-flags do not support (ModeNotSupported).

```csharp
public static ElicitationGateResult GateElicitationRequest(JsonObject clientCaps, 
    string mode = "form")
```

| parameter | description |
| --- | --- |
| clientCaps | The client's declared capabilities (raw map). |
| mode | The mode the server intends to use; defaults to `"form"` (the absent-mode baseline, R-20.3-c). |

## Return Value

The gate result.
