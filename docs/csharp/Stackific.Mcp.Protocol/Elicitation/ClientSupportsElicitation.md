# Elicitation.ClientSupportsElicitation method

Returns `true` when *clientCaps* declares the `elicitation` capability — the MUST-declare-to-use rule (spec §20.1, R-20.1-a). A client that does not declare it is treated as not supporting elicitation.

```csharp
public static bool ClientSupportsElicitation(JsonObject clientCaps)
```

| parameter | description |
| --- | --- |
| clientCaps | The client's declared capabilities (raw map). |

## Return Value

`true` when elicitation is declared.
