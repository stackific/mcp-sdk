# Elicitation.ClientSupportsElicitationMode method

Returns `true` when the client declaring *clientCaps* supports *mode*, applying the empty-object-equals-form-only equivalence (spec §20.1, R-20.1-c, R-20.1-f). `form` is supported whenever `elicitation` is declared; `url` requires the `elicitation.url` sub-flag.

```csharp
public static bool ClientSupportsElicitationMode(JsonObject clientCaps, string mode)
```

| parameter | description |
| --- | --- |
| clientCaps | The client's declared capabilities (raw map). |
| mode | The mode to test. |

## Return Value

`true` when the mode is supported.
