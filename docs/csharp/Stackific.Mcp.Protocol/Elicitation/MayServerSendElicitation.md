# Elicitation.MayServerSendElicitation method

Convenience predicate: `true` exactly when [`GateElicitationRequest`](./GateElicitationRequest.md) permits a server to send an `elicitation/create` request of *mode* (spec §20.1, R-20.1-d, R-20.1-e).

```csharp
public static bool MayServerSendElicitation(JsonObject clientCaps, string mode = "form")
```

| parameter | description |
| --- | --- |
| clientCaps | The client's declared capabilities (raw map). |
| mode | The mode the server intends to use; defaults to `"form"`. |

## Return Value

`true` when permitted.
