# Elicitation.SupportedElicitationModes method

Returns the set of elicitation modes a client supports, applying the empty-object-equals-form-only equivalence: declaring `elicitation` always implies `form` (the implicit baseline), and `url` is added only when the `elicitation.url` sub-flag is present (spec §20.1, R-20.1-c, R-20.1-f). Returns an empty list when `elicitation` is not declared at all.


| --- | --- |

## Return Value

The supported modes (`[]`, `["form"]`, or `["form","url"]`).
