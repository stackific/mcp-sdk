# Elicitation.BuildUrlElicitRequest method

Builds a well-formed url-mode [`ElicitRequestURLParams`](../ElicitRequestURLParams.md) (spec §20.2, §20.3). The `url` is checked for validity (R-20.3-n) and the `elicitationId` for non-emptiness (R-20.3-k) before the request is built.

```csharp
public static ElicitRequestURLParams BuildUrlElicitRequest(string message, string elicitationId, 
    string url)
```

| parameter | description |
| --- | --- |
| message | The explanation of why the interaction is needed. |
| elicitationId | The opaque server-scoped correlation id; MUST be non-empty. |
| url | The URL the user navigates to; MUST be a valid absolute URL. |

## Return Value

The url-mode params.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | When *elicitationId* is empty (R-20.3-k) or *url* is invalid (R-20.3-n). |
