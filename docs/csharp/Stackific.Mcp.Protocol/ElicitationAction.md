# ElicitationAction enumeration

The user's response action to an elicitation request (spec §20.5). Exactly one of these literals is returned in [`Action`](./ElicitResult/Action.md); the distinctions apply to both form and URL modes.

```csharp
public enum ElicitationAction
```

## Values

| name | value | description |
| --- | --- | --- |
| Accept | `0` | The user explicitly approved and submitted (spec §20.5). For form mode, the result carries the collected `content`; for URL mode it signals consent to the interaction, not its completion. |
| Decline | `1` | The user explicitly declined the request (spec §20.5). `content` is typically omitted. |
| Cancel | `2` | The user dismissed without an explicit choice — for example closed the dialog, clicked away, pressed Escape, or the URL failed to load (spec §20.5). `content` is typically omitted. |
