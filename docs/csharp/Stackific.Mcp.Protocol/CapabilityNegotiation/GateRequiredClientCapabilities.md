# CapabilityNegotiation.GateRequiredClientCapabilities method

Gates a request against the capabilities it requires (§6.4, R-6.4-h). Returns `null` when every required capability is declared (the request is allowed); otherwise returns the `-32003``MissingRequiredClientCapability` error whose `data.requiredCapabilities` lists exactly the required-but-undeclared capabilities. Mirrors TS `gateRequiredClientCapabilities`.

```csharp
public static JsonRpcError? GateRequiredClientCapabilities(JsonObject declared, JsonObject required)
```

| parameter | description |
| --- | --- |
| declared | The capabilities from the current request's `_meta`. |
| required | The capabilities the server needs to process the request. |

## Return Value

The blocking `-32003` error, or `null` when the request is allowed.
