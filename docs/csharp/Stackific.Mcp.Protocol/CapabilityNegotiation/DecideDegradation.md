# CapabilityNegotiation.DecideDegradation method

Decides how to handle an operation when the other peer may not declare the optional behavior it would use (R-6.4-l, R-6.4-m): Proceed when the peer declares it, Fallback when it does not but the behavior is optional, and Reject only when the missing behavior is mandatory. A peer MUST NOT reject merely because the other declared fewer capabilities (R-6.4-m). Mirrors TS `decideDegradation`.
