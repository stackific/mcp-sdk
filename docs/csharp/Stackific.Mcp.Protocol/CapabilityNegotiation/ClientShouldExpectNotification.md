# CapabilityNegotiation.ClientShouldExpectNotification method

Returns `true` when a client should expect *notification* given the server's declared capabilities. When the gating sub-flag is absent or `false`, the client MUST NOT expect the notification (R-6.3-h, R-6.3-l, R-6.3-o). Mirrors TS `clientShouldExpectNotification`.

```csharp
public static bool ClientShouldExpectNotification(string notification, JsonObject serverCaps)
```

| parameter | description |
| --- | --- |
| notification | The notification name. |
| serverCaps | The server's declared capabilities. |

## Return Value

`true` when the notification may be expected.
