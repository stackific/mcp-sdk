# CapabilityNegotiation.ClientDeclares method

Returns `true` when the client's raw capability map declares *capability* (§6.1). Presence of an object means supported; `elicitation.form` is the implicit baseline (true whenever `elicitation` is present, R-6.2-e), while `elicitation.url`, `sampling.context`, and `sampling.tools` require their own sub-flag object. Mirrors TS `clientDeclares`.

```csharp
public static bool ClientDeclares(JsonObject caps, string capability)
```

| parameter | description |
| --- | --- |
| caps | The raw client capabilities map. |
| capability | A capability name or dotted sub-flag path (for example `elicitation.url`). |

## Return Value

`true` when declared.
