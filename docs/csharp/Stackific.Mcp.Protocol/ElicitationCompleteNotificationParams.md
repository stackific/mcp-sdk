# ElicitationCompleteNotificationParams record

The parameters of the `notifications/elicitation/complete` notification (spec §20.6), which a server MAY send to signal that a URL-mode out-of-band interaction has completed. A client MUST ignore a notification referencing an unknown or already-completed [`ElicitationId`](./ElicitationCompleteNotificationParams/ElicitationId.md).
