# PaginationUtilities.PaginationCacheKey method

Produces a per-page cache key for a paginated request, enforcing per-cursor cache isolation: a cached page for one cursor value MUST NOT be served for a request bearing a different cursor (including the first-page request, which omits `cursor`). The empty string `""` is a present cursor distinct from absence. Mirrors TS `paginationCacheKey`. (R-12.5-a, R-13.5-i)

```csharp
public static string PaginationCacheKey(string method, string? cursor)
```

| parameter | description |
| --- | --- |
| method | The list method name. |
| cursor | The request's `cursor`, or `null` for the first page. |

## Return Value

A cache key isolating this page from every other cursor.
