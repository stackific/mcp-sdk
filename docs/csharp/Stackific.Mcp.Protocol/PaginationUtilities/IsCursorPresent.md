# PaginationUtilities.IsCursorPresent method

Returns `true` when a `cursor` is a present value (including the empty string). A client MUST echo any present `nextCursor` — even `""` — on the next request; only `null` signals "no cursor". Mirrors TS `isCursorPresent`. (R-12.1-a)
