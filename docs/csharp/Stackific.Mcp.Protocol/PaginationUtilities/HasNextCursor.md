# PaginationUtilities.HasNextCursor method

Returns `true` when a `nextCursor` is PRESENT in a result, indicating more results MAY follow. The empty string `""` is a present cursor — only `null` (absent) is treated as "no more". Mirrors TS `hasNextCursor`. (R-12.2-c, R-12.3-b/d)

```csharp
public static bool HasNextCursor(string? nextCursor)
```

| parameter | description |
| --- | --- |
| nextCursor | The result's `nextCursor`, or `null` when absent. |

## Return Value

`true` when a (possibly empty) cursor is present.
