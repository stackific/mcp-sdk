# Resources.BuildResourceNotFoundError method

Builds the modern resource-not-found error: `-32602` (Invalid params) carrying the offending `uri` in `data.uri`. A server MUST signal non-existence with this error — NOT an empty `contents` result. (§17.5, §17.6, R-17.5-z, R-17.6-a, R-17.6-b)
