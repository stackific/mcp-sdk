# ProgressRateLimiter class

A per-token rate limiter for `notifications/progress` emissions (RC-3 / SHOULD) — the C# counterpart of the TypeScript `ProgressRateLimiter`. Each token has an independent last-emit time so a slow token is not penalized by a fast one; string and number tokens are tracked independently. The current time is injected (no wall-clock), so behavior is deterministic.
