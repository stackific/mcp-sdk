# CacheHints constructor

Response-cache hints carried by a cacheable result (spec §13.1): a client-cache time-to-live in milliseconds and the sharing scope. Both are REQUIRED on a cacheable result.

```csharp
public CacheHints(long TtlMs, CacheScope CacheScope)
```

| parameter | description |
| --- | --- |
| TtlMs | How long the client MAY treat the result as fresh, in milliseconds (minimum 0). |
| CacheScope | The sharing scope of the cached result. |
