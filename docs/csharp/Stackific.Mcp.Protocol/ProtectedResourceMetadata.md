## Remarks

[`Resource`](./ProtectedResourceMetadata/Resource.md) is REQUIRED and MUST equal the server's canonical resource identifier (R-23.2-h). [`AuthorizationServers`](./ProtectedResourceMetadata/AuthorizationServers.md) is REQUIRED for MCP, MUST be present, and MUST contain at least one entry (R-23.2-i). [`ScopesSupported`](./ProtectedResourceMetadata/ScopesSupported.md) and [`BearerMethodsSupported`](./ProtectedResourceMetadata/BearerMethodsSupported.md) are OPTIONAL. Additional RFC 9728 fields are preserved on round-trip via [`Extra`](./ProtectedResourceMetadata/Extra.md).
