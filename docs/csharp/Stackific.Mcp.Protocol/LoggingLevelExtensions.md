## Remarks

[`LoggingLevel`](./LoggingLevel.md) is defined alongside the deprecated logging-message feature; its members are declared least-severe-first, so the enum's underlying integer value is its ascending-severity index. The deprecated `io.modelcontextprotocol/logLevel``_meta` key opts a request in at a minimum severity: when present, a server SHOULD emit only log notifications at or above it; when absent, it MUST NOT emit log notifications for the request (R-4.3-l, R-4.3-m).
