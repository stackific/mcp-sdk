# UnsupportedDialectException class

Raised when a tool schema declares a JSON Schema dialect this implementation cannot validate against (spec §16.4(9), R-16.4-t). The implementation MUST handle an unsupported dialect by signalling an error rather than silently ignoring the declaration or treating the schema as permissive.
