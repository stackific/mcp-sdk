# DcrRetry.ApplicationTypeForRedirectUris method

Classifies a set of redirect URIs as native or web and returns the `application_type` a client SHOULD register, consistent with those URIs (R-23.15-a, R-23.15-b, R-23.15-c). Redirect URIs that all resolve to a loopback host (`localhost`, `127.0.0.0/8`, or IPv6 `::1`, as decided by IsLoopback) indicate a native application → Native; otherwise Web.

```csharp
public static ApplicationType ApplicationTypeForRedirectUris(IReadOnlyList<string> redirectUris)
```

| parameter | description |
| --- | --- |
| redirectUris | The client's redirect URIs. |

## Return Value

The `application_type` consistent with the redirect URIs.
