## Remarks

The host creates and coordinates many clients; each client is bound one-to-one to exactly one server, and servers are isolated from one another. The host is NOT a JSON-RPC role on the wire, so it does not appear here.

This endpoint role is deliberately distinct from [`Role`](./Role.md) (the content-author / audience role of §14.7, whose values are `user` and `assistant`). Modeled as an enum — the idiomatic C# form of a closed set of wire strings — with the same JsonStringEnumConverter pattern used by [`Role`](./Role.md) and [`IconTheme`](./IconTheme.md), so it round-trips to the literal lowercase wire values. (AC-01.1)
