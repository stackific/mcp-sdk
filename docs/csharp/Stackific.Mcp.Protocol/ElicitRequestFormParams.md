# ElicitRequestFormParams record

Form-mode elicitation parameters (spec §20.3, `mode: "form"`): in-band structured data collection against a restricted JSON Schema. The collected data IS exposed to the client, so a server MUST NOT use form mode for sensitive information such as passwords, API keys, tokens, or payment credentials (§20.7).
