# IncompatibleProtocolError class

An actionable error a client surfaces to its caller when no protocol revision is mutually supported (§5.4-d, §5.5-j). It carries both sides' revision sets for diagnostics and, unlike an [`McpError`](../Stackific.Mcp.JsonRpc/McpError.md), is NOT a wire error — it never goes on the wire. Mirrors TS `IncompatibleProtocolError`.
