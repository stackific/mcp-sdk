# TokenRequests.BuildRefresh method

Builds the refresh-token token-request body, fixing `grant_type=refresh_token` and carrying the same `resource` parameter so the refreshed token stays audience-bound (R-23.9-e, R-23.9-f).

```csharp
public static RefreshTokenRequest BuildRefresh(string refreshToken, string clientId, 
    string resource, string? scope = null)
```

| parameter | description |
| --- | --- |
| refreshToken | The refresh token being exchanged (R-23.9-e). |
| clientId | The client identifier. |
| resource | The SAME canonical resource identifier as Step 2 (R-23.9-e). |
| scope | OPTIONAL narrowed scopes (R-23.9-f). |
