# LogRateLimiter class

A global rate limiter for `notifications/message` log emissions (RC-3 / SHOULD) — the C# counterpart of the TypeScript `LogRateLimiter`. Unlike [`ProgressRateLimiter`](./ProgressRateLimiter.md), a single shared throttle window applies to the entire log stream (all messages share the one `notifications/message` channel). The current time is injected, so behavior is deterministic.
