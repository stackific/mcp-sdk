# ExtensionMethodRouter.Dispatch method

Dispatches *method* with *parameters*, but only when the owning extension is in *activeSet* (spec R-24.5-c). An unknown method or an inactive owning extension is refused with `-32602` so a caller can convert the outcome into a core error response when it chooses to reject rather than ignore (R-24.3-f).
