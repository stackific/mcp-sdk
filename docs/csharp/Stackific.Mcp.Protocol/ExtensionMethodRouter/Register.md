# ExtensionMethodRouter.Register method

Registers *handler* for an extension-defined *method*. The method MUST be in *identifier*'s derived namespace (R-24.5-b) and MUST NOT already be registered (no redefinition, R-24.5-i).

```csharp
public ExtensionMethodRouter Register(string identifier, string method, 
    Func<object?, object?> handler)
```

| parameter | description |
| --- | --- |
| identifier | The owning extension identifier. |
| method | The namespaced method string. |
| handler | The handler invoked on dispatch. |

## Return Value

This router, for fluent chaining.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | When the method is not namespaced under the identifier or is already registered. |
