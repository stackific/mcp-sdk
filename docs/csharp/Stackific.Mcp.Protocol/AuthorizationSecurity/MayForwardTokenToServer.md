# AuthorizationSecurity.MayForwardTokenToServer method

Returns `true` when a client MAY send the access token it holds for *tokenIssuer* to the MCP server whose authorization server is *serverIssuer* — strictly only when the issuers match exactly. A client MUST NOT send a token to an MCP server other than one issued by that server's authorization server (R-23.19-c).

```csharp
public static bool MayForwardTokenToServer(string tokenIssuer, string serverIssuer)
```

| parameter | description |
| --- | --- |
| tokenIssuer | The issuer that minted the token the client holds. |
| serverIssuer | The issuer of the target server's authorization server. |

## Return Value

`true` when the token may be forwarded.
