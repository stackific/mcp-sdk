# AuthorizationSecurity.CheckResourceParameterBinding method

Validates the audience-binding requirement: the SAME `resource` parameter, identifying the MCP server by its canonical URI, MUST be present in BOTH the authorization request and the token request, regardless of advertised AS support (R-23.19-a).
