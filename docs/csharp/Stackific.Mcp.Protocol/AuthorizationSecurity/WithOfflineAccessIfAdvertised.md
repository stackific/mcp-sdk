# AuthorizationSecurity.WithOfflineAccessIfAdvertised method

Adds `offline_access` to a scope list when, and only when, the authorization-server metadata advertises it in `scopes_supported`, for a client that wants a refresh token (R-23.19-s). The result is deduplicated.

```csharp
public static IReadOnlyList<string> WithOfflineAccessIfAdvertised(IReadOnlyList<string> scopes, 
    IReadOnlyList<string>? asScopesSupported)
```

| parameter | description |
| --- | --- |
| scopes | The current scope list. |
| asScopesSupported | The selected authorization server's `scopes_supported`. |

## Return Value

The scopes, possibly with `offline_access` appended.
