# AuthorizationSecurity.CheckBearerHeaderOnly method

Validates that the access token is presented ONLY in the `Authorization: Bearer` request header and NEVER in the URI query string (R-23.19-p). Reuses S36's [`UrlContainsAccessTokenInQuery`](../AccessTokenUsage/UrlContainsAccessTokenInQuery.md) and requires an `Authorization` header to be present.

```csharp
public static AuthorizationResult CheckBearerHeaderOnly(string requestUrl, 
    bool hasAuthorizationHeader)
```

| parameter | description |

| requestUrl | The request URL to inspect for a query-string token. |
| hasAuthorizationHeader | Whether the request carries an `Authorization` header. |
