# AuthorizationSecurity.ServerScopesOmitOfflineAccess method

Validates that a server (protected resource) does NOT include `offline_access` in its `WWW-Authenticate``scope` or in its `scopes_supported`, as a server SHOULD ensure — refresh tokens are not a resource requirement (R-23.19-u).
