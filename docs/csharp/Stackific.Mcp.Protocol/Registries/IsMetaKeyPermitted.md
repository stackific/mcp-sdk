# Registries.IsMetaKeyPermitted method

Returns `true` when *key* MAY appear in `_meta` — either because it is a registry-reserved key (see [`IsReservedMetaKey`](./IsReservedMetaKey.md)) or because it is an extension-defined key carried under a valid non-reserved prefix (spec R-AppC-a, R-AppC-j, AC-46.3, AC-46.12). A bare key that is neither reserved-by-exception nor prefixed is NOT permitted.

```csharp
public static bool IsMetaKeyPermitted(string key)
```

| parameter | description |
| --- | --- |
| key | The candidate key. |

## Return Value

`true` when permitted.
