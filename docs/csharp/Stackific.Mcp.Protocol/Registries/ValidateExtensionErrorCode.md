# Registries.ValidateExtensionErrorCode method

Validates an extension/custom error *code* against the §22 collision rule: it must be an integer and MUST NOT equal any reserved code (spec R-AppB-a). The double-typed parameter lets a caller pass a fractional value to exercise the integer check, mirroring the TS helper.

```csharp
public static CustomErrorCodeValidation ValidateExtensionErrorCode(double code)
```

| code | The candidate code. |
