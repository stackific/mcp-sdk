# Registries.IsReservedMetaKey method

Returns `true` when *key* is reserved by this document and so MAY appear in `_meta` without being treated as an unknown/custom key: any key under the reserved `io.modelcontextprotocol/`/`mcp` prefix, or one of the four bare-by-exception keys (spec R-AppC-a, AC-46.3). Extension-defined keys outside the reserved prefix are NOT reserved by this predicate (use [`IsMetaKeyPermitted`](./IsMetaKeyPermitted.md) to confirm a key MAY appear at all).
