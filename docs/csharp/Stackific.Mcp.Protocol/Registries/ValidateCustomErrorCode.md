# Registries.ValidateCustomErrorCode method

Validates a custom error *code* against Appendix B's collision rule and flags whether a usable code lies inside the reserved `-32000..-32099` range (spec R-AppB-a, R-AppB-b, AC-46.1, AC-46.2). Delegates the integer/collision check to [`ValidateExtensionErrorCode`](./ValidateExtensionErrorCode.md) so the two stay in lockstep.
