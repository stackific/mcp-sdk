# Pkce.GenerateCodeVerifier method

Generates a high-entropy PKCE `code_verifier` (R-23.5-b): 32 random bytes base64url-encode to a 43-character unreserved string. Randomness is injectable via *randomSource* for deterministic tests; the default draws from RandomNumberGenerator.

```csharp
public static string GenerateCodeVerifier(Func<int, byte[]>? randomSource = null)
```

| parameter | description |
| --- | --- |
| randomSource | OPTIONAL byte source `(n) => n random bytes`; defaults to the CSPRNG. |

## Return Value

The generated verifier.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | When an injected source yields a verifier outside the 43–128 unreserved range. |
