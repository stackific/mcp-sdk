# ContinuationId.IsString method

Returns `true` when *value* is a string continuation identifier — the most common form, e.g. pagination cursors and `requestState` tokens (spec §4.5 / R-4.5-c). Clients MUST NOT parse, decode, or alter a string continuation id.

```csharp
public static bool IsString(object? value)
```

| parameter | description |
| --- | --- |
| value | The candidate continuation identifier. |

## Return Value

`true` when the value is a CLR string (or a string-valued JSON node).
