# CreateTaskResult record

The augmented result a server returns in place of a request's ordinary result to signal that a task handle was created (spec §25.3). On the wire it is a `Result` whose `resultType` is the literal string `"task"` (supplied by the runtime; see [`Task`](./ResultTypes/Task.md)) and which carries all [`McpTask`](./McpTask.md) fields directly. A client that declared the Tasks capability MUST inspect `resultType` on each eligible response and handle the task case.
