# IconSecurity.SelectIcon method

Selects the most suitable icon from *icons* for a target render size and optional theme (§14.2 icon selection). Icons whose `src` fails [`IsValidIconSrc`](./IsValidIconSrc.md) are skipped. Preference order: an exact theme match outranks a theme-agnostic icon, which outranks a theme mismatch; then, among equal theme ranks, an icon declaring `any` (or no sizes — treated as scalable) or whose nearest declared dimension is the smallest that still meets or exceeds *desiredSizePx* is preferred, falling back to the closest smaller size.

```csharp
public static Icon? SelectIcon(IReadOnlyList<Icon> icons, int desiredSizePx, 
    IconTheme? theme = default)
```

| parameter | description |
| --- | --- |
| icons | The advertised icons (for example [`Icons`](../Implementation/Icons.md)). |
| desiredSizePx | The desired rendered size in pixels (the larger of width/height). |
| theme | The preferred background theme, or `null` when the caller has no preference. |

## Return Value

The best-matching icon, or `null` when none has a usable `src`.
