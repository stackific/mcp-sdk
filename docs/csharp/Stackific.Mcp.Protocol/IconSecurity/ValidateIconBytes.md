# IconSecurity.ValidateIconBytes method

Validates an icon's byte content before rendering (R-14.2-r – R-14.2-u, AC-20.25–28).

```csharp
public static string ValidateIconBytes(ReadOnlySpan<byte> bytes, string? declaredMimeType = null, 
    IReadOnlySet<string>? allowedTypes = null)
```

| parameter | description |
| --- | --- |
| bytes | The candidate image bytes. |
| declaredMimeType | The advisory declared MIME type, if any. |
| allowedTypes | The permitted rendered types; defaults to [`DefaultImageAllowlist`](./DefaultImageAllowlist.md). |

## Return Value

The MIME type detected from the magic bytes.

## Exceptions

| exception | condition |
| --- | --- |
| [IconValidationError](../IconValidationError.md) | On an unknown type, a disallowed type, or a declared/detected mismatch. |

## Remarks

1. Detects the actual MIME type from magic bytes, ignoring the declared type.
2. Rejects when the detected type is unknown.
3. When *declaredMimeType* is provided, rejects on a mismatch.
4. Rejects a type outside the *allowedTypes* set.
