# IconSecurity.SvgHasActiveContent method

Returns `true` when an SVG document carries ACTIVE content that would execute on render — a `<script>` element, an inline `on…=` event-handler attribute, or a `javascript:` URI (§14.2 hardening). Such an SVG is refused by [`ValidateIconBytes`](./ValidateIconBytes.md).

```csharp
public static bool SvgHasActiveContent(string svg)
```

| parameter | description |
| --- | --- |
| svg | The decoded SVG document text. |

## Return Value

`true` when active content is present.
