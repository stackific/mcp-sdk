# ConformanceRequirements.TransportCredentialConveyance enumeration

How credentials are conveyed for a transport under the §29.8 conformance points (spec §29.8 items 4 &amp; 5). Distinct from the §23 [`CredentialConveyance`](./CredentialConveyance.md) in that its third case is None (neither §29.8-d nor §29.8-e applies), not a best-practice mechanism.

| None | `0` | No credential-conveyance rule applies to this transport (R-29.8: any non-HTTP, non-stdio transport). |
| Bearer | `1` | Credentials ride as an OAuth bearer token (HTTP-based transport, R-29.8-d). |
| Environment | `2` | Credentials are obtained from the process environment (stdio transport, R-29.8-e). |
