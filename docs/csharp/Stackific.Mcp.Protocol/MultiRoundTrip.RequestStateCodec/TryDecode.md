# MultiRoundTrip.RequestStateCodec.TryDecode method

Decodes a client-supplied *token* into its continuation state, VALIDATING it as untrusted input (§11.3, R-11.3-h, R-11.3-i): a malformed, mis-versioned, or out-of-range token is rejected with `false` rather than trusted. Never throws on bad input.

```csharp
public static bool TryDecode(string? token, out int round)
```

| parameter | description |
| --- | --- |
| token | The opaque token echoed by the client. |
| round | The decoded round count when `true`. |

## Return Value

`true` when the token is well-formed and trustworthy.
