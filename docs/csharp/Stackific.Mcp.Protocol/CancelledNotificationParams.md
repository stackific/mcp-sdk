# CancelledNotificationParams record

Parameters of the `notifications/cancelled` notification (spec §15.2.1), sent by a party to cancel an in-flight request that it issued earlier in the same direction. Cancellation is best-effort and races are tolerated: a receiver MAY ignore it when the request is unknown, already complete, or uncancellable, and the canceller SHOULD ignore any late response (§15.2.2/§15.2.3).
