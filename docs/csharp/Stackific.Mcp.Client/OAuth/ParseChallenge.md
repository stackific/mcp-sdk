# OAuth.ParseChallenge method

Parses a `401` response's `WWW-Authenticate` header into the structured `Bearer` challenge a client reacts to (spec §23.1, R-23.1-z): the `resource_metadata` URI to fetch and the required `scope`. Returns `null` when the response carried no parseable `Bearer` challenge.
