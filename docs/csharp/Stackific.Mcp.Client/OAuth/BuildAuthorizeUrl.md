# OAuth.BuildAuthorizeUrl method

Builds the authorization-request URL (spec §23.5): the AS metadata's `authorization_endpoint` with `response_type=code`, the client id, redirect URI, scope, state, PKCE challenge (`code_challenge_method=S256`), and the RFC 8707 `resource` parameter for audience binding (§23.6). Existing query parameters on the endpoint are preserved.

```csharp
public static string BuildAuthorizeUrl(JsonObject asMeta, string clientId, string redirectUri, 
    string resource, string scope, string state, string codeChallenge)
```

| parameter | description |
| --- | --- |
| asMeta | The authorization-server metadata. |
| clientId | The client identifier. |
| redirectUri | The redirect URI the code is returned to. |
| resource | The protected resource (audience binding, §23.6). |
| scope | The requested scope. |
| state | The CSRF state to round-trip and verify (§23.5). |
| codeChallenge | The PKCE `S256` challenge. |

## Return Value

The fully-formed authorization URL.
