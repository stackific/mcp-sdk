# OAuth.RegisterClientAsync method

Performs dynamic client registration (spec §23.4 / RFC 7591): POSTs to the AS metadata's `registration_endpoint` with the client name, redirect URIs, the `authorization_code` grant, and an `application_type` classified from the redirect URIs (loopback ⇒ `native`). Throws on a hard failure; for retryable-aware handling use [`TryRegisterClientAsync`](./TryRegisterClientAsync.md).

```csharp
public static Task<RegisteredClient> RegisterClientAsync(HttpClient http, JsonObject asMeta, 
    string clientName, IReadOnlyList<string> redirectUris)
```

| parameter | description |
| --- | --- |
| http | The HTTP client to use. |
| asMeta | The authorization-server metadata. |
| clientName | The human-readable client name. |
| redirectUris | The client's redirect URIs. |

## Return Value

The registered client (id and optional secret).
