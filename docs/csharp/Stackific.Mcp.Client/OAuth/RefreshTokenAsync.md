# OAuth.RefreshTokenAsync method

Refreshes an access token (spec §23.9): POSTs a form-urlencoded body to the AS metadata's `token_endpoint` with `grant_type=refresh_token`, the refresh token, the client id, and the SAME RFC 8707 `resource` parameter so the refreshed token stays audience-bound (R-23.9-e). An OPTIONAL narrowed *scope* MAY be supplied (R-23.9-f). The new refresh token (when issued) is returned on [`RefreshToken`](../OAuth.TokenResponse/RefreshToken.md).

```csharp
public static Task<TokenResponse> RefreshTokenAsync(HttpClient http, JsonObject asMeta, 
    string clientId, string refreshToken, string resource, string? scope = null)
```

| parameter | description |
| --- | --- |
| http | The HTTP client to use. |
| asMeta | The authorization-server metadata. |
| clientId | The client identifier. |
| refreshToken | The refresh token being exchanged. |
| resource | The same canonical resource identifier as the original request (audience binding). |
| scope | OPTIONAL narrowed scopes (R-23.9-f). |

## Return Value

The token-endpoint response (a fresh access token, still audience-bound).

| exception | condition |
| --- | --- |
| [McpError](../../Stackific.Mcp.JsonRpc/McpError.md) | When the AS has no `token_endpoint`, or the response is invalid. |
