# McpClient.PollTaskUntilTerminalAsync method

Polls `tasks/get` until the task reaches a terminal status — `completed`, `failed`, or `cancelled` (spec §25.5, §25.7) — then returns the final detailed task object. Honors the task's recommended `pollIntervalMs` (adopting the latest observed value, §25.7) and supports an overall timeout and a cancellation signal.

```csharp
public Task<JsonObject> PollTaskUntilTerminalAsync(string taskId, TimeSpan? timeout = default, 
    long fallbackIntervalMs = 500, CancellationToken cancellationToken = default)
```

| parameter | description |
| --- | --- |
| taskId | The task id. |
| timeout | An optional overall timeout; on expiry an [`McpError`](../../Stackific.Mcp.JsonRpc/McpError.md) is thrown. |
| fallbackIntervalMs | The poll interval used when the task advertises no `pollIntervalMs`. |
| cancellationToken | Cancels the poll loop. |

## Return Value

The terminal detailed task object.
