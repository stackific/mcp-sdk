# McpClient.RequestWithInputAsync method

Sends a request and drives the multi-round-trip loop (spec §11): on an `input_required` result it fulfills each input request via the registered handlers, accumulates the responses, echoes `requestState`, and resends the original method with the original arguments until a final result.
