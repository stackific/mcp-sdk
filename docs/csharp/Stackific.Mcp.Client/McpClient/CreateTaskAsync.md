# McpClient.CreateTaskAsync method

Invokes a tool as a task (spec §25.3): the client must have declared the Tasks extension, and the server returns a task handle (`resultType: "task"`) for an eligible tool. Poll with [`GetTaskAsync`](./GetTaskAsync.md) or drive to completion with [`PollTaskUntilTerminalAsync`](./PollTaskUntilTerminalAsync.md).

```csharp
public Task<JsonObject> CreateTaskAsync(string name, JsonObject? arguments = null, 
    long? ttlMs = -9223372036854775808)
```

| parameter | description |
| --- | --- |
| name | The tool name. |
| arguments | The arguments object. |
| ttlMs | The requested task lifetime in milliseconds, sent as `task.ttl` (§25.4). The default 5-minute lifetime (300000) is used when this is the sentinel [`DefaultTaskTtlMs`](./DefaultTaskTtlMs.md); pass `null` for an explicitly unbounded lifetime; pass a non-negative number to override. |

## Return Value

The raw result object (a `CreateTaskResult` when the server made it a task).
