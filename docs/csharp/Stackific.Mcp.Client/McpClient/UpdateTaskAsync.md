# McpClient.UpdateTaskAsync method

Supplies input to a task awaiting it via `tasks/update` (spec §25.8): the responses are keyed by a currently-outstanding `inputRequests` key from the task's `input_required` state. The server binds the outstanding subset, drops stale keys (R-25.8-h), and advances the task.

```csharp
public Task<JsonObject> UpdateTaskAsync(string taskId, JsonObject inputResponses)
```

| parameter | description |
| --- | --- |
| taskId | The task id. |
| inputResponses | The responses keyed by outstanding input-request key. |

## Return Value

The empty acknowledgement (`resultType: "complete"`); observe the new state via [`GetTaskAsync`](./GetTaskAsync.md) (§25.8).
