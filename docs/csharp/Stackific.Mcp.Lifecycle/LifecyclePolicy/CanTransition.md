# LifecyclePolicy.CanTransition method

Returns `true` when the transition from *from* to *to* is permitted by the §27.2 state machine.

```csharp
public static bool CanTransition(LifecycleState from, LifecycleState to)
```

| parameter | description |
| --- | --- |
| from | The current lifecycle state. |
| to | The proposed next lifecycle state. |

## Return Value

`true` when the transition is legal; otherwise `false`.

## Remarks

Permitted: Active → Deprecated, Deprecated → Active, Deprecated → Removed. Forbidden: any same-state "transition" (R-27.1-e), Active → Removed (R-27.2-b), and any transition out of Removed (Removed is terminal).
