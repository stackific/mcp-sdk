# LifecyclePolicy.AddCalendarMonths method

Adds *months* calendar months to *date*, clamping the day to the last valid day of the target month when the original day overflows (for example Jan 31 + 1 month → Feb 28).

```csharp
public static DateOnly AddCalendarMonths(DateOnly date, int months)
```

| parameter | description |
| --- | --- |
| date | The base date. |
| months | The number of calendar months to add (may be zero or negative). |

## Return Value

The resulting date with end-of-month clamping applied.

## Remarks

This mirrors the TypeScript `addCalendarMonths`, which performs the same end-of-month clamping. Int32) already clamps to the last day of the target month, so this is a thin, intent-revealing wrapper.
