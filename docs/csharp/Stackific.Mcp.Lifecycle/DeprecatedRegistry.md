## Remarks

The registry is a derived, consolidated view; the per-feature notices at the cross-referenced defining sections are authoritative and resolve any conflict. New implementations SHOULD NOT adopt the registered deprecated features, and existing implementations SHOULD migrate before each feature's earliest removal (R-27.3-a, R-27.3-b).
