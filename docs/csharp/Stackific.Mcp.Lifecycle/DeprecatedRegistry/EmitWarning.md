# DeprecatedRegistry.EmitWarning method

Emits a runtime deprecation warning through the environment-idiomatic, out-of-band channel (stderr) (spec §27.4, R-27.4-d).

```csharp
public static void EmitWarning(string feature, string migration, TextWriter? writer = null)
```

| parameter | description |
| --- | --- |
| feature | The name of the deprecated feature being exercised. |
| migration | The documented migration guidance. |
| writer | An optional sink; defaults to Error when `null`. |

## Remarks

IMPORTANT: this is advisory only. It MUST NOT inject the warning into the protocol wire format and does not alter message semantics (R-27.4-e). It is the C# counterpart of the TypeScript `emitDeprecationWarning`, which routes to `console.warn` (stderr). A custom *writer* may be supplied (for tests or alternative sinks); when omitted, the warning is written to Error.
