# JsonRpcMessageSerializer class

Classifies raw JSON into a [`JsonRpcMessage`](./JsonRpcMessage.md) per the structural rules of §3.1, and renders messages back to wire text. Malformed input is rejected with the correct JSON-RPC error (parse → -32700, structurally invalid → -32600).
