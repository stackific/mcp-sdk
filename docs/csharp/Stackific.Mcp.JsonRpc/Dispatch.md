## Remarks

Provides the minimal, pure, unit-testable dispatch surface: given a classified request and a registry of known methods, it produces the correct error response when the method is unrecognized or the params are invalid. The standard JSON-RPC error codes −32601 (method not found) and −32602 (invalid params) are used because they originate from the JSON-RPC 2.0 specification (see [`ErrorCodes`](./ErrorCodes.md) for the full MCP registry).
