# RequestId structure

A JSON-RPC request identifier (spec §3.2): either a JSON string or a JSON number, and never `null`. The wire type is preserved exactly — a numeric id round-trips as a number and a string id as a string, and the two are never coerced into one another. Integral numbers are kept as a 64-bit integer so large ids round-trip without the precision loss a Double would introduce.
