# McpError class

A protocol-level failure that maps to a JSON-RPC error response (spec §22). Throwing an [`McpError`](./McpError.md) anywhere in request processing lets the transport translate it into the correct `error` object — and, on HTTP, the correct status code.

public class McpError : Exception

## Remarks

This is distinct from a feature-level error (for example a tool that runs but fails, reported with `isError: true` inside a successful result, §22.5). Use [`McpError`](./McpError.md) only when the request itself cannot be dispatched or processed.
