# Dispatch.DispatchRequest method

Validates *request* against *registry* and returns the dispatch outcome.

```csharp
public static DispatchOutcome DispatchRequest(JsonRpcRequest request, MethodRegistry registry)
```

| parameter | description |
| --- | --- |
| request | A request produced by message classification. |
| registry | The set of methods the receiver handles. |

## Return Value

[`Success`](../DispatchOutcome/Success.md) when the method is registered and its params are valid; otherwise a failure carrying an error response whose `id` echoes the request id with the same JSON type and value (R-3.2-e, R-3.2-f, R-3.2-g).

## Remarks

A failure is produced when any of the following hold:

* The method name is not in *registry* → method-not-found (−32601, R-3.3-j).
* [`RequiresParams`](../MethodDescriptor/RequiresParams.md) is `true` and `params` is absent (for example a method whose per-request `_meta` is REQUIRED) → invalid-params (−32602, R-3.3-i).
* [`ParamsValidator`](../MethodDescriptor/ParamsValidator.md) is provided and the present `params` fails validation → invalid-params (−32602, R-3.3-k).
