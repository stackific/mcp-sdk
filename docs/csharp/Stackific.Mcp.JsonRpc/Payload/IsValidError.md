# Payload.IsValidError method

Returns `true` when *node* is a valid wire `error` object (§3.8): a JSON object with a `code` that is an integer within the IEEE-754 safe-integer range, a string `message`, and an OPTIONAL `data` member of any JSON type. Additional members are allowed (passthrough). (R-3.8-a, R-3.8-c, R-3.8-e, §2.5)

```csharp
public static bool IsValidError(JsonNode? node)
```

| parameter | description |
| --- | --- |
| node | The candidate error node. |

## Return Value

`true` when the node is a structurally valid error object.

## Remarks

The legal value set for `code` is defined in §22 and is a conformance rule, not a parse rule — any safe integer parses here. A fractional code (for example `−32601.5`) or a string code is rejected.
