# Payload.IsKnownResultType method

Returns `true` when *value* is one of the two spec-defined `resultType` values (`"complete"` or `"input_required"`).

```csharp
public static bool IsKnownResultType(string value)
```

| parameter | description |
| --- | --- |
| value | The `resultType` string to test. |

## Return Value

`true` when the value is recognized.

## Remarks

A receiver that encounters an unrecognized `resultType` MUST treat the whole response as an error (R-3.6-f) and MUST NOT read any other result members (R-3.6-g).
