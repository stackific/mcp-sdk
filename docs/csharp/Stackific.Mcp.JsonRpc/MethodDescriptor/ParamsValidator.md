# MethodDescriptor.ParamsValidator property

Optional validator for the incoming `params` object. When provided and the request's `params` fails validation (the delegate returns `false`), [`DispatchRequest`](../Dispatch/DispatchRequest.md) returns an invalid-params error response. (R-3.3-k)

```csharp
public Func<JsonObject, bool>? ParamsValidator { get; set; }
```

## Remarks

This is the C# analogue of the TypeScript `paramsSchema` (a Zod schema). It is invoked only when `params` is present; leave it `null` to skip schema validation (the method accepts any params, subject only to the [`RequiresParams`](./RequiresParams.md) presence check).
