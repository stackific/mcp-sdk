# JsonRpcErrorResponse constructor

A JSON-RPC error response (§3.5.2/§22.1): carries an [`Error`](./Error.md) and, normally, the answered [`Id`](./Id.md). The id MAY be absent only when the originating request's id could not be determined (§22.6).

```csharp
public JsonRpcErrorResponse(RequestId? Id, JsonRpcError Error)
```

| parameter | description |
| --- | --- |
| Id | The identifier of the request being answered, or `null` when undeterminable. |
| Error | The error object (§3.8). |
