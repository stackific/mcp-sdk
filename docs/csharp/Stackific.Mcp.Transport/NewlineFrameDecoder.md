# NewlineFrameDecoder class

A stateful newline-delimited frame decoder: buffers partial frames across [`Push`](./NewlineFrameDecoder/Push.md) calls and emits each complete unit as soon as its `\n` delimiter arrives (spec §7.2). The C# counterpart of the TypeScript `NewlineFrameDecoder`.
