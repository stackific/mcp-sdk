## Remarks

[`Clean`](./TransportCloseInfo/Clean.md)`true` is an orderly shutdown each side had the opportunity to observe (R-7.2-t). [`Clean`](./TransportCloseInfo/Clean.md)`false` is an abrupt disconnection — the channel dropped without an orderly close — which a transport MUST still make observable (R-7.5-a, R-7.5-b).
