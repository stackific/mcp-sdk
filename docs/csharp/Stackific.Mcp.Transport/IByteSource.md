# IByteSource interface

A push-based source of inbound bytes — the C# analogue of a Node `Readable` stream's `'data'`/`'end'` events (spec §8 topology). A real process adapter raises [`DataReceived`](./IByteSource/DataReceived.md) for each chunk read from a child stream and [`Ended`](./IByteSource/Ended.md) when the stream reaches EOF; an in-memory test pipe raises the same events synchronously.
