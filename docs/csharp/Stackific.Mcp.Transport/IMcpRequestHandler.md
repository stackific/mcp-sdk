# IMcpRequestHandler interface

The transport-facing surface of an MCP server: it accepts a parsed request and produces the final JSON-RPC response, emitting any request-scoped notifications through the supplied [`IServerNotifier`](./IServerNotifier.md). Implemented by `McpServer`; consumed by every server transport (the in-memory bridge and the Streamable HTTP adapter), which handle framing, headers, and routing.
