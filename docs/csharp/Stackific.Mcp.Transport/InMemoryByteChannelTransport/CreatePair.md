# InMemoryByteChannelTransport.CreatePair method

Creates a linked pair of in-memory byte-channel transports. Anything one endpoint sends is delivered to the other; closing or disconnecting either endpoint makes both observe the close (spec §7.1, §7.4, §7.2 clean close, §7.5). Mirrors TypeScript `createInMemoryTransportPair`.
