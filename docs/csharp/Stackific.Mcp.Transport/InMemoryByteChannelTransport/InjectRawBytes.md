# InMemoryByteChannelTransport.InjectRawBytes method

Feeds arbitrary raw bytes into this endpoint's receive path, as if they had arrived on the wire. Used to exercise receiver-side decode-error handling (for example a corrupt or non-UTF-8 unit). Not part of the [`IByteChannelTransport`](../IByteChannelTransport.md) contract — a test/simulation affordance. Mirrors TypeScript `InMemoryTransport.injectRawBytes`.
