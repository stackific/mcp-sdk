# InMemoryByteChannelTransport.Disconnect method

Simulates an abrupt disconnection (channel dropped without an orderly close). Both endpoints observe it via [`OnClose`](./OnClose.md) with `Clean: false`, so neither side blocks as though the channel were still live (spec §7.5; R-7.5-a, R-7.5-b). Not part of the contract — a test/simulation affordance. Mirrors TypeScript `InMemoryTransport.disconnect`.
