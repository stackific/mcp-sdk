# IChildProcess interface

The minimal view of a child process the stdio transport needs (spec §8 topology). Modeled so the three streams can be in-memory pipes in tests (no real OS process), while a real Process is adapted to the same shape by [`ProcessChild`](./ProcessChild.md). The C# counterpart of the TypeScript `ChildProcessLike`.
