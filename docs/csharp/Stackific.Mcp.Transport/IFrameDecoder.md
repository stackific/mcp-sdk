# IFrameDecoder interface

Splits a byte stream back into the byte boundaries of individual messages, using framing alone — the decoder MUST NOT parse the JSON body to find where one message ends and the next begins (spec §7.2; R-7.2-b, R-7.2-c, R-7.2-d).

## Remarks

A decoder is stateful: it buffers bytes that do not yet form a complete unit and emits each complete unit as soon as its delimiter arrives. The C# counterpart of the TypeScript `FrameDecoder`.
