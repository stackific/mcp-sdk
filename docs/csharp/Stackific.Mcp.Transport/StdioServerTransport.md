# StdioServerTransport class

The server side of a stdio connection: reads client requests/notifications from stdin and writes responses/notifications to stdout (spec §8 server role). The C# counterpart of the TypeScript `StdioServerTransport`.

## Remarks

Enforces the server stream-role rule — it MUST NOT write a JSON-RPC request to stdout and MUST NOT write non-MCP content there; diagnostics belong on stderr (R-8.3-b, R-8.5-a, R-8.5-b). Graceful shutdown is observed when stdin reaches EOF, at which point the server SHOULD exit promptly (R-8.6.2-b); the server MAY also initiate shutdown by closing stdout (R-8.6.2-c) via [`CloseAsync`](./StdioServerTransport/CloseAsync.md).
