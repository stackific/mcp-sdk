# MalformedIdError.BuildParseErrorResponse method

Builds a parse-error response for a request whose id could not be read (spec §7.2; R-7.2-h). Mirrors TypeScript `buildParseErrorResponse`.

```csharp
public static JsonRpcErrorResponse BuildParseErrorResponse(bool nullId = false)
```

| parameter | description |
| --- | --- |
| nullId | When `true`, the response carries `"id": null`; when `false`, the id member is omitted entirely. Both forms are valid (R-7.2-h). |

## Return Value

The parse-error response. Its [`Id`](../../Stackific.Mcp.JsonRpc/JsonRpcErrorResponse/Id.md) is always `null`; the *nullId* choice (explicit `null` vs omitted) is realized when the response is serialized, since the C# wire writer omits a null id — both wire forms decode back to a null id.
