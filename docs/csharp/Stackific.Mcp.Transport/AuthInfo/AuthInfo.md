# AuthInfo constructor

The validated authorization identity a transport threads into request processing when a request carried a bearer token (spec §23). It is per-request input (never connection state, §4.4) and is surfaced to tool handlers via the request context.

```csharp
public AuthInfo(string Token, string? ClientId = null, IReadOnlyList<string>? Scopes = null, 
    string? Audience = null, long? ExpiresAt = null)
```

| parameter | description |
| --- | --- |
| Token | The raw bearer token presented on the request. |
| ClientId | The authenticated client identifier, if known. |
| Scopes | The granted OAuth scopes. |
| Audience | The token's bound audience (the protected resource), if known (§23.6). |
| ExpiresAt | The token expiry as a Unix timestamp in seconds, if known. |
