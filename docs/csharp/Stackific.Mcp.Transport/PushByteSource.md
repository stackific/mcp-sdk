# PushByteSource class

An in-memory, push-based [`IByteSource`](./IByteSource.md) + [`IByteSink`](./IByteSink.md) pipe used to drive the stdio transport in tests without spawning a real OS process. Writing bytes synchronously raises [`DataReceived`](./PushByteSource/DataReceived.md); [`End`](./PushByteSource/End.md) raises [`Ended`](./PushByteSource/Ended.md) exactly once.
