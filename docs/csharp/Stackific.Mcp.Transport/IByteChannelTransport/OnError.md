# IByteChannelTransport.OnError method

Registers a handler for receiver-side transport/parse-level errors — for example an inbound unit that is not well-formed UTF-8 or not a single JSON value (R-7.6-b, R-7.6-c). These surface on the side that received the bad unit, as an observable failure, rather than being silently dropped or thrown back into an unrelated sender's [`Send`](./Send.md) (R-7.5-j).

```csharp
public IDisposable OnError(Action<TransportError> handler)
```

| parameter | description |
| --- | --- |
| handler | Invoked once per receiver-side decode error. |

## Return Value

An IDisposable that unsubscribes *handler* when disposed.

## Remarks

This is distinct from a JSON-RPC error response (a normal, fully delivered message) and from a send failure (surfaced synchronously by [`Send`](./Send.md)).
