# IByteChannelTransport.Send method

Sends one message over the channel. MUST NOT silently drop it: on a closed or failed channel this MUST surface an observable failure (throw a [`TransportError`](../TransportError.md)) rather than discarding the message (R-7.2-q, R-7.2-s, R-7.5-i, R-7.5-j).
