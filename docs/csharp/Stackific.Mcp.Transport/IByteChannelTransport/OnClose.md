# IByteChannelTransport.OnClose method

Registers a handler invoked once when the channel becomes unusable — by a clean close or an abrupt disconnection (R-7.2-t, R-7.5-a). A handler registered after the channel already closed still observes the close exactly once.

```csharp
public IDisposable OnClose(Action<TransportCloseInfo> handler)
```

| parameter | description |
| --- | --- |
| handler | Invoked once with the close information. |

## Return Value

An IDisposable that unsubscribes *handler* when disposed.
