# IMessageFramer interface

Encodes messages to delimited byte units and produces decoders that recover them. An [`IMessageFramer`](./IMessageFramer.md) is the §7.2 framing guarantee made concrete. The C# counterpart of the TypeScript `MessageFramer`.
