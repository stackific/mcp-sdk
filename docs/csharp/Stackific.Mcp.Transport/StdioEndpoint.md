# StdioEndpoint class

Common stdio plumbing for both endpoints: newline framing over a byte sink, a stateful decoder over a byte source, malformed-line tolerance, and the observable message/error/close surface of [`IByteChannelTransport`](./IByteChannelTransport.md). Subclasses supply the concrete subprocess lifecycle. The C# counterpart of the TypeScript `StdioEndpoint` abstract base.
