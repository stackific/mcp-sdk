# StreamableHttpServerOptions.AllowedOrigins property

The exact `Origin` values the server accepts (DNS-rebinding defense, §9.11). When `null` (the default), the server accepts any request whose `Origin` is loopback-origin-shaped (a `http(s)://localhost` / `127.0.0.1` / `[::1]` origin, any port) and rejects every other present `Origin` with a `403`. An empty set rejects every present `Origin`. A request with no `Origin` header always passes (non-browser clients). (R-9.11-a, R-9.11-b)
