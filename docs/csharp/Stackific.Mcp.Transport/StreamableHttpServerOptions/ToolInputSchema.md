# StreamableHttpServerOptions.ToolInputSchema property

Resolves a tool's `inputSchema` by tool name so the adapter can validate the request's `Mcp-Param-*` headers against the body (§9.5.4). When `null` (the default), no `Mcp-Param-*` receiver validation is performed — an unrecognized `Mcp-Param-*` header is then simply ignored, exactly as before. Returns `null` for an unknown tool.
