## Remarks

An [`IByteChannelTransport`](./IByteChannelTransport.md) is a bidirectional channel that carries the [`JsonRpcMessage`](../Stackific.Mcp.JsonRpc/JsonRpcMessage.md) union as complete UTF-8 JSON values, preserves integrity, delivers in both directions, never silently drops a message, and defines an observable clean close and an observable abrupt disconnection.

A transport does NOT interpret method/params/result or perform capability or version negotiation; those are core-protocol concerns carried unchanged. This is the C# counterpart of the TypeScript `Transport` interface. It is deliberately named distinctly from the higher-level request/response [`ClientTransport`](./ClientTransport.md) so the two coexist: [`ClientTransport`](./ClientTransport.md) is an RPC abstraction (send a request, await its response); this is the lower-level observable byte channel the stdio and in-memory reference transports implement.

Handler registration methods return an IDisposable whose disposal unsubscribes the handler — the idiomatic C# analogue of the TypeScript `Unsubscribe` function.
