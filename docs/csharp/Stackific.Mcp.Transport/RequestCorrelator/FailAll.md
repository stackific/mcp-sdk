# RequestCorrelator.FailAll method

Fails every outstanding request — the action a transport takes on abrupt or clean disconnection so no in-flight request can hang (spec §7.5; R-7.5-c, R-7.5-d, R-7.5-e). Mirrors TypeScript `RequestCorrelator.failAll`.

```csharp
public IReadOnlyList<RequestId> FailAll(TransportError error)
```

| parameter | description |
| --- | --- |
| error | The transport-level error to fault each outstanding Task with. |

## Return Value

The ids that were failed, in registration order.

## Remarks

After this returns the correlator holds no outstanding requests, so the same ids MAY be reissued against a fresh connection (R-7.5-f, R-7.7-b) — no state is bound to the lost connection.
