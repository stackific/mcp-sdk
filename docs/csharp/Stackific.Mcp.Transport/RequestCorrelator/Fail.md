# RequestCorrelator.Fail method

Fails a single outstanding request with a transport-level error, faulting its Task so the caller can observe the failure rather than waiting forever (spec §7.5; R-7.5-d, R-7.5-e). Mirrors TypeScript `RequestCorrelator.fail`.

```csharp
public bool Fail(RequestId id, TransportError error)
```

| parameter | description |
| --- | --- |
| id | The id of the outstanding request to fail. |
| error | The transport-level error to fault the Task with. |

## Return Value

`true` if the request was outstanding and is now failed.
