# TransportContract.ExtractEnvelopeForMirroring method

Extracts the envelope fields a transport MAY mirror into transport-level metadata for routing/inspection (for example HTTP headers; see S14/S15) (spec §7.4; R-7.4-e). Mirrors TypeScript `extractEnvelopeForMirroring`.

## Return Value

The mirrorable context, or `null`.

## Remarks

The returned values are read from the message body, which remains the authoritative source of truth — the mirror is a derived copy, never an alternative input. Returns `null` when the body carries no valid envelope, so a transport never mirrors fabricated values.
