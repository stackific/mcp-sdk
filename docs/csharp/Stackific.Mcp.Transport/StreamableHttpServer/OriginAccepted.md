# StreamableHttpServer.OriginAccepted method

Decides whether a present `Origin` is accepted (DNS-rebinding defense, §9.11). When *allowedOrigins* is `null`, only loopback-shaped origins are accepted (the loopback-safe default for a local server); when it is a set, the match is exact. (R-9.11-a/b)

```csharp
public static bool OriginAccepted(string origin, IReadOnlySet<string>? allowedOrigins)
```

| parameter | description |
| --- | --- |
| origin | The request's non-empty `Origin` header value. |
| allowedOrigins | The configured accepted origins, or `null` for the loopback default. |

## Return Value

`true` when the origin is accepted.
