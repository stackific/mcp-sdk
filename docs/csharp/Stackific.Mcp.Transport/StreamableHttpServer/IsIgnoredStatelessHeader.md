# StreamableHttpServer.IsIgnoredStatelessHeader method

Returns `true` when *name* is a header this stateless transport MUST ignore: a session-identifier header (`Mcp-Session-Id`/`X-Session-Id`/`Session-Id`) or the `Last-Event-ID` resumption header. (R-9.9-b/c/d/g, R-9.6.2-h) Comparison is case-insensitive.

```csharp
public static bool IsIgnoredStatelessHeader(string name)
```

| parameter | description |
| --- | --- |
| name | The header name. |

## Return Value

`true` when the header MUST be ignored.

## Remarks

The adapter never reads these headers when processing a request (no session affinity, no resumption), so an incoming value is effectively ignored. This predicate makes the rule explicit and testable, and is the basis for any explicit stripping a host wishes to apply.
