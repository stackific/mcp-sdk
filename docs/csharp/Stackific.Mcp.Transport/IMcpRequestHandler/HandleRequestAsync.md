# IMcpRequestHandler.HandleRequestAsync method

Processes one JSON-RPC request and returns the final response (a success or an error response). Protocol failures are returned as error responses, not thrown (spec §22).


| parameter | description |
| --- | --- |
| request | The parsed request. |
| notifier | The sink for request-scoped notifications. |
| authInfo | The validated bearer identity, if the request was authenticated. |
