# StdioClientTransport.CloseAsync method

Graceful shutdown (spec §8.6.2-a): (1) close the child's stdin (EOF — the only portable graceful signal), (2) wait for the process to exit, (3) forcibly terminate it if it does not exit within `ShutdownGraceMs` (R-8.6.3-a). Resolves once the process has exited (or been force-terminated). The close is observable via [`OnClose`](../StdioEndpoint/OnClose.md) with `Clean: true`.
