# RequestOptions class

Per-request options for a client transport send: the sink for request-scoped notifications (progress/logging) that arrive on this request's stream (spec §9.6.2), an optional progress token to opt into progress, extra `_meta` to carry (for example W3C trace context, §15.4), a timeout, and a cancellation signal that closes the request's stream (§9.6.2/§15.2).
