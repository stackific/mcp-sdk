# StreamableHttpServerOptions record

Configures the Streamable HTTP adapter's transport hardening (spec §9.3, §9.5, §9.9, §9.11). All members are optional; the defaults preserve the adapter's pre-hardening behaviour except for the security checks the spec mandates (Origin validation, Content-Type/Accept validation), which are loopback-safe by default.
