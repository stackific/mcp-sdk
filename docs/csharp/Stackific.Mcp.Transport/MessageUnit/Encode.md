# MessageUnit.Encode method

Encodes a [`JsonRpcMessage`](../../Stackific.Mcp.JsonRpc/JsonRpcMessage.md) to its UTF-8 JSON bytes, without any framing (spec §7.2). Mirrors TypeScript `encodeMessageUnit`.

```csharp
public static byte[] Encode(JsonRpcMessage message)
```

| parameter | description |
| --- | --- |
| message | The message to encode. |

## Return Value

The framing-less UTF-8 JSON bytes.

## Remarks

The serializer escapes any embedded newline inside a string as the two-character sequence `\n`, so the produced bytes never contain a raw `0x0a` — which is what makes newline framing unambiguous (R-7.2-d).
